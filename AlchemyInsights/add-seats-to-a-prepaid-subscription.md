# <a name="add-seats-to-a-prepaid-subscription"></a>เพิ่มที่นั่งกับการจ่ายล่วงหน้า

1. ในศูนย์ดูแล ไป**การเรียกเก็บเงิน**\>หน้าการ[บอกรับเป็นสมาชิก](https://go.microsoft.com/fwlink/p/?linkid=842054)

2. เลือกการบอกรับเป็นสมาชิกที่จ่ายล่วงหน้าคุณต้องการเพิ่มผู้ใช้

3. ในเมนู**การกระทำเพิ่มเติม**เลือก**ต่ออายุ หรือเพิ่มสิทธิ์การใช้งาน ด้วยโทเค็นการ**

4. เลือกตัวเลือก**หมายเลขผลิตภัณฑ์ใหม่ และที่ไม่ได้ใช้** (ถ้าคุณสมัครใช้งานนี้ถูกสร้างขึ้น ด้วยหมายเลขผลิตภัณฑ์ที่สั่งซื้อ ณที่ตั้งการขายปลีก คุณอาจมีตัวเลือกในการเพิ่มผู้ใช้ให้กับวิธีการชำระเงินอื่น ๆ)

5. เลือก**ถัดไป**

6. ใส่หมายเลขผลิตภัณฑ์ แล้วเลือก**ถัดไป**

7. เลือกตัวเลือกในการเพิ่มผู้ใช้เพิ่มเติม เลือก**ถัดไป**จาก นั้นเลือก**Redeem**

คุณกำลังใช้ศูนย์ดูแลแบบใหม่หรือไม่ สำหรับขั้นตอนใหม่ ดู[สิทธิ์การใช้งานเพื่อเพิ่ม หรือขยายการบอกรับเป็นสมาชิกที่ได้รับการชำระเงินสำหรับการใช้หมายเลขผลิตภัณฑ์](https://docs.microsoft.com/en-us/office365/admin/misc/add-licenses-using-product-key)
