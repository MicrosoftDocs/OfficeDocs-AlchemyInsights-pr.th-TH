# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>การใช้กล่องจดหมายที่ใช้ร่วมกันบนอุปกรณ์เคลื่อนที่

คุณไม่สามารถเข้าถึงกล่องจดหมายที่ใช้ร่วมกันจากโปรแกรมประยุกต์ Outlook ที่ติดตั้งอยู่บนอุปกรณ์เคลื่อนที่ คุณต้องการคุณลักษณะนี้หรือไม่ ให้ทีมงานวิศวกรรมของ Outlook ทราบ [ลงคะแนนเสียงที่นี่](https://go.microsoft.com/fwlink/?linked=862116)!
  
คุณสามารถใช้กล่องจดหมายที่ใช้ร่วมกันบนอุปกรณ์เคลื่อนที่โดยการเปิดเบราว์เซอร์ของคุณลงชื่อเข้าใช้ Office ๓๖๕และจากนั้นไปยัง Outlook บนเว็บ สำหรับคำแนะนำโปรดดูที่[การเพิ่มกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บ](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)
  
ถ้าคุณต้องเข้าใช้งานกล่องจดหมายที่ใช้ร่วมกันโดยใช้แอป Outlook บนอุปกรณ์เคลื่อนที่ให้พิจารณาสร้างกลุ่ม Office ๓๖๕แทนการใช้กล่องจดหมายที่ใช้ร่วมกัน สำหรับข้อมูลเพิ่มเติมโปรดดูที่การ[เปรียบเทียบกลุ่ม](https://docs.microsoft.com/office365/admin/create-groups/compare-groups)
  
เรียนรู้เพิ่มเติม:[รับความช่วยเหลือในแอปสำหรับ Outlook สำหรับ iOS และ Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
