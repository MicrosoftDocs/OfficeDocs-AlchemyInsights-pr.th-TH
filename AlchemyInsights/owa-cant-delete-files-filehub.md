# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>ไม่สามารถลบไฟล์หรือรูปถ่ายจากไฟล์ใน Outlook บนเว็บ

**ไฟล์** จะแสดงไฟล์และรูปถ่ายที่แนบมากับข้อความในกล่องจดหมายของคุณ เมื่อต้องการเอาไฟล์และรูปถ่ายออกจากไฟล์คุณจำเป็นต้องลบข้อความอีเมลที่พวกเขาแนบอยู่ หลังจากลบข้อความแล้วคุณจำเป็นต้องลบข้อความออกจากโฟลเดอร์รายการที่ถูกลบของคุณอีกด้วย
