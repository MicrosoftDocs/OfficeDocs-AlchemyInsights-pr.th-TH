# <a name="delete-a-sharepoint-site"></a>ลบไซต์ SharePoint

1. ในศูนย์การจัดการ SharePoint ใหม่บนหน้าไซต์ให้คลิกกล่องกาเครื่องหมายที่อยู่ถัดจากไซต์ที่คุณต้องการลบ
    
2. บนแถบเครื่องมือ ribbon ให้คลิก**ลบ**
