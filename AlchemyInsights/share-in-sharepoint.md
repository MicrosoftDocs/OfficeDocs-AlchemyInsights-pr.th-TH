# <a name="how-to-share-in-sharepoint-online"></a>วิธีการใช้ร่วมกันใน SharePoint แบบออนไลน์

เมื่อต้องการแชร์ไซต์ ให้ทําดังนี้
  
1. เปิดเว็บไซต์ในเว็บเบราว์เซอร์
    
2. ถ้าไซต์อยู่ในกลุ่ม Microsoft 365 ให้คลิกไอคอน การตั้งค่า ที่ด้านขวาบน แล้วคลิก**สิทธิ์ของไซต์** คลิก**เชิญบุคคล**แล้วเลือกเพิ่มสมาชิกลงในกลุ่ม Microsoft 365 หรือแชร์เฉพาะไซต์ 
    
    สําหรับไซต์การสื่อสาร ให้คลิก**แชร์ไซต์**ที่ด้านขวาบน แล้วป้อนชื่อของบุคคลนั้น แล้วเลือกระดับสิทธิ์ (อ่าน แก้ไข หรือควบคุมทั้งหมด) 
    
    สําหรับไซต์ทีมคลาสสิก ให้คลิก**แชร์**ที่ด้านขวาบน ป้อนชื่อของบุคคลที่คุณต้องการเชิญหรือพิมพ์ ทุกคน เพื่อแชร์กับทุกคนในองค์กรของคุณ คลิก**แชร์**
    
[เรียนรู้วิธีแชร์ไฟล์หรือโฟลเดอร์](https://go.microsoft.com/fwlink/?linkid=511430)
