# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>การแก้ไขปัญหาการตั้งค่า Outlook แบบเคลื่อนที่สําหรับกล่องจดหมายที่ใช้ร่วมกัน

คุณไม่สามารถเข้าถึงกล่องจดหมายที่ใช้ร่วมกันจากแอป Outlook ที่ติดตั้งบนอุปกรณ์เคลื่อนที่ คุณต้องการคุณลักษณะนี้หรือไม่ แจ้งให้ทีมวิศวกรของ Outlook ทราบ [โหวตที่นี่](https://go.microsoft.com/fwlink/?linked=862116)!
  
คุณสามารถใช้กล่องจดหมายที่แชร์บนอุปกรณ์เคลื่อนที่เท่านั้นโดยเปิดเบราว์เซอร์ของคุณ สําหรับคําแนะนํา ให้ดูที่[เพิ่มกล่องจดหมายที่แชร์ใน Outlook บนเว็บ](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207) จาก Outlook บนเว็บ คุณจะสามารถเข้าถึงกล่องจดหมายที่ใช้ร่วมกันได้
  
เรียนรู้เพิ่มเติม:[รับความช่วยเหลือในแอปสําหรับ Outlook สําหรับ iOS และ Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
