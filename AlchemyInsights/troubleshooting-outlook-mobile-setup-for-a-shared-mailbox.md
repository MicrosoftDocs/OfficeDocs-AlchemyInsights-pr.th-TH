# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>การแก้ไขปัญหาการตั้งค่า Outlook mobile สำหรับกล่องจดหมายที่แชร์

คุณไม่สามารถเข้าถึงกล่องจดหมายที่แชร์จากแอป Outlook ที่ติดตั้งบนอุปกรณ์เคลื่อนที่ได้ คุณต้องการฟีเจอร์นี้หรือไม่ ให้ทีมวิศวกร Outlook ทราบ [โหวตที่นี่](https://go.microsoft.com/fwlink/?linked=862116)!
  
คุณสามารถใช้กล่องจดหมายที่แชร์บนอุปกรณ์เคลื่อนที่ได้โดยการเปิดเบราว์เซอร์ของคุณแล้วลงชื่อเข้าใช้จากนั้นไปยัง Outlook บนเว็บ สำหรับคำแนะนำให้ดู[ที่เพิ่มกล่องจดหมายที่แชร์ใน Outlook บนเว็บ](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207) จาก Outlook บนเว็บคุณจะสามารถเข้าถึงกล่องจดหมายที่แชร์ได้
  
เรียนรู้เพิ่มเติม: [รับวิธีใช้ในแอปสำหรับ Outlook สำหรับ iOS และ Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
