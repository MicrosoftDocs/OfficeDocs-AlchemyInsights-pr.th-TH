# <a name="move-a-folder"></a>ย้ายโฟลเดอร์

คุณสามารถลากโฟลเดอร์ภายใต้ **'รายการโปรด'** เพื่อจัดลำดับใหม่ได้
  
คุณไม่สามารถเรียงลำดับใหม่โฟลเดอร์ภายใต้**โฟลเดอร์**หรือย้ายโฟลเดอร์จะกลายเป็น โฟลเดอร์ย่อย

*หมายเหตุ:* ถ้าคุณย้ายโฟลเดอร์โฟลเดอร์ย่อย กลายเป็นการสร้างโฟลเดอร์ใหม่ และคัดลอกอีเมลของคุณทั้งหมดจากโฟลเดอร์ย่อยไปเป็นโฟลเดอร์ใหม่
  
เมื่อต้องการเรียนรู้เพิ่มเติม ดู[การทำงานกับโฟลเดอร์ข้อความใน Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a)
