# <a name="troubleshoot-issues-with-microsoft-intune"></a>การแก้ไขปัญหาเกี่ยวกับ Microsoft Intune

ตรวจทานแหล่งข้อมูลที่แสดงรายการด้านล่างเพื่อแก้ไขปัญหาของคุณได้แล้วในตอนนี้
  
1. ใช้การ [แก้ไขปัญหาของ Intune ของพอร์ทัล](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) เพื่อวินิจฉัยและแก้ไขปัญหาทั่วไปบางอย่าง ตรวจทานเอกสาร [นี้](https://docs.microsoft.com/intune/help-desk-operators) สำหรับรายละเอียดเพิ่มเติม  
2. ตรวจทาน [Intune TechNet ](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)เพื่อค้นหาคำตอบและวิธีแก้ไขปัญหาทั่วไป  
3. ตรวจสอบสถานภาพ [บริการ](https://portal.office.com/AdminPortal/Home#/servicehealth) เพื่อดูสถานะของบริการ   
4. ตรวจทาน [เอกสารประกอบของ Intune](https://docs.microsoft.com/intune/) สำหรับการวางแผนวิธีการแก้ไขปัญหาบทความและอื่นๆ
