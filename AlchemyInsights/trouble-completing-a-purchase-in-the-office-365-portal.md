# <a name="trouble-completing-a-purchase-in-the-microsoft-365-admin-center"></a>ปัญหาในการดำเนินการซื้อใน Microsoft 365 admin ศูนย์

เมื่อพยายามที่จะทำการซื้อ ถ้าคุณได้รับพร้อมท์เตือนหลายครั้งเพื่อเข้าสู่ระบบไปยังศูนย์ดูแล Microsoft 365 หรือคุณได้รับข้อความที่มีการค้นพบปัญหา ขั้นตอนแรกที่ดีที่สุดคือการ ลองใช้โหมดแบบ inprivate/incognito ของเบราว์เซอร์ของคุณ หลังจากเปิดเป็นหน้าต่างแบบ inprivate/incognito ไปไซต์[admin.microsoft.com](https://admin.microsoft.com)และเข้าสู่ระบบ ด้วยข้อมูลประจำตัวของผู้ดูแลระบบ จากนั้น ดำเนินการเสร็จสิ้นการซื้อของคุณ
