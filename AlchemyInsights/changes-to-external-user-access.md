# <a name="changes-to-external-user-access"></a>การเปลี่ยนแปลงการเข้าถึงของผู้ใช้ภายนอก

ตั้งแต่วันที่ 23 มีนาคม 2018 ผู้ใช้ภายนอกจะไม่สามารถเข้าถึงเนื้อหาที่แชร์กับทุกคน ผู้ใช้ภายนอกจะสามารถเข้าถึงเนื้อหาที่แชร์กับผู้ใช้หรือกับกลุ่มที่ผู้ใช้อยู่เท่านั้น หากคุณต้องการเก็บการตั้งค่าที่มีอยู่ของคุณ โปรดดู:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
