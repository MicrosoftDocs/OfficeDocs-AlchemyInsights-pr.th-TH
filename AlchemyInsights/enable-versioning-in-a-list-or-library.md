# <a name="enable-versioning-for-a-list-or-library"></a>การเปิดใช้งานการกำหนดเวอร์ชันสำหรับรายการหรือไลบรารี

1. ไปที่รายการหรือไลบรารี
    
2. คลิก**การตั้งค่า**แล้วคลิกการตั้ง**ค่าไลบรารี**หรือ**การตั้งค่ารายการ**
    
3. คลิก**การตั้งค่าการกำหนดเวอร์ชัน**
    
4. เลือกการตั้งค่าที่คุณต้องการแล้วคลิก**ตกลง**
