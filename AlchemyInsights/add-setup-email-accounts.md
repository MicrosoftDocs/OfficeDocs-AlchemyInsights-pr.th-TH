# <a name="set-up-email-accounts"></a>ตั้งค่าบัญชีอีเมล

1. ใน Outlook**File** > ให้คลิก**เพิ่มบัญชี****ข้อมูล** > ไฟล์

2. ป้อนที่อยู่อีเมลของคุณ แล้วคลิก**เชื่อมต่อ**

3. ป้อนรหัสผ่านและคลิก**ตกลง**

4. คุณสามารถทําซ้ําขั้นตอนเหล่านี้เพื่อเพิ่มหลายบัญชี

**หมายเหตุ**: บัญชีอีเมลบางบัญชี เช่น Gmail, Yahoo และ iCloud คุณจะต้องตั้งค่าการตรวจสอบสิทธิ์สองปัจจัยเพื่อใช้ Outlook เพื่อส่งและรับอีเมล ดู[เพิ่มบัญชีอีเมลไปยัง Outlook](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b.aspx)สําหรับข้อมูลเพิ่มเติม
