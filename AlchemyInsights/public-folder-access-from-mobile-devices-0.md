# <a name="public-folder-access-from-mobile-devices"></a>การเข้าถึงโฟลเดอร์สาธารณะจากอุปกรณ์โมบาย

โฟลเดอร์'สาธารณะ'จะไม่สามารถเข้าถึงได้จากอุปกรณ์โมบาย
  
ถ้าคุณต้องการมีฟังก์ชันการทำงานคล้ายกับโฟลเดอร์สาธารณะ และการเข้าถึงจากอุปกรณ์โมบาย ดู[ศึกษาเกี่ยวกับ Office 365 กลุ่ม](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2)สำหรับทางเลือกดีกว่า
