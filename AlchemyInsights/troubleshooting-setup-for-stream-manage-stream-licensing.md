# <a name="managing-microsoft-stream-licenses"></a>การจัดการสิทธิ์การใช้งาน Microsoft Stream

**สิ่งสำคัญ:** คุณจำเป็นต้องอยู่ในบทบาทผู้ดูแลระบบผู้เช่าเพื่อดำเนินการตามที่อธิบายไว้ในหัวข้อนี้

คุณจำเป็นต้องเป็นผู้ดูแลระบบส่วนกลางหรือผู้ดูแลระบบการจัดการผู้ใช้ สำหรับข้อมูลเพิ่มเติมให้ดูที่[เกี่ยวกับบทบาทผู้ดูแลระบบ](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles)

คุณสามารถกำหนด[สิทธิ์การใช้งานให้กับบัญชีผู้ใช้ที่มี Office ๓๖๕ PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410)
