# <a name="copy-and-paste-in-outlookcom"></a>คัดลอก และวางใน Outlook.com

ไม่มีปุ่มคัดลอกหรือวางใน Outlook.com แต่คุณสามารถใช้แป้นพิมพ์ลัด:

**บนพีซี:** CTRL + C เพื่อคัดลอก CTRL + X เพื่อตัด และ CTRL + V เพื่อวาง

**บน Mac เป็น:** CMD + C เพื่อคัดลอก CMD + X เพื่อตัด และ CMD + V เพื่อวาง
