# <a name="create-a-new-folder"></a>สร้างโฟลเดอร์ใหม่

1. ภายใต้**โฟลเดอร์**ในบานหน้าต่างนำทาง เลือก**โฟลเดอร์ใหม่** 
    
2. พิมพ์ชื่อสำหรับโฟลเดอร์ แล้วกด Enter
    
เมื่อต้องการเรียนรู้เพิ่มเติม โปรดดู[สร้างโฟลเดอร์ใน Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a)
