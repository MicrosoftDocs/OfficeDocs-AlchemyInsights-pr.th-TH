# <a name="already-paid"></a>ชำระแล้วหรือไม่

ถ้าคุณได้ส่งการชำระเงินเรียบร้อยแล้ว คุณสามารถแจ้งให้เราทราบ:
  
1. ไปหน้าการ[บอกรับเป็นสมาชิก](https://go.microsoft.com/fwlink/p/?linkid=842054)หรือเลือก**การเรียกเก็บเงิน** \> **บอกรับเป็นสมาชิก**
    
2. เลือกการสมัครใช้งาน และในส่วนของการ**เรียกเก็บเงินของฉัน****ดูสูตร**ที่เลือก
    
3. บนหน้า**รายละเอียดรายการ**เลือก**ชำระแล้วหรือไม่?** คุณจะถูกขอให้ใส่ข้อมูลที่เกี่ยวข้องกับการชำระเงินที่ส่ง 
    
4. เมื่อมีการป้อนข้อมูลที่จำเป็นทั้งหมด เลือก**ส่ง**
