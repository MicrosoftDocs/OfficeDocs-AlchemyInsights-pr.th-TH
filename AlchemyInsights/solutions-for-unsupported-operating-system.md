# <a name="solutions-for-unsupported-operating-system"></a>โซลูชั่นสําหรับ "ระบบปฏิบัติการที่ไม่สนับสนุน"

เมื่อคุณพยายามติดตั้ง Microsoft 365 คุณอาจเห็นข้อความแสดงข้อผิดพลาด**ของระบบปฏิบัติการที่ไม่ได้รับการสนับสนุน**[system requirements](https://products.office.com/office-system-requirements)
  
ไมโครซอฟท์ 365 ต้องการ:
  
- พีซี: Windows 10, Windows 8.1, Windows 8, วินเซิร์ฟ 1, วินเซิร์ฟวินเนม 2016, Windows Server 2012 R2, Windows Server 2012 หรือ Windows Server 2008 R2

    หรือ

- Mac OS X 10.10

ถ้าได้รับข้อความนี้ขณะพยายามติดตั้ง Office บนอุปกรณ์เคลื่อนที่หรือ Chromebook ให้ดูที่[วิธีการติดตั้งและเรียกใช้ Microsoft Office บน Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)
