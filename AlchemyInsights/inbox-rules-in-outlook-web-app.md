# <a name="inbox-rules-in-outlook-web-app"></a>กฎของกล่องขาเข้าใน Outlook Web App

1. ในเว็บเบราว์เซอร์ ให้ลงชื่อเข้าใช้ Outlook Web App โดยใช้ URL ที่บุคคลที่จัดการอีเมลสําหรับองค์กรของคุณ ใส่ชื่อผู้ใช้และรหัสผ่านของคุณ แล้วคลิก**ลงชื่อเข้าใช้**
2. ที่ด้านบนของหน้า ให้คลิก**ตัวเลือก>การตั้งค่า**
3. ใน**ตัวเลือก**ให้คลิก**จัดระเบียบอีเมล >กฎกล่องจดหมายเข้า**

สําหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการทํางานกับกฎของกล่องขาเข้า โปรดดู[กฎกล่องขาเข้า ใน Outlook Web App](https://support.office.com/article/inbox-rules-in-outlook-web-app-edea3d17-00c9-434b-b9b7-26ee8d9f5622)
