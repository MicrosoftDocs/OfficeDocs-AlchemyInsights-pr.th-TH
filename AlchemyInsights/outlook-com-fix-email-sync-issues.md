# <a name="fix-outlookcom-email-sync-issues"></a>แก้ไขปัญหาการซิงค์อีเม Outlook.com

ถ้าคุณไม่สามารถส่ง หรือรับอีเมล ตรวจสอบ[สถานะของบริการ](https://go.microsoft.com/fwlink/p/?linkid=837482&amp;clcid=0x409)ครั้งแรก
  
ถ้าคุณจะไม่ได้รับอีเมลจากผู้ส่งเฉพาะ เช่นข้อความการตั้งค่ารหัสผ่านใหม่ ขั้นแรกให้แน่ใจว่า ผู้ส่งที่ไม่ได้อยู่บนของ[บล็อกผู้ส่งและรายการโดเมน](https://outlook.live.com/mail/options/mail/junkEmail/blockedSendersAndDomains)
  
ถ้าผู้ส่งไม่อยู่ในรายการผู้ส่งที่ถูกบล็อกของคุณ คุณจำเป็นต้องติดต่อผู้ส่ง และแจ้งให้ทราบคุณไม่ได้รับอีเมที่บัญชี Outlook.com ของคุณ โปรดดูผู้ส่ง[https://postmaster.live.com](https://go.microsoft.com/fwlink/p/?linkid=872135&amp;clcid=0x409)ซึ่งพวกเขาสามารถค้นหาข้อมูลเกี่ยวกับวิธีปฏิบัติที่ดีที่สุด แนะนำการแก้ไขปัญหาเบื้องต้น และการเชื่อมโยงการติดต่อทีมสนับสนุน Outlook.com ผู้ส่ง
  
ถ้าคุณไม่ได้รับอีเมเลย ดู[Outlook.com การแก้ไขปัญหาการซิงค์อีเมล](https://support.office.com/article/d39e3341-8d79-4bf1-b3c7-ded602233642?wt.mc_id=Office_Outlook_com_Alchemy)
