# <a name="47500-server-busy-please-try-again-later"></a>เซิร์ฟเวอร์ 4.7.500 ว่าง โปรดลองอีกครั้งในภายหลัง

ข้อผิดพลาดนี้เกิดขึ้นเมื่อปริมาณการรับส่งข้อมูลอีเมลจากที่อยู่ IP ของแหล่งที่มาเกินขีดจํากัดตามชื่อเสียง (หรือไม่มีชื่อเสียง) ของที่อยู่ IP ต้นทาง

การบล็อกอีเมลจากที่อยู่ IP ต้นทางจะหมดอายุภายในหนึ่งชั่วโมง ถ้าที่อยู่ IP ของแหล่งที่มาเป็นเซิร์ฟเวอร์อีเมลในสถานที่ที่เป็นของคุณ ให้ตรวจสอบการกําหนดค่าของตัวเชื่อมต่อกระแสจดหมาย ถ้าลักษณะการทํางานยังคงอยู่นานกว่าหนึ่งชั่วโมง ติดต่อฝ่ายสนับสนุนเพื่อขอข้อยกเว้นสําหรับที่อยู่ IP ของแหล่งที่มา
