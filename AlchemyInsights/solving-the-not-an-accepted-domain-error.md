# <a name="got-a-parameter-cannot-be-found-error"></a>มีข้อผิดพลาด ' ไม่พบพารามิเตอร์ ' ใช่หรือไม่

ถ้าคุณได้รับข้อความแสดงข้อผิดพลาด **"ไม่พบพารามิเตอร์ที่ตรงกับชื่อพารามิเตอร์ ' EmailAddresses"** หมายความว่าใช้เวลานานขึ้นสำหรับ Microsoft ๓๖๕เพื่อเสร็จสิ้นการตั้งค่าผู้เช่าของคุณหรือโดเมนแบบกำหนดเองของคุณถ้าคุณเพิ่งเพิ่ม
  
ขั้นตอนการตั้งค่าอาจใช้เวลาถึง4ชั่วโมงในการดำเนินการให้เสร็จสมบูรณ์ รอสักครู่เพื่อให้กระบวนการตั้งค่ามีเวลาให้เสร็จสิ้นแล้วลองอีกครั้ง
