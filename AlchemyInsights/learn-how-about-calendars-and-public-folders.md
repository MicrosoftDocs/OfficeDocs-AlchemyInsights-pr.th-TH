# <a name="learn-about-using-public-folders-in-outlook"></a>เรียนรู้เกี่ยวกับการใช้โฟลเดอร์'สาธารณะ'ใน Outlook

โฟลเดอร์'สาธารณะ'ใน Outlook ให้ยอดเยี่ยมสำหรับการรวบรวม จัดระเบียบ และแบ่งปันข้อมูลเกี่ยวกับหัวข้อเฉพาะหรือโครงการภายในองค์กรของคุณ สำหรับข้อมูลเพิ่มเติม ดู[สร้างและโฟลเดอร์สาธารณะใน Outlook ที่ใช้ร่วมกัน](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97)
