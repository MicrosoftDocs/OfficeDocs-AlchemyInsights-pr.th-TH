# <a name="control-creation-of-microsoft-365-groups"></a>การควบคุมการสร้างกลุ่ม Microsoft 365

โดยค่าเริ่มต้น ผู้ใช้ทั้งหมดภายในบริษัทสามารถสร้างกลุ่มได้ ถ้าจําเป็น คุณสามารถเปลี่ยนแปลงข้อมูลนี้ และกําหนดสิทธิเพื่อสร้างกลุ่มให้กับกลุ่มคนภายในบริษัทของคุณได้ โปรดทราบว่าสิ่งนี้จะมีผลต่อการสร้างกลุ่มจากบริการ Microsoft 365 ใดๆ (เช่น Outlook, Teams, Planner และอื่นๆ)
  
คําแนะนํา:[จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
