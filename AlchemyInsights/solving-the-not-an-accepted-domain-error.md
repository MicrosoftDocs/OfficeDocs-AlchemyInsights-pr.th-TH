# <a name="got-a-parameter-cannot-be-found-error"></a>มีข้อผิดพลาด 'พารามิเตอร์ที่ไม่พบ' หรือไม่

ถ้าคุณได้รับข้อความแสดงข้อผิดพลาด **"ไม่พบพารามิเตอร์ที่ตรงกับชื่อพารามิเตอร์ 'EmailAddresses"** แสดงว่ากําลังใช้เวลานานกว่าสําหรับ Microsoft 365 เพื่อเสร็จสิ้นการตั้งค่าผู้เช่าหรือโดเมนแบบกําหนดเองของคุณถ้าคุณเพิ่งเพิ่มเข้าไป
  
กระบวนการติดตั้งอาจใช้เวลาถึง 4 ชั่วโมงเพื่อให้เสร็จสิ้น รอสักครู่เพื่อให้กระบวนการตั้งค่ามีเวลาที่จะเสร็จสิ้น แล้วลองอีกครั้ง
