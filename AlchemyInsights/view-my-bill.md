# <a name="view-my-bill-or-invoice"></a>ดูใบเรียกเก็บเงินหรือใบแจ้งหนี้ของฉัน

1. ในศูนย์การจัดการ Microsoft 365 ให้ไปที่**Billing** \> หน้า[การเรียกเก็บเงิน&การชําระเงิน](https://go.microsoft.com/fwlink/p/?linkid=848039)

2. เลือกช่วงวันที่จากรายการดรอปดาวน์ แล้วเลือกลําดับที่คุณต้องการดู

3. เมื่อต้องการพิมพ์ใบเรียกเก็บเงิน ให้ดาวน์โหลดใบเรียกเก็บเงินเป็นไฟล์ PDF และคลิก**พิมพ์**

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ดูใบเรียกเก็บเงินหรือใบแจ้งหนี้ของคุณ](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/view-your-bill-or-invoice)
