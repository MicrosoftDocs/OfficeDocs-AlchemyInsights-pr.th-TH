# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>เว็บไซต์สาธารณะแบบออนไลน์ของ SharePoint ได้ถูกหยุด

เว็บไซต์สาธารณะแบบออนไลน์ของ SharePoint ได้ถูกหยุดไว้

ถ้าคุณได้สร้างเว็บไซต์สาธารณะแบบออนไลน์ของ SharePoint คุณยังสามารถดำเนินต่อโดยใช้ถึง 31 มีนาคม 2018 ถ้าคุณต้องการมีเว็บไซต์สาธารณะต่อไป คุณจำเป็นต้องสร้างขึ้นใหม่กับผู้ให้บริการใหม่ เนื้อหาบนเว็บไซต์สาธารณะของคุณแบบออนไลน์ของ SharePoint ถูกลบ

สำหรับข้อมูลเกี่ยวกับการ discontinuation เว็บไซต์สาธารณะ ดู[รายละเอียดเกี่ยวกับกระบวนการ discontinuation คุณลักษณะ](https://go.microsoft.com/fwlink/?linkid=866980)
