# <a name="how-to-share-sharepoint-files-or-folders"></a>วิธีการใช้ร่วมกันแฟ้มหรือโฟลเดอร์ SharePoint

แฟ้มที่คุณเก็บบนไซต์ SharePoint มักจะพร้อมใช้งานสําหรับทุกคนที่มีสิทธิ์ในไซต์ แต่คุณอาจต้องการใช้แฟ้มหรือโฟลเดอร์ที่ระบุร่วมกับผู้ที่ไม่สามารถเข้าถึงไซต์ได้ เมื่อคุณใช้แฟ้มและโฟลเดอร์ร่วมกัน
