# <a name="stuck-on-processing-order"></a>ติดอยู่ในลําดับการประมวลผล

หากคุณเลือกที่จะอัปเกรดการสมัครใช้งาน Microsoft 365 ของคุณ แต่คําสั่งซื้อถูกติดค้างอยู่บน "การประมวลผลคําสั่งซื้อของคุณ" โปรดดูที่[แดชบอร์ดสถานภาพบริการ](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/servicehealth) **SP207374** กระบวนการปรับรุ่นผู้เช่าของคุณจะต้องเตรียมใช้งาน ซึ่งอาจล่าช้า เราคาดว่ากระบวนการจะเสร็จสมบูรณ์ภายใน 24 ชั่วโมง; อย่างไรก็ตาม เราพบกรณีที่หายากที่คําขอเหล่านี้ใช้เวลานานถึง 48 ชั่วโมง โปรดเปิดคําขอรับบริการหากการดําเนินการสั่งซื้อใช้เวลานานกว่า 48 ชั่วโมง
