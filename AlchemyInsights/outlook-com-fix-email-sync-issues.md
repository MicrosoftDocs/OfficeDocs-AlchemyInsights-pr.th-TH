# <a name="fix-outlookcom-email-sync-issues"></a>แก้ไขปัญหาการซิงค์อีเมลOutlook.com

ถ้าคุณไม่สามารถส่งหรือรับอีเมล ก่อนอื่นให้ตรวจสอบสถานะ[การบริการ](https://go.microsoft.com/fwlink/p/?linkid=837482&amp;clcid=0x409)
  
ถ้าคุณไม่ได้รับอีเมลจากผู้ส่งรายใดรายหนึ่ง เช่น ข้อความรีเซ็ตรหัสผ่าน ก่อนอื่นให้ตรวจสอบว่าผู้ส่งไม่ได้อยู่บน[รายชื่อผู้ส่งและโดเมนที่ถูกบล็อก](https://outlook.live.com/mail/options/mail/junkEmail/blockedSendersAndDomains)ของคุณ
  
หากผู้ส่งไม่อยู่ในรายการผู้ส่งที่ถูกบล็อก คุณจะต้องติดต่อผู้ส่งและแจ้งให้ผู้ส่งทราบว่าคุณไม่ได้รับอีเมลที่บัญชีOutlook.comของคุณ โปรดดูผู้ส่งที่[https://postmaster.live.com](https://go.microsoft.com/fwlink/p/?linkid=872135&amp;clcid=0x409)ซึ่งพวกเขาสามารถค้นหาข้อมูลเกี่ยวกับ Outlook.comแนวทางปฏิบัติที่ดีที่สุด
  
หากคุณไม่ได้รับอีเมลเลย ให้ดูที่[แก้ไขปัญหาการซิงค์อีเมล Outlook.com](https://support.office.com/article/d39e3341-8d79-4bf1-b3c7-ded602233642?wt.mc_id=Office_Outlook_com_Alchemy)
