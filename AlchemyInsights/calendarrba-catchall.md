# <a name="create-and-manage-room-mailboxes"></a>สร้างและจัดการกล่องจดหมายของห้อง

เมื่อต้องการสร้างและปรับเปลี่ยนกล่องจดหมายของห้องในศูนย์การจัดการ Exchange ให้ดูที่[การสร้างและจัดการกล่องจดหมายของห้อง](https://docs.microsoft.com/Exchange/recipients/room-mailboxes)

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับกล่องจดหมายของห้องและอุปกรณ์ (หรือที่เรียกว่ากล่องจดหมายของทรัพยากร) ให้ดูที่[กล่องจดหมายของห้องและอุปกรณ์](https://docs.microsoft.com/microsoft-365/admin/manage/room-and-equipment-mailboxes)
