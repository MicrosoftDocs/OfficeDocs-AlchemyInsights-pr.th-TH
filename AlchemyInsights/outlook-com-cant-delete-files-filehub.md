# <a name="cant-delete-files-or-photos-from-files"></a>ไม่สามารถลบไฟล์หรือภาพถ่ายจากไฟล์

ในOutlook.com**ไฟล์**จะแสดงไฟล์และรูปถ่ายที่แนบไปกับข้อความในกล่องจดหมายของคุณ เมื่อต้องการเอาแฟ้มและรูปถ่ายออกจากไฟล์ คุณจะต้องลบข้อความอีเมลที่แนบอยู่ หลังจากลบข้อความแล้ว คุณยังต้องลบข้อความออกจากโฟลเดอร์รายการที่ถูกลบด้วย

เมื่อต้องการเรียนรู้เพิ่มเติม ให้อ่าน[วิธีการลบแฟ้มหรือภาพถ่ายใน Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d?wt.mc_id=Office_Outlook_com_Alchemy)
