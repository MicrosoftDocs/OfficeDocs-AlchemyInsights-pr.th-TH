# <a name="troubleshoot-error-404-file-not-found"></a>แก้ไขปัญหาข้อผิดพลาด 404 ไม่พบแฟ้ม

ข้อผิดพลาด 404 ได้รับเมื่อผู้ใช้พยายามเข้าถึงไซต์หรือแฟ้มใน SharePoint หรือ OneDrive ซึ่งมักเกิดจากไซต์หรือแฟ้มหรือกลุ่มการเปลี่ยนชื่อ ย้าย หรือลบ ตัวอย่างเช่น: ผู้ใช้จะพบข้อผิดพลาด 404 พยายามเข้าถึงคอลเลกชันไซต์ราก และถูกลบ

เมื่อต้องการแก้ไขข้อผิดพลาด 404 สําหรับไซต์ที่ถูกเปลี่ยนชื่อ ย้าย หรือถูกลบ:

สําหรับไซต์คลาสสิกที่มีอยู่ในศูนย์การจัดการแบบคลาสสิก ให้ดูที่[การคืนค่าไซต์คอลเลกชันที่ถูกลบ](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

สําหรับไซต์สมัยใหม่ (การติดต่อสื่อสาร ที่เชื่อมต่อกลุ่ม หรือไซต์อื่นๆ) ที่มีอยู่ในศูนย์การจัดการ SharePoint ใหม่ ให้ดูที่[การดูและคืนค่าไซต์ที่ถูกลบในศูนย์การจัดการ SharePoint ใหม่](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

เมื่อต้องการแก้ไขข้อผิดพลาด 404 สําหรับแฟ้ม (หรือรายการอื่น ๆ) ที่ถูกเปลี่ยนชื่อ ย้าย หรือถูกลบ:

ไปที่ไซต์ SharePoint หรือ OneDrive และดูถังรีไซเคิลจากเนื้อหาของไซต์ ดู[คืนค่ารายการในถังรีไซเคิลของไซต์ SharePoint](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be#ID0EAADAAA=Online)

ถ้าคุณยังคงไม่สามารถค้นหารายการคุณสามารถค้นหาบันทึกการตรวจสอบถ้าเปิดใช้งานการบันทึกดู[ค้นหาบันทึกการตรวจสอบใน Microsoft 365 Security &ศูนย์ปฏิบัติตามกฎระเบียบ](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance?redirectSourcePath=%252fclient%252fsearch-the-audit-log-in-the-office-365-security-compliance-center-0d4d0f35-390b-4518-800e-0c7ec95e946c)
