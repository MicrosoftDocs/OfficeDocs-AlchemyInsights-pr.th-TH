# <a name="calendar-freebusy-issues-in-hybrid-environments"></a>ปฏิทินว่าง/ไม่ว่างปัญหาในสภาพแวดล้อมไฮบริดสลี

ถ้าผู้ใช้มีปัญหาในการดูข้อมูลว่าง/ไม่ว่างปฏิทินในสภาพแวดล้อมไฮบริดสลี สําหรับข้อมูลเพิ่มเติมเกี่ยวกับตัวช่วยสร้างการตั้งค่าคอนฟิกไฮบริดสลี ให้ดูที่[ตัวช่วยสร้างการตั้งค่าคอนฟิกไฮบริดสลี](https://go.microsoft.com/fwlink/p/?linkid=528149)

สําหรับขั้นตอนการแก้ไขปัญหาในสภาพแวดล้อมไฮบริดสลี ให้ดูที่[การแก้ไขปัญหาการปรับใช้ไฮบริดสลี](https://technet.microsoft.com/library/jj659053.aspx)
