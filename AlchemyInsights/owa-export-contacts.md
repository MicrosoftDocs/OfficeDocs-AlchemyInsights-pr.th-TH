# <a name="export-contacts"></a>ส่งออกที่ติดต่อ

1. ที่ด้านล่างของหน้า เลือกไอคอนบุคคลเพื่อไปเพจบุคคล
2. บนแถบเครื่องมือ เลือก**จัดการ** > **ส่งออกที่ติดต่อ** 
3. เลือกที่จะส่งออกผู้ติดต่อหรือเฉพาะผู้ติดต่อทั้งหมดจากโฟลเดอร์ใดโฟลเดอร์หนึ่ง จากนั้น**ส่งออก**
4. ที่ด้านล่างของหน้า เลือก**บันทึก**เพื่อบันทึก "contacts.csv" ในโฟลเดอร์ดาวน์โหลดเริ่มต้น หรือ เลือกลูกศรอยู่ถัดจาก**บันทึก**และเลือก**บันทึกเป็น**

เมื่อต้องการเรียนรู้เพิ่มเติม ดู[ส่งออกผู้ติดต่อจาก Outlook](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f#ID0EAACAAA=Outlook_on_the_web)
