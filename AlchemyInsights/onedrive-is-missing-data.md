# <a name="onedrive-is-missing-data"></a>OneDrive เป็นข้อมูลที่หายไป

ถ้าไฟล์ OneDrive จำนวนมากของคุณได้รับการลบเขียนทับเสียหายหรือติดมัลแวร์เราจะมอบตัวเลือกต่างๆให้กับผู้ใช้เพื่อช่วยในกระบวนการกู้คืน

- [ค้นหาไฟล์ที่สูญหายหรือขาดหายไปใน OneDrive](https://go.microsoft.com/fwlink/?linkid=2125166)
