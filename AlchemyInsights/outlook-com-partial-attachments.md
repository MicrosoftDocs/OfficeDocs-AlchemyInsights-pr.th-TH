# <a name="cant-preview-all-of-an-attachment"></a>ไม่สามารถแสดงตัวอย่างสิ่งที่แนบมาทั้งหมด

เราตระหนักถึงปัญหาที่ลูกค้าบางรายไม่สามารถแสดงตัวอย่างสิ่งที่แนบมาทั้งหมด แก้ปัญหา กรุณาเปิดสิ่งที่แนบมาในหน้าต่างใหม่
