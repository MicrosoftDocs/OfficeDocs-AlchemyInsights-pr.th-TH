# <a name="restore-deleted-files-or-folders-in-onedrive"></a>คืนค่าไฟล์หรือโฟลเดอร์ที่ถูกลบใน OneDrive

ถ้าคุณลบแฟ้มหรือโฟลเดอร์ใน OneDrive โดยบังเอิญ คุณอาจกู้คืนแฟ้มนั้นในภายหลังจากถังรีไซเคิลของ OneDrive

* [คืนค่าไฟล์หรือโฟลเดอร์ที่ถูกลบใน OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)

ถ้าไฟล์ OneDrive จํานวนมากถูกลบ เขียนทับ เสียหาย หรือติดไวรัสโดยมัลแวร์ คุณสามารถคืนค่า OneDrive ทั้งหมดของคุณไปยังเวลาก่อนหน้าได้

* [คืนค่าวันไดรฟ์ของคุณ](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
