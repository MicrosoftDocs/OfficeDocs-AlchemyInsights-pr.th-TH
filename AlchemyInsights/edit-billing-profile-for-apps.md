# <a name="edit-billing-profile-for-apps"></a>แก้ไขโปรไฟล์การเรียกเก็บเงินสำหรับแอป

## <a name="to-change-the-billing-profile-on-apps"></a>เมื่อต้องการเปลี่ยนโปรไฟล์การเรียกเก็บเงินบนแอป

1. ไปที่หน้าการ**เรียกเก็บเงิน**  >  **[ของผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)** จากนั้นเลือกแท็บ**แอป**

2. ค้นหาและเลือกแอปที่คุณต้องการจัดการ  

3. ถัดจาก **โปรไฟล์การเรียกเก็บเงิน**ให้เลือก **แก้ไข** และเลือกโปรไฟล์การเรียกเก็บเงินที่คุณต้องการเชื่อมโยงกับแอป

## <a name="edit-billing-profiles"></a>แก้ไขโปรไฟล์การเรียกเก็บเงิน

1. ไปที่หน้าใบ**เรียกเก็บ**เงิน  >  **[& การชำระ](https://go.microsoft.com/fwlink/p/?linkid=848039)** เงินจากนั้นเลือกโปรไฟล์การ**เรียกเก็บเงิน**

2. เลือกโปรไฟล์การเรียกเก็บเงินจากรายการเพื่อแก้ไข
