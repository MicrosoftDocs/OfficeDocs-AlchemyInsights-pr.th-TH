# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>รับการสนับสนุนในแอปสำหรับแอป Outlook สำหรับอุปกรณ์เคลื่อนที่

1. ในแอป Outlook สำหรับอุปกรณ์เคลื่อนที่ให้เปิด**เมนู**

2. เลือกเครื่องหมายคำถามเพื่อเปิดหน้าจอ** &amp; คำติชมของวิธีใช้**

3. แตะ **ติดต่อฝ่ายสนับสนุน** และอธิบายปัญหาของคุณ

4. แตะ **ส่ง** และคุณจะได้รับการแจ้งเตือนภายในแอปเมื่อผู้ติดต่อฝ่ายสนับสนุนของคุณ

สำหรับคำแนะนำโดยละเอียดให้ดู[ที่รับความช่วยเหลือในแอปสำหรับ Outlook สำหรับ iOS และ Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support)
