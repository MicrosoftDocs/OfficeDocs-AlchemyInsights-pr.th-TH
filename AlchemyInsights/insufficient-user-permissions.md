# <a name="insufficient-user-permissions"></a>สิทธิ์ของผู้ใช้ไม่เพียงพอ

ปัญหานี้เกิดขึ้นเมื่อข้อกําหนด[เบื้องต้นที่จําเป็น](https://docs.microsoft.com/configmgr/tenant-attach/device-sync-actions#prerequisites)จะไม่เป็นไปตาม โปรดตรวจสอบให้แน่ใจว่า ผู้ใช้มีการแจ้งเตือน**ทรัพยากร**สิทธิ์ภายใต้คอลเลกชัน**คลาส**วัตถุในตัวจัดการการตั้งค่าคอนฟิก
