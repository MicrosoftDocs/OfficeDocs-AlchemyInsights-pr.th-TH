# <a name="get-the-microsoft-365-education-plans"></a>รับแผนการศึกษาของ Microsoft 365

ถ้าสถาบันของคุณมีคุณสมบัติสําหรับ Microsoft 365 Education อยู่แล้ว คุณสามารถเพิ่มการสมัครใช้งาน A1, A3 หรือ A5 จาก**ศูนย์การจัดการ Microsoft 365** 

1. ไปที่**ผลิตภัณฑ์>การเรียกเก็บเงิน[& บริการ](https://go.microsoft.com/fwlink/p/?linkid=868433)**

2. ใช้ช่องค้นหาเพื่อค้นหาใน "A1", "A2" หรือ "A3"

3. เลือกจํานวนผู้ใช้ที่จะใช้แผน

หากสถาบันของคุณยังไม่ได้ตั้งค่า Microsoft 365 Education แต่คุณต้องการเพิ่ม Microsoft 365 ฟรี คุณสามารถ[เริ่มต้นใช้งาน](https://www.microsoft.com/education/products/office)ได้โดยให้อีเมลของคุณ

 เปรียบเทียบแผน, ดูแผน[สําหรับนักเรียน](https://www.microsoft.com/microsoft-365/academic/compare-office-365-education-plans?activetab=tab:primaryr1)และ[สําหรับครูของคุณ, คณาจารย์, และเจ้าหน้าที่](https://www.microsoft.com/microsoft-365/academic/compare-office-365-education-plans?activetab=tab:primaryr2).
