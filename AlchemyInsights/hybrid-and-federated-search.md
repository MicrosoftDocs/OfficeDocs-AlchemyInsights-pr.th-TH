# <a name="hybrid-and-federated-searches"></a>การค้นหาไฮบริดและการติดต่อกับภายนอก 

สำหรับข้อมูลเกี่ยวกับไฮบริดสลีและการค้นหาการติดต่อกับภายนอกใน SharePoint แบบออนไลน์ให้ดูการ[ค้นหาไฮบริดสลีใน SharePoint](https://docs.microsoft.com/sharepoint/hybrid/hybrid-search-in-sharepoint)

สำหรับข้อมูลเกี่ยวกับการวางแผนการค้นหาแบบรวมศูนย์ไฮบริดสลีให้ดู[การค้นหาการติดต่อกับภายนอกไฮบริดสลีสำหรับ SharePoint Server](https://docs.microsoft.com/sharepoint/hybrid/plan-hybrid-federated-search)
