# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>ผู้ใช้ภายนอกไม่สามารถส่งอีเมลไปยัง Microsoft ๓๖๕กลุ่มได้

ใช้ขั้นตอนเหล่านี้เพื่ออนุญาตให้ผู้ใช้ภายนอกส่งอีเมลไปยังกลุ่ม Microsoft ๓๖๕ของคุณ:

1. ลงชื่อเข้าใช้[admin center](https://admin.microsoft.com/)  >  **Groups**  >  **กลุ่ม**กลุ่มศูนย์การจัดการ
2. เลือกกลุ่มที่คุณต้องการจัดการเพื่อเปิดบานหน้าต่างการตั้งค่า
3. บนแท็บ**การตั้งค่า**ให้เลือก**อนุญาตให้ผู้ส่งภายนอกส่งอีเมลไปยังกลุ่มนี้**
4. เลือก**บันทึก**</br>
    อนุญาตให้มีการแสดงการตั้งค่าได้มากถึงหนึ่งชั่วโมง
