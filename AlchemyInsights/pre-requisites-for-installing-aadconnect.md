# <a name="pre-requisites-for-installing-aadconnect"></a>ข้อกําหนดเบื้องต้นสําหรับการติดตั้ง AADConnect

ต้องติดตั้งการเชื่อมต่อ AD Azure บน Windows Server 2012 หรือรุ่นที่ใหม่กว่า เซิร์ฟเวอร์นี้ต้องเป็นโดเมนที่เข้าร่วม และอาจเป็นตัวควบคุมโดเมนหรือเซิร์ฟเวอร์สมาชิก  สําหรับรายการเต็มของความต้องการเชื่อมต่อ AD Azure และข้อกําหนดเบื้องต้น[Prerequisites for Azure AD Connect](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-install-prerequisites)
