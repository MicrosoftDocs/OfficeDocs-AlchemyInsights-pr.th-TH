# <a name="control-who-creates-microsoft-365-groups-and-microsoft-teams"></a>ควบคุมว่าใครเป็นผู้สร้างกลุ่ม Microsoft 365 และทีมของ Microsoft

โดยค่าเริ่มต้น ผู้ใช้ทั้งหมดภายในองค์กรสามารถสร้างกลุ่ม Microsoft 365 และทีม Microsoft คุณสามารถเปลี่ยนค่าเริ่มต้นและกําหนดสิทธิ์เพื่อสร้างกลุ่ม Microsoft 365 หรือ Teams เป็นชุดย่อยของผู้ใช้ภายในองค์กรของคุณ การเปลี่ยนแปลงนี้ยังส่งผลต่อการสร้างกลุ่มจากบริการ Microsoft 365 เช่น Outlook, Teams, Planner และอื่นๆ

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365](https://support.office.com/article/Manage-who-can-create-Office-365-Groups-4c46c8cb-17d0-44b5-9776-005fced8e618)และ[Microsoft Teams](https://aka.ms/rtsf)ได้
