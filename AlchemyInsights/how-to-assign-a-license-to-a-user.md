# <a name="assign-or-unassign-licenses-to-users"></a>มอบหมายหรือยกเลิกการมอบหมายสิทธิ์การใช้งานให้กับผู้ใช้

ผู้ใช้ต้องมีสิทธิ์การใช้งานเพื่อเข้าถึงบริการ ต่อไปนี้เป็นขั้นตอน**ในการเพิ่ม เปลี่ยนแปลง หรือเอา**สิทธิ์การใช้งานออกจากผู้ใช้
  
1. ในศูนย์การจัดการ ให้ไปที่**Users** \> [ผู้ใช้ที่ใช้งานอยู่](https://go.microsoft.com/fwlink/p/?linkid=834822)

2. เลือกชื่อของผู้ใช้ที่คุณต้องการมอบหมาย เปลี่ยน หรือเอาสิทธิ์การใช้งานออก

3. คลิก**จัดการสิทธิ์การใช้งานผลิตภัณฑ์**แล้วเลือกสิทธิ์การใช้งานที่คุณต้องการกําหนดให้กับผู้ใช้รายนี้

    โดยค่าเริ่มต้น บริการทั้งหมดที่เกี่ยวข้องกับสิทธิ์การใช้งานนั้นจะถูกกําหนดให้กับผู้ใช้โดยอัตโนมัติ เมื่อต้องการจํากัดบริการที่พร้อมใช้งานสําหรับผู้ใช้ ให้ล้างกล่องกาเครื่องหมายสําหรับบริการที่คุณไม่ต้องการให้ผู้ใช้นั้นมีอยู่

4. ที่ด้านล่างของบานหน้าต่าง ให้คลิก**บันทึกการเปลี่ยนแปลง**

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การกําหนดสิทธิ์การใช้งานให้กับผู้ใช้](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)และ[เอาสิทธิ์การใช้งานออกจากผู้ใช้](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user)
