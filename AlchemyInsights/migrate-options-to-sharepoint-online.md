# <a name="migrate-options-to-sharepoint-online"></a>ย้ายตัวเลือกไปยัง SharePoint แบบออนไลน์

มีตัวเลือกที่แตกต่างกันที่มีอยู่ในการโยกย้ายเนื้อหาไปยัง SharePoint แบบออนไลน์ขึ้นอยู่กับขนาดและปริมาณของแฟ้มที่คุณต้องการย้าย
  
สำหรับข้อมูลเพิ่มเติมโปรดดูที่[การโยกย้ายไปยัง SharePoint แบบออนไลน์](https://go.microsoft.com/fwlink/?linkid-2022029)
