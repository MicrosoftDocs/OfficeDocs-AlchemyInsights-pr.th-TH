# <a name="quick-edit"></a>แก้ไขด่วน

มีสองวิธีในการแก้ไขรายการในรายการ SharePoint คุณสามารถแก้ไขรายการเดียวในมุมมองรายการได้ซึ่งเป็นวิธีการเริ่มต้น หรือทำงานได้อย่างมีประสิทธิภาพยิ่งขึ้นคุณสามารถแก้ไขหลายรายการในมุมมองการแก้ไขด่วนได้ คุณต้องมีสิทธิ์การใช้งานระดับที่ได้รับการสนับสนุนหรือสูงกว่าในการแก้ไขรายการ ดู[แก้ไขรายการใน SharePoint Online](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7)
