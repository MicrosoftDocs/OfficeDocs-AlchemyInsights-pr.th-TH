# <a name="sharepoint-migration-throttling-with-503-errors"></a>มีข้อผิดพลาด 503 การควบคุมปริมาณการโยกย้าย SharePoint

จะปรากฏขึ้นคุณจะย้ายไปยัง SharePoint Online และได้รับข้อผิดพลาด 503 โปรดทำตามขั้นตอนด้านล่างเพื่อให้เราอาจช่วยคุณได้เร็วที่สุด 

1. คลิก**ติดต่อฝ่ายสนับสนุน**และ**ร้องขอการบริการใหม่**
2. สำหรับชื่อเรื่องและคำอธิบาย พิมพ์**SharePoint โยกย้ายการควบคุมปริมาณกับ 503**
3. เมื่อมีการส่งตั๋วโดยสาร กรุณาปรับปรุง ด้วยข้อมูลต่อไปนี้:
    - จำนวนปล่อยของ (ตัวอย่างเช่น จำนวน TBs) การโยกย้าย
    - การย้ายเริ่มต้น และวันสิ้นสุด
    - อธิบายตำแหน่งที่คุณกำลังย้ายเนื้อหาของคุณจาก เช่น SharePoint Server กล่อง GDrive แฟ้มที่ใช้ร่วมกัน เป็นต้น...
    - ประมาณจำนวนของข้อผิดพลาดการควบคุมปริมาณ (ตัวอย่างเช่น x ควบคุมปริมาณต่อชั่วโมงหรือไม่) และเมื่อไม่ได้ควบคุมปริมาณที่ทำ
    - เครื่องมือการโยกย้ายที่คุณกำลังใช้ (เช่น SPMT หรือ ShareGate)
