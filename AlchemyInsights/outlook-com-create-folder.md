# <a name="create-a-folder-in-outlookcom"></a>สร้างโฟลเดอร์ใน Outlook.com

1. ที่ด้านล่างของบานหน้าต่าง'โฟลเดอร์' เลือก**โฟลเดอร์ใหม่** คุณอาจต้องเลื่อนลงในบานหน้าต่างโฟลเดอร์มองเห็นปุ่ม**โฟลเดอร์ใหม่**
2. ในกล่องข้อความ ใส่ชื่อสำหรับโฟลเดอร์ และจากนั้น กด Enter

เมื่อต้องการเรียนรู้เพิ่มเติม โปรดดู[สร้างโฟลเดอร์ใน Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)
