# <a name="550-57708-access-denied-traffic-not-accepted-from-this-ip"></a>550 5.7.708 เข้าปฏิเสธ, การจราจรไม่ยอมรับจาก IP นี้

ข้อผิดพลาดนี้เกิดขึ้นเมื่อมีการส่งอีเมลจากที่อยู่ IP ชื่อเสียงที่รู้จักต่ําซึ่งโดยทั่วไปลูกค้าใหม่ใช้

หากคุณต้องส่งอีเมลจากที่อยู่ IP เหล่านี้ก่อนจึงจะสามารถซื้อสิทธิ์การใช้งาน ได้ โปรดติดต่อฝ่ายสนับสนุนเพื่อขอข้อยกเว้นจนกว่าคุณจะสามารถซื้อสิทธิ์การใช้งานได้
