# <a name="welcome-to-the-microsoft-365-lighthouse-private-preview"></a>ยินดีต้อนรับสู่ตัวอย่างประภาคาร Microsoft 365 ส่วนตัว

คุณมีส่วนร่วมในการแสดงตัวอย่างส่วนตัวของประภาคาร Microsoft 365 ถ้าคุณต้องการให้ข้อเสนอแนะเกี่ยวกับ Microsoft 365 Lighthouse ให้โพสต์รายละเอียดลงใน [ไซต์ User Voice](https://aka.ms/M365Lighthouseuservoice)ของเรา หากคุณมีข้อสงสัยหรือประสบปัญหาขณะใช้ประภาคาร Microsoft 365 โปรดสร้างคําขอรับการสนับสนุนเพื่อขอความช่วยเหลือเพิ่มเติม
