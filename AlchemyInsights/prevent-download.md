# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>ป้องกันแฟ้มจากการถูกดาวน์โหลดจาก SharePoint หรือ OneDrive

มีวิธีการต่างๆที่สามารถใช้เพื่อป้องกันหรือบล็อกการดาวน์โหลดเกิดขึ้น โปรดไปที่ลิงก์ด้านล่างสำหรับวิธีการที่เกี่ยวข้อง:

- [บล็อกการดาวน์โหลดแฟ้มด้วยการเข้าถึงแบบมีเงื่อนไข Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [ควบคุมการเข้าถึงจากอุปกรณ์ที่ไม่มีการจัดการ](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [การกำหนดค่านโยบาย IRM บนไลบรารีเอกสาร SharePoint เพื่อจำกัดการดาวน์โหลดเนื้อหา](https://docs.microsoft.com/office365/securitycompliance/set-up-irm-in-sp-admin-center)
