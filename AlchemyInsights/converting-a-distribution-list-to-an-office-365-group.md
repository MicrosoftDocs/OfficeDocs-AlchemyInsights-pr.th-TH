# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>แปลงกลุ่มการแจกจ่ายเป็นกลุ่ม Microsoft 365

คุณสามารถแปลงกลุ่มการแจกจ่ายเป็น Microsoft 365 กลุ่มได้หากพวกเขาเป็นไปตามเงื่อนไขที่กําหนด เมื่อต้องการแปลงกลุ่มหรือแก้ไขปัญหาการแปลง ให้ดูที่[การปรับรุ่นรายการการแจกจ่ายเป็น Microsoft 365 Groups ใน Outlook](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists)
