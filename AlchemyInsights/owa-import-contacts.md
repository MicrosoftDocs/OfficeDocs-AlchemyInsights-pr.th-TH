# <a name="import-contacts"></a>นำเข้าผู้ติดต่อ

ก่อนที่จะนำเข้าผู้ติดต่อไปยัง Outlook บนเว็บ คุณจำเป็นต้องส่งออกจากผู้อื่นทางอีเมลให้บริการเป็นแฟ้ม.csv จากนั้น ทำตามคำแนะนำด้านล่างนี้

1. ที่ด้านล่างของหน้า เลือกไอคอนบุคคลเพื่อไปเพจบุคคล
2. บนแถบเครื่องมือ เลือก**จัดการ** > **นำเข้าผู้ติดต่อ**
3. เลือก**เรียกดู**เลือกแฟ้ม CSV ของคุณ และ**จากนั้น เลือก**
4. เลือกการ**นำเข้า**

เมื่อต้องการเรียนรู้เพิ่มเติม ดู[นำเข้าผู้ติดต่อไปยัง Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web)
