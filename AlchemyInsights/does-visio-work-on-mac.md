# <a name="does-visio-work-on-a-mac"></a>Visio ทำงานบน Mac ได้หรือไม่

ไม่มีแอปพลิเคชัน Visio บนเดสก์ท็อปสำหรับ Mac อย่างไรก็ตามตราบใดที่ผู้ใช้ที่จำเป็นต้องใช้แอปจะมี [สิทธิ์การใช้งาน Visio online](https://docs.microsoft.com/microsoft-365/admin/add-users/add-users)ที่ได้รับมอบหมายพวกเขาสามารถ [ดูสร้างและแก้ไขไดอะแกรมใน Visio Online](https://support.office.com/article/06f04845-91b8-4e8f-881f-a43c970735fc?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1799) ในเว็บเบราว์เซอร์ได้
