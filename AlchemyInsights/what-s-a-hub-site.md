# <a name="whats-a-hub-site"></a>ไซต์ฮับคืออะไร

ไซต์ฮับจะช่วยคุณจัดระเบียบไซต์และให้การนำทางและการใช้ตราสินค้าทั่วไป ผู้ดูแลระบบส่วนกลางหรือผู้ดูแลระบบ SharePoint จะสามารถสร้างไซต์ฮับและระบุว่าผู้ใช้ใดบ้างที่สามารถเชื่อมโยงไซต์ของพวกเขากับไซต์ฮับได้ สำหรับข้อมูลเพิ่มเติมให้ดูที่[ไซต์ SharePoint hub](https://go.microsoft.com/fwlink/?linkid=869388)
