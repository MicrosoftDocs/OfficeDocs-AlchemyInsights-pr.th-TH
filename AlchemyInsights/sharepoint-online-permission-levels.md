# <a name="sharepoint-online-permission-levels"></a>ระดับสิทธิ์ของ SharePoint Online

SharePoint มาพร้อมกับระดับสิทธิ์เริ่มต้นที่คุณสามารถใช้ได้เช่นการสนับสนุนและการดูเท่านั้น ถ้าพวกเขาไม่ตรงกับความต้องการของคุณคุณสามารถสร้างระดับสิทธิ์เพิ่มเติมได้ วิธีที่ง่ายที่สุดในการติดตามสิทธิ์ถ้าคุณไม่สามารถแก้ไขหรือลบระดับเริ่มต้นและถ้าคุณเพิ่มบุคคลลงในกลุ่มและกำหนดระดับสิทธิ์ให้กับกลุ่มเหล่านั้นแทนที่จะกำหนดระดับสิทธิ์ให้กับผู้ใช้แต่ละราย
  
- [การทำความเข้าใจเกี่ยวกับระดับสิทธิ์](https://go.microsoft.com/fwlink/?linkid=867071)
    
- [วิธีการสร้างและแก้ไขระดับสิทธิ์](https://go.microsoft.com/fwlink/?linkid=867072)
