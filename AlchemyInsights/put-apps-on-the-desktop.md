# <a name="put-apps-on-the-desktop"></a>วางแอปบนเดสก์ท็อป

ค้นหาแอปในเมนู **เริ่ม** แล้วคว้าไอคอนแอปได้ด้วยการกดปุ่มเมาส์ซ้ายค้างไว้และย้ายไอคอนไปยังเดสก์ท็อป

นอกจากนี้คุณยังสามารถกด **แป้น Windows + R** แล้วพิมพ์คำสั่งนี้:

`shell:appsfolder`

การทำเช่นนี้จะเปิดโฟลเดอร์**แอปพลิเคชัน**ใน**File Explorer** ในตอนนี้คุณสามารถลากแล้วปล่อยไอคอนแอปไปยังเดสก์ท็อป
