# <a name="create-and-use-a-shared-mailbox"></a>การสร้างและใช้กล่องจดหมายที่ใช้ร่วมกัน

กล่องจดหมายที่ใช้ร่วมกันช่วยให้บุคคลมากกว่าหนึ่งคนส่งและรับเมลแทนสำหรับนามแฝงเช่น**sales@contoso.com**โดยไม่ต้องกำหนดสิทธิ์การใช้งานอื่น
  
[สร้างกล่องจดหมายที่ใช้ร่วมกัน](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)อธิบายวิธีการ: 
  
- การสร้างกล่องจดหมายที่ใช้ร่วมกัน
    
- การให้สิทธิ์ผู้ใช้ของคุณในการใช้กล่องจดหมายที่ใช้ร่วมกัน
    
- เข้าถึงกล่องจดหมายที่ใช้ร่วมกันใน Outlook หรือใน Outlook บนเว็บ
