# <a name="change-your-password"></a>เปลี่ยนรหัสผ่านของคุณ

รหัสผ่านของคุณ Outlook.com จะเหมือนกับรหัสผ่านของบัญชี Microsoft ของคุณ
  
เมื่อต้องการเปลี่ยนรหัสผ่านของคุณ ไป[https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409)และเลือก**เปลี่ยนรหัสผ่าน** 
  
ถ้าคุณลืมรหัสผ่านของคุณ ไปที่[บัญชี Microsoft ที่หน้าการตั้งค่ารหัสผ่านใหม่](https://go.microsoft.com/fwlink/p/?linkid=841909)
  
เมื่อต้องการเรียนรู้เพิ่มเติม ดู[การเปลี่ยนแปลงรหัสผ่านของคุณใน Outlook.com](https://go.microsoft.com/fwlink/?linkid=873109)
