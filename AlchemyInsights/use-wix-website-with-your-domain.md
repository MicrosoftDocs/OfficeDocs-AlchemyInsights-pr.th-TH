# <a name="using-wix-website-with-office-365-purchased-or-managed-domains"></a>การใช้เว็บไซต์ Wix กับ Office ๓๖๕ที่ซื้อหรือโดเมนที่มีการจัดการ

- [อัปเดตระเบียน DNS เพื่อเก็บเว็บไซต์ของคุณไว้กับผู้ให้บริการโฮสต์ปัจจุบันของคุณ](https://docs.microsoft.com/microsoft-365/admin/dns/update-dns-records-to-retain-current-hosting-provider)
- บทความ Wix "การเชื่อมต่อโดเมนกับ Wix โดยใช้วิธีการชี้" แนะนำให้ใช้การชี้ (การเพิ่มระเบียน DNS ต่อลิงก์ด้านบน) แทนการเปลี่ยนเซิร์ฟเวอร์ชื่อเมื่อใช้ Office ๓๖๕
- ถ้าคุณยังคงเลือกที่จะเปลี่ยนชื่อเซิร์ฟเวอร์เป็น Wix คุณจะต้อง  [สร้างระเบียน DNS ที่ Wix สำหรับ Microsoft](https://docs.microsoft.com/microsoft-365/admin/dns/create-dns-records-at-wix?view=o365-worldwide)
- ถ้าโดเมนของคุณถูกซื้อจาก Microsoft ชื่อเซิร์ฟเวอร์ไม่สามารถเปลี่ยนแปลงได้ ถ้าคุณจำเป็นต้องเปลี่ยนชื่อเซิร์ฟเวอร์ที่ Microsoft ซื้อโดเมนจะต้องถูก  [โอนไปยังผู้ให้บริการโฮสต์อื่นหลังจาก๖๐วัน](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/transfer-a-domain-from-microsoft-to-another-host)
