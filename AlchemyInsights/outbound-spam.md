# <a name="outbound-spam"></a>สแปมขาออก

หากคุณเห็นข้อความแสดงข้อผิดพลาดที่เป็นเท็จสําหรับสแปมขาออก (ข้อความขาออกที่ดีถูกทําเครื่องหมายว่าเป็นสแปม) ให้เปิดตั๋วสนับสนุนที่มีข้อความตัวอย่าง
