# <a name="permissions-and-public-folders"></a>สิทธิ์และโฟลเดอร์สาธารณะ

คุณสามารถเปลี่ยนสิทธิ์บนโฟลเดอร์สาธารณะของคุณได้โดยใช้ Outlook, ศูนย์การจัดการ Exchange (EAC) หรือ PowerShell:
  
- สำหรับคำแนะนำใน Outlook ให้[คลิกที่นี่](https://support.office.com/article/Set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005.aspx)
    
- สำหรับ EAC ให้ดู [บทความนี้](https://technet.microsoft.com/library/jj651147%28v=exchg.150%29.aspx.aspx#Anchor_1) สำหรับคำแนะนำ 
    
- สำหรับ Powershell ให้ดู [บทความนี้](https://technet.microsoft.com/library/bb124743%28v=exchg.160%29.aspx.aspx) สำหรับคำแนะนำเกี่ยวกับการใช้ PublicFolderClientPermission commandlet ถ้าคุณต้องการคำแนะนำในการเชื่อมต่อกับ Exchange Powershell ให้คลิก[ที่นี่](https://technet.microsoft.com/library/jj984289%28v=exchg.160%29.aspx.aspx)
    
ถ้า **ผู้ใช้ภายนอกไม่สามารถส่งอีเมลไปยังโฟลเดอร์สาธารณะที่เปิดใช้งานจดหมาย**ได้เหตุผลอาจเป็นไปได้ว่าโฟลเดอร์สาธารณะจะไม่มีสิทธิ์ที่จำเป็นสำหรับการส่งอีเมลภายนอก คุณสามารถแก้ไขปัญหานี้ได้โดยใช้คำแนะนำของ Outlook[ที่นี่](https://technet.microsoft.com/library/aa997560%28v=exchg.150%29.aspx.aspx#Anchor_1)หรือคำแนะนำ PowerShell[ที่นี่](https://support.microsoft.com/help/2984402/-5.7.1-smtp-550-5.7.1-resolver.rst.authrequired-nondelivery-report-when-external-users-try-to-send-mail-to-mail-enabled-public-folders-in-office-365.aspx)
