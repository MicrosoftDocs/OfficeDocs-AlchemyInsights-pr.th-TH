# <a name="how-permissions-inheritance-works-in-sharepoint"></a>วิธีการทำงานของการสืบทอดสิทธิ์ใน SharePoint

โดยค่าเริ่มต้น สิทธิ์ใน SharePoint จะถูกสืบทอดจากค่าที่สูงกว่าในลำดับชั้น ดังนั้น แฟ้มสืบทอดสิทธิ์จากโฟลเดอร์ ซึ่งสืบทอดสิทธิ์จากไลบรารี ที่สืบทอดสิทธิ์จากไซต์ ซึ่งสืบทอดสิทธิ์จากไซต์คอลเลกชัน
  
สำหรับข้อมูลเกี่ยวกับการเอาสิทธิ์เฉพาะ และคืนค่าการสืบทอด ดู[แก้ไข และจัดการสิทธิ์สำหรับรายการหรือไลบรารี](https://go.microsoft.com/fwlink/?linkid=869946)
