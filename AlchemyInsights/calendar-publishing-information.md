# <a name="calendar-publishing-information"></a>ข้อมูลการประกาศปฏิทิน

- ผู้ดูแลระบบ Microsoft 365 สามารถเปิดหรือปิดใช้งานปฏิทินและการเผยแพร่ปฏิทินร่วมกันสําหรับองค์กร สําหรับข้อมูลเพิ่มเติม ให้ดูที่[เปิดการประกาศปฏิทินเพื่อให้ผู้อื่นสามารถใช้ข้อมูลร่วมกัน](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)และ[แชร์ปฏิทินกับผู้ใช้ภายนอก](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users)

- ผู้ใช้ Outlook สามารถแชร์ปฏิทินของตนเองได้ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การใช้ปฏิทิน Outlook ร่วมกับบุคคลอื่น](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
