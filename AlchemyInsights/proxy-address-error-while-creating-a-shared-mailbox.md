# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>ข้อผิดพลาดที่อยู่พร็อกซีขณะกำลังสร้างกล่องจดหมายที่แชร์

ถ้าคุณพยายามสร้างกล่องจดหมายที่แชร์และได้รับข้อผิดพลาดนี้ "ที่อยู่พร็อกซี" SMTP:alias@domain.com "ถูกใช้อยู่แล้ว ...", ที่อยู่อีเมลที่คุณเลือกจะถูกนำไปใช้โดยกล่องจดหมายอื่นในองค์กรของคุณแล้ว
  
คุณจำเป็นต้องค้นหาผู้ใช้กลุ่มหรือกล่องจดหมายที่แชร์ที่มีที่อยู่อีเมลนี้แล้วลบหรือเปลี่ยนที่อยู่อีเมลของผู้ใช้ จากนั้นคุณสามารถสร้างกล่องจดหมายที่แชร์ด้วยที่อยู่อีเมลที่ว่างได้ ใช้การค้นหาบนโฮมเพจเพื่อค้นหา
  
ถ้าคุณไม่ต้องการลบอีเมลแอดเดรสที่มีอยู่ให้เลือกที่อยู่อีเมลใหม่สำหรับกล่องจดหมายที่แชร์ของคุณ
