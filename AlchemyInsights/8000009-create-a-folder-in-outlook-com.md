# <a name="create-a-folder"></a>การสร้างโฟลเดอร์

1. ในบานหน้าต่างด้านซ้าย เลือก**สร้างโฟลเดอร์ใหม่**ที่อยู่ถัดจาก**โฟลเดอร์** 
    
2. ในกล่องโฟลเดอร์ใหม่ ใส่ชื่อสำหรับโฟลเดอร์ แล้วกด Enter
    
เมื่อต้องการเรียนรู้เพิ่มเติม โปรดดู[สร้างโฟลเดอร์ใน Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114)
