# <a name="change-your-profile-name"></a>เปลี่ยนชื่อส่วนกำหนดค่าของคุณ

1. ไปที่[ข้อมูลบัญชี Microsoft ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=860841)
    
2. เลือก**แก้ไขชื่อ** 
    
3. ใส่ชื่อแรกและชื่อสุดท้าย และเลือก**บันทึก** 
    
โปรดรอจนถึง 24 ชั่วโมงสำหรับการเปลี่ยนแปลงชื่อส่วนกำหนดค่าของคุณมีผลบังคับใช้
  
เมื่อต้องการเรียนรู้เพิ่มเติม ดู[เปลี่ยนรูปถ่ายส่วนกำหนดค่าของคุณหรือชื่อใน Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110)
