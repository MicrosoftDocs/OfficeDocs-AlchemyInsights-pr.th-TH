# <a name="creating-email-signatures"></a>การสร้างลายเซ็นอีเมล

บทความเหล่านี้จะช่วยให้คุณสร้างลายเซ็นอีเมลใน Outlook
  
- ฉันต้องการสร้างลายเซ็นอีเมลโดยใช้ Outlook บน Windows:[สร้างและเพิ่มลายเซ็นลงในข้อความ](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)
  
- ฉันต้องการสร้างลายเซ็นอีเมลโดยใช้ Outlook บนเว็บ:[สร้างและเพิ่มลายเซ็นอีเมลใน Outlook บนเว็บ](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)

- ฉันต้องการสร้างลายเซ็นอีเมลโดยใช้Outlook.com:[สร้างและเพิ่มลายเซ็นอีเมลในOutlook.com](https://support.office.com/article/776d9006-abdf-444e-b5b7-a61821dff034.aspx)

- ฉันต้องการสร้างลายเซ็นอีเมลสากลสําหรับอีเมลทั้งหมดที่ส่งจากบริษัทของฉัน:[สร้างลายเซ็นและข้อจํากัดความรับผิดชอบทั่วทั้งองค์กร](https://docs.microsoft.com/microsoft-365/admin/setup/create-signatures-and-disclaimers)

 **เคล็ดลับ:** ลายเซ็นทั้งองค์กรจะเรียกว่า "ข้อจํากัดความรับผิดชอบ"
