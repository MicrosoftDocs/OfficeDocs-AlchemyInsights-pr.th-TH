# <a name="check-in-several-files-at-once"></a>เช็คอินหลายไฟล์พร้อมกัน

การตรวจสอบแฟ้มเข้าและออกจากไลบรารี SharePoint เป็นวิธีที่ดีในการติดตามและควบคุมวิธีการปรับปรุงเนื้อหา ถ้าคุณต้องการทำการเปลี่ยนแปลงแฟ้มบนไซต์และคุณต้องการให้แน่ใจว่าไม่มีใครสามารถแก้ไขได้ให้ตรวจสอบแฟ้มนั้น เมื่อคุณได้เช็คเอาท์แฟ้มแล้วคุณสามารถแก้ไขได้แบบออนไลน์หรือออฟไลน์และบันทึกหลายครั้งถ้าจำเป็น

ด้านล่างมีลิงก์ที่เป็นประโยชน์เกี่ยวกับคุณลักษณะนี้

- [วิธีการเช็คเอาท์เช็คอินหรือละทิ้งการเปลี่ยนแปลงไฟล์](https://support.office.com/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de)

- [คำถามยอดนิยมเกี่ยวกับการเช็คเอาท์เช็คอินและเวอร์ชัน](https://support.office.com/article/Top-questions-about-check-out-check-in-and-versions-7E941339-E972-4C7A-A79A-80A1FCF84076)

- [การตรวจสอบในหลายไฟล์พร้อมกัน](https://support.office.com/article/check-out-check-in-or-discard-changes-to-files-in-a-library-7e2c12a9-a874-4393-9511-1378a700f6de)

- [เปิดใช้งานหรือปิดใช้งานการกำหนดรุ่น/เช็คเอาท์ในไลบรารี](https://support.office.com/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37)
