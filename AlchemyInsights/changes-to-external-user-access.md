# <a name="changes-to-external-user-access"></a>การเปลี่ยนแปลงการเข้าถึงของผู้ใช้ภายนอก

เริ่มต้นวันที่23มีนาคม๒๐๑๘ผู้ใช้ภายนอกจะไม่สามารถเข้าถึงเนื้อหาที่ถูกแชร์กับทุกคนผู้ใช้ที่ได้รับการรับรองความถูกต้องทั้งหมดหรือผู้ใช้แบบฟอร์มทั้งหมดได้แม้ว่าพวกเขาจะได้รับเชิญหรือเพิ่มไปยังกลุ่มที่ได้รับอนุญาตแล้ว ผู้ใช้ภายนอกจะสามารถเข้าถึงเนื้อหาที่ถูกแชร์กับผู้ใช้หรือกลุ่มที่ผู้ใช้เป็นสมาชิกได้เท่านั้น ถ้าคุณต้องการเก็บการตั้งค่าที่มีอยู่ของคุณโปรดดู: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
