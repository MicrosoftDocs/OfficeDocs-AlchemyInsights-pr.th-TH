# <a name="changing-public-folder-permissions"></a>การเปลี่ยนแปลงการอนุญาตสำหรับโฟลเดอร์สาธารณะ

ผู้ใช้ของคุณและคุณสามารถเปลี่ยนการอนุญาตสำหรับโฟลเดอร์สาธารณะใน Outlook ดู[ตั้งค่า หรือเปลี่ยนแปลงสิทธิ์สำหรับโฟลเดอร์สาธารณะ](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005) ผู้ดูแลยังสามารถเปลี่ยนแปลงสิทธิ์ในศูนย์ดูแล Exchange:
  
1.  ใน Microsoft 365 admin ศูนย์ ไปยัง**ศูนย์ Admin** \> **แลกเปลี่ยน**
    
2. เลือก**โฟลเดอร์สาธารณะ**
    
จากที่นี่ คุณสามารถเปลี่ยนสิทธิ์สำหรับแต่ละโฟลเดอร์สาธารณะ โดยกำหนดสิทธิ์ผู้ใช้ของกลุ่มความปลอดภัย เมื่อต้องการเปลี่ยนแปลงสิทธิ์สำหรับโฟลเดอร์สาธารณะ คุณต้องเป็นเจ้าของ
