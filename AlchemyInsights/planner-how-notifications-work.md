# <a name="planner-how-notifications-work"></a>โปรแกรมวางแผน: การแจ้งเตือนทำงานอย่างไร

โปรแกรมวางแผนของ Microsoft สามารถตั้งค่าการแจ้งเตือนทางอีเมลเช่นเดียวกับการแจ้งเตือนแอพบนมือถือ

- สำหรับข้อมูลเกี่ยวกับประเภทของการแจ้งเตือนสมาชิกแผนหรือเจ้าของกลุ่มจะได้รับโปรดดูที่[อยู่ด้านบนของงานและวางแผนด้วยอีเมลและการแจ้งเตือน](https://support.office.com/article/Stay-on-top-of-tasks-and-plans-with-email-and-notifications-cce223d6-b0ae-43cf-a080-266e2414a859)

- เมื่อต้องการเปลี่ยนการตั้งค่าการแจ้งเตือนให้ตรวจสอบ[เจ้าของกลุ่ม: เปิดหรือปิดอีเมลที่เป็นกลุ่มเกี่ยวกับกิจกรรมงานในแผน](https://support.office.com/article/group-owners-turn-group-emails-about-task-activity-on-or-off-for-a-plan-f1b0d681-2aa6-4ce5-9703-4614607d4cd0)และการ[วางแผน: จัดการอีเมลของกลุ่มสำหรับแผนการวางแผนของคุณ](https://support.office.com/article/plan-members-manage-group-emails-for-your-planner-plans-46f989a0-a34d-4db9-993b-dd596af7a5d2)

- ตรวจสอบ[วิธีการทำงานของโปรแกรมวางแผน Microsoft](https://techcommunity.microsoft.com/t5/planner-blog/how-microsoft-planner-works/ba-p/1214736#M703)สำหรับสถานการณ์ที่จะป้องกันไม่ให้สมาชิกของแผนการรับการแจ้งเตือน
