# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>ใช้เงื่อนไขหรือสูตรขั้นสูงในขั้นตอนการ

ในขั้นตอนของ Microsoft คุณสามารถกำหนดขั้นตอนการเพื่อที่จะทำงานก็ต่อเมื่อเงื่อนไขเป็นจริง สำหรับข้อมูล ดูที่การ[เพิ่มเงื่อนไขในการเป็นขั้นตอน](https://go.microsoft.com/fwlink/?linkid=872112)
  
เมื่อต้องการสร้างเป็นสูตรขั้นสูง ดู[เค้าร่างภาษาข้อกำหนดเวิร์กโฟลว์สำหรับการตรรกะ Azure apps](https://aka.ms/logicexpressions)
