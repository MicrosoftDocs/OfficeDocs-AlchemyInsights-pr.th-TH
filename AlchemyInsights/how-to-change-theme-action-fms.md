# <a name="how-to-change-your-theme"></a>วิธีการเปลี่ยนชุดรูปแบบของคุณ

เมื่อต้องการเปลี่ยนชุดรูปแบบของคุณใน Outlook.com หรือ Outlook บนเว็บ ไปที่การ**ตั้งค่า**และเลือกชุดรูปแบบภายใต้**ชุดรูปแบบ** เมื่อต้องการดูชุดรูปแบบที่พร้อมใช้งานทั้งหมด เลือก**มุมมองทั้งหมด**ภายใต้**ชุดรูปแบบ**
