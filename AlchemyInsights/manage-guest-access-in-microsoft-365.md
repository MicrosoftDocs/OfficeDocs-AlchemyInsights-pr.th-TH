# <a name="manage-guest-access-in-microsoft-365"></a>จัดการการเข้าถึงของผู้เยี่ยมชมใน Microsoft ๓๖๕

สำหรับคำแนะนำที่สมบูรณ์ในการตั้งค่าการแชร์ของผู้เยี่ยมชมในกลุ่ม SharePoint และทีมให้ดูที่: 

- ทำ[งานร่วมกันกับผู้เยี่ยมชมในทีม](https://docs.microsoft.com/microsoft-365/solutions/collaborate-as-team?view=o365-worldwide)โดยใช้**Microsoft ทีม** 

- ทำงาน[ร่วมกันกับผู้เยี่ยมชมในไซต์](https://docs.microsoft.com/microsoft-365/solutions/collaborate-in-site?view=o365-worldwide)โดยใช้**SharePoint** 

- ทำงาน[ร่วมกันกับผู้เยี่ยมชมในกลุ่มภายนอกของคุณ](https://docs.microsoft.com/yammer/work-with-external-users/create-and-manage-external-groups?redirectSourcePath=%252farticle%252f9ccd15ce-0efc-4dc1-81bc-4a424ab6f92a.aspx)โดยใช้**Yammer** 

- [จัดการการเข้าถึงของผู้เยี่ยมชม](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-guest-access-in-groups?view=o365-worldwide)ใน**กลุ่ม**
