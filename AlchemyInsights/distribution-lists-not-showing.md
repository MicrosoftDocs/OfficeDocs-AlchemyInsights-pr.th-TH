# <a name="distribution-group-not-showing-in-groups-list"></a>กลุ่มการแจกจ่ายที่ไม่แสดงในรายการกลุ่ม

อาจใช้เวลาถึง๖๐นาทีก่อนที่รายการการแจกจ่ายใหม่ของคุณจะปรากฏในศูนย์การจัดการ Exchange ศูนย์การจัดการ Exchange
  
ถ้าคุณต้องการเพิ่มสมาชิกหรือทำการแก้ไขอื่นๆในตอนนี้โปรดใช้ฟีเจอร์กลุ่มใน[ศูนย์การจัดการ Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)
