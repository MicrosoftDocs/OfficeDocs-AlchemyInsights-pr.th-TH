# <a name="how-to-delete-a-folder"></a>วิธีการลบโฟลเดอร์

1. ในบานหน้าต่างโฟลเดอร์ คลิกขวาโฟลเดอร์หรือโฟลเดอร์ย่อยที่คุณต้องการลบ
2. เลือก**ลบโฟลเดอร์**
3. เลือก**ตกลง**เพื่อยืนยัน
