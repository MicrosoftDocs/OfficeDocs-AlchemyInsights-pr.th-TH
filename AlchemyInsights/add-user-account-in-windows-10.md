# <a name="add-a-user-account-in-windows-10"></a>เพิ่มบัญชีผู้ใช้ใน Windows 10

ต่อไปนี้เป็นวิธีการเพิ่มบัญชีผู้ใช้ใน Windows 10:

1. ไปที่**เริ่ม**  >  **การตั้งค่า**  >  **บัญชี**ผู้ใช้ของ  >  **ครอบครัว & บุคคลอื่น**(หรือ**บุคคลอื่น**ถ้าคุณกำลังใช้ Windows 10 Enterprise) >**เพิ่มบุคคลอื่นลงในพีซีเครื่องนี้**
2. ทำตามคำแนะนำเพื่อเพิ่มบัญชี Microsoft หรือบัญชีผู้ใช้ภายในเครื่อง

[เรียนรู้เพิ่มเติมเกี่ยวกับการตั้งค่าบัญชีผู้ใช้](https://support.microsoft.com/help/17197/)
