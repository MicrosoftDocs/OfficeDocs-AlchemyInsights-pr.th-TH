# <a name="activating-windows-10-subscriptions"></a>การเปิดใช้งานการสมัครใช้งาน Windows 10

Microsoft 365 ธุรกิจ การศึกษา และองค์กรให้สิทธิ์การใช้งานดิจิทัลสําหรับ Windows 10 เป็นส่วนหนึ่งของการสมัครใช้งาน เมื่อต้องการเปิดใช้งานการสมัครใช้งาน อุปกรณ์ของคุณต้องเปิดใช้งาน Windows 10 Pro (1703 หรือรุ่นที่ใหม่กว่า) สําหรับรายชื่อทั้งหมดของข้อกําหนดสําหรับการเปิดใช้งานการสมัครใช้งาน Windows 10 ให้ดูที่[ข้อกําหนดการเปิดใช้งานการสมัครใช้งาน Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements)
