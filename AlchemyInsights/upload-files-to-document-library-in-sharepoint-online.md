# <a name="upload-files-to-onedrive-or-sharepoint"></a>อัปโหลดแฟ้มไปยัง OneDrive หรือ SharePoint

คุณสามารถอัปโหลดแฟ้มไปยังไลบรารีเอกสารใน SharePoint แบบออนไลน์โดยเพียงแค่ลากไฟล์เหล่านั้นจากคอมพิวเตอร์ของคุณและปล่อยลงในไลบรารีเอกสาร ด้วย OneDrive คุณสามารถซิงค์ไฟล์ระหว่างคอมพิวเตอร์ของคุณกับคลาวด์เพื่อให้คุณสามารถเข้าถึงไฟล์ของคุณได้จากทุกที่ นอกจากนี้คุณยังสามารถทำงานกับไฟล์ที่ซิงค์โดยตรงใน File Explorer และเข้าถึงไฟล์ของคุณแม้ว่าคุณจะออฟไลน์ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการอัพโหลดและซิงค์โปรดไปที่ลิงก์ด้านล่าง
