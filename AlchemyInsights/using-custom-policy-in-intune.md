# <a name="using-custom-policy-in-intune"></a>การใช้นโยบายแบบกําหนดเองใน Intune

Intune มีการตั้งค่ามากมายในตัวสําหรับอุปกรณ์ Android, iOS, macOS และ Windows คุณยังสามารถสร้างโปรไฟล์แบบกําหนดเอง แล้วปรับใช้การตั้งค่าแบบกําหนดเองเหล่านี้ไปยังอุปกรณ์ของคุณโดยใช้ Intune โปรไฟล์ที่กําหนดเองได้รับการออกแบบมาเพื่อเพิ่มการตั้งค่าอุปกรณ์และคุณลักษณะที่ไม่ได้สร้างขึ้นใน Intune

[การตั้งค่าแบบกําหนดเองสําหรับอุปกรณ์ Android ใน Intune](https://docs.microsoft.com/intune/custom-settings-android)

[การตั้งค่าแบบกําหนดเองสําหรับอุปกรณ์ Android องค์กรใน Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[การตั้งค่าแบบกําหนดเองสําหรับอุปกรณ์ iOS ใน Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[การตั้งค่าแบบกําหนดเองสําหรับอุปกรณ์ macOS ใน Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[การตั้งค่าแบบกําหนดเองสําหรับอุปกรณ์ Windows 10 ใน Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[การตั้งค่าแบบกําหนดเองสําหรับ Windows โฮโลแกรมสําหรับอุปกรณ์ทางธุรกิจใน Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
