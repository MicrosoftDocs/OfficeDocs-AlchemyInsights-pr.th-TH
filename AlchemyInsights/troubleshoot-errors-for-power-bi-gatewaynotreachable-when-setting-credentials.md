# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable เมื่อตั้งค่าข้อมูลประจำตัว

คุณอาจพบ GatewayNotReachable เมื่อพยายามตั้งค่าข้อมูลประจำตัวสำหรับแหล่งข้อมูลภายในบริการ Power BI การดำเนินการนี้อาจเป็นผลลัพธ์ของเกตเวย์ที่ล้าสมัย ติดตั้งเกตเวย์ล่าสุดแล้วลองอีกครั้ง

สำหรับข้อมูลเพิ่มเติม: [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
