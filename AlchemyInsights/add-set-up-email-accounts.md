# <a name="problems-setting-up-an-exchange-email-account-in-outlook"></a>ปัญหาการตั้งค่าบัญชีผู้ใช้อีเมล Exchange ใน Outlook

ถ้าคุณมีปัญหาในการตั้งค่าบัญชีผู้ใช้ Exchange ใน Outlook คุณสามารถเรียกใช้[เครื่องมือการตั้งค่าบัญชี Outlook](https://aka.ms/SaRA-OutlookSetupProfile)บนเครื่องที่ได้รับผลกระทบ มันจะดําเนินการตรวจสอบต่างๆและให้การแก้ไขปัญหาที่เป็นไปได้ปัญหาการตั้งค่าบัญชีของคุณ
  
ถ้าคุณยังคงมีปัญหา ให้ดูที่[Outlook ไม่สามารถตั้งค่าส่วนกําหนดค่าใหม่ โดยใช้การค้นหาอัตโนมัติของอัตราแลกเปลี่ยนสําหรับกล่องจดหมาย Exchange แบบออนไลน์](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover)
