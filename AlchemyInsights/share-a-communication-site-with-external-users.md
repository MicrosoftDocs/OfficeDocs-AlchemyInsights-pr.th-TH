# <a name="share-a-communication-site-with-external-users"></a>ใช้ร่วมกันกับไซต์สื่อสารกับผู้ใช้ภายนอก

เมื่อต้องการเปิดใช้งานผู้ใช้ภายนอกที่ใช้ร่วมกันสำหรับไซต์การสื่อสาร ให้ทำตามขั้นตอนเหล่านี้: 
  
1. เปิด**ศูนย์ดูแลแบบสมัยใหม่**- ถ้าคุณอยู่ในศูนย์กลางการดูแลแบบคลาสสิกคุณจะเห็นการ**ลองเดี๋ยวนี้**หรือ ** ลองแสดงตัวอย่าง ** ปุ่มจากด้านบนขวา วิธีนี้จะนำคุณไปยังศูนย์ดูแลแบบสมัยใหม่ 
  
2. เลือก**ไซต์ที่ใช้งานอยู่**
  
3. เลือกไซต์ที่มีการ**สื่อสารไซต์แม่** 
  
4. เลือกปุ่ม**ใช้ร่วมกัน**จากเมนูบนสุด 
  
 **หมายเหตุ:** ถ้าคุณมีมากกว่าหนึ่งไซต์เลือก จะปรากฏขึ้นปุ่ม**ใช้ร่วมกัน**ถูกปิดใช้งาน เลือกไซต์เดียวเท่านั้น
