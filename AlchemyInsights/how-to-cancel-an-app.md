# <a name="how-to-cancel-software-as-a-service-apps"></a>วิธียกเลิกแอปซอฟต์แวร์แบบบริการ 

1. ในศูนย์การจัดการ Microsoft 365 ไปยังหน้า[บริการ &ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)**สําหรับการเรียกเก็บเงิน** > แล้วเลือกแท็บ**แอป**

2. ค้นหาและเลือกแอปที่คุณต้องการจัดการ

3. ภายใต้**การตั้งค่า& การกระทํา**ให้เลือก**ยกเลิกการบอกรับเป็นสมาชิก**
