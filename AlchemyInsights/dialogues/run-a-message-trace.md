# <a name="run-a-message-trace"></a>เรียกใช้การติดตามข้อความ

การติดตามข้อความจะเรียกใช้ในส่วนลํานักงานจดหมายของศูนย์การรักษา&ของ Office 365

เมื่อต้องการเรียนรู้วิธีการเรียกใช้การติดตามข้อความ ให้ดูการติดตามข้อความในศูนย์การรักษา [ความปลอดภัยของ Office 365 &การปฏิบัติตาม](https://go.microsoft.com/fwlink/?linkid=2103855)นโยบาย
