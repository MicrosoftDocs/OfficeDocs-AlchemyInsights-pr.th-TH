# <a name="change-your-password"></a>เปลี่ยนรหัสผ่านของคุณ

รหัสผ่านOutlook.comของคุณเหมือนกับรหัสผ่านบัญชี Microsoft ของคุณ
  
เมื่อต้องการเปลี่ยนรหัสผ่านของคุณ ให้ไปที่[ความปลอดภัยของบัญชี Microsoft](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409)และเลือก**ความปลอดภัยของรหัสผ่าน**
  
หากคุณลืมรหัสผ่าน ให้ไปที่หน้า[กู้คืนบัญชีผู้ใช้ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=841909)
  
เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[การเปลี่ยนรหัสผ่านของคุณใน Outlook.com](https://support.office.com/article/2138d690-811c-4545-b2f3-e4dbe80c9735?wt.mc_id=Office_Outlook_com_Alchemy)
