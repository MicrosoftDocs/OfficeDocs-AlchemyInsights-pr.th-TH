# <a name="share-a-onedrive-or-sharepoint-file-or-folder-with-external-users"></a>ใช้ OneDrive หรือ SharePoint แฟ้มหรือโฟลเดอร์ร่วมกันกับผู้ใช้ภายนอก

1. ในเบราว์เซอร์ ไปที่แฟ้มหรือโฟลเดอร์ที่คุณต้อง การใช้ร่วมกัน เลือก แล้ว คลิก**ใช้ร่วมกัน**
    
2. ในกล่องโต้ตอบเชื่อมโยงส่ง คลิกลูกศรเพื่อเปลี่ยนชนิดของการเชื่อมโยง
    
3. ภายใต้**ที่คุณต้องการทำงานสำหรับการเชื่อมโยงนี้**เลือก**ผู้ใดก็ตาม**ถ้าคุณต้องการให้ผู้รับเพื่อให้สามารถเชื่อมโยงที่ใช้ร่วมกัน หรือ**เฉพาะบุคคล**เพื่อให้เข้าถึงสำหรับผู้ใช้คุณระบุ 
    
    ถ้าตัวเลือกเหล่านี้ไม่พร้อมใช้งาน คุณจำเป็นต้องทำให้แน่ใจว่า จะใช้ได้ สำหรับองค์กรของคุณ และ สำหรับไซต์คอลเลกชัน เมื่อต้องการทำเช่นนี้ ดู[เปิดภายนอกร่วมกัน หรือปิดสำหรับ SharePoint แบบออนไลน์](https://go.microsoft.com/fwlink/?linkid=866426)
    
4. คลิก**นำไปใช้**
    
5. ป้อนชื่อของบุคคลที่คุณต้องการใช้ร่วมกับ แล้ว คลิ ก**ส่ง**
