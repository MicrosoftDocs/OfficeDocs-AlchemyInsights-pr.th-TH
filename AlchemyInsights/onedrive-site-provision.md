# <a name="onedrive-site-provisioning"></a>การเตรียมใช้งานเว็บไซต์ OneDrive

สำหรับบัญชีที่สร้างขึ้นใหม่ผู้ใช้จำเป็นต้องลงชื่อเข้าใช้ศูนย์ดูแลของ Microsoft ๓๖๕ด้วยบัญชีของตนเองและคลิกไอคอน OneDrive เพื่อเริ่มต้นการจัดเตรียมของไซต์ OneDrive ด้วยตนเอง
ทำตาม[คำแนะนำการแก้ไขปัญหาเบื้องต้น](https://docs.microsoft.com/sharepoint/support/sites/troubleshooting-guide-for-sites-stopped-at-provisioning)สำหรับความช่วยเหลือ

อย่างไรก็ตามถ้าผู้ใช้มีอยู่แล้วคลิกไอคอน OneDrive และไซต์ส่วนบุคคลของเขาไม่ได้เตรียมใช้งานภายใน24ชั่วโมงโปรดเข้าสู่ระบบตั๋วสนับสนุนสำหรับความช่วยเหลือเพิ่มเติม
