# <a name="invoice-and-payment-information"></a>ข้อมูลใบแจ้งหนี้และการชําระเงิน

**ชำระ**

เมื่อคุณซื้อการสมัครใช้งาน Microsoft 365 สําหรับธุรกิจโดยตรงจาก Microsoft คุณสามารถชําระเงินด้วยบัตรเครดิตหรือบัตรเดบิตได้  ในบางประเทศ/ภูมิภาค คุณยังสามารถชําระเงินด้วยบัญชีธนาคารได้ด้วย  คุณสามารถเรียนรู้เกี่ยวกับการจัดการและแก้ไขปัญหาวิธีการชําระเงินเหล่านี้จาก[เพิ่ม อัพเดต หรือเอาบัตรเครดิตหรือตัวเลือกการชําระเงินอื่นๆ สําหรับ Microsoft 365 สําหรับธุรกิจ](https://go.microsoft.com/fwlink/?linkid=2118133)  ซึ่งรวมถึงการแก้ไขปัญหาเมื่อบัตรเครดิตของคุณถูกปฏิเสธและอัปเดตข้อมูลการชําระเงิน

**ใบแจ้งหนี้/บิล**

คุณสามารถค้นหาและดาวน์โหลดใบแจ้งหนี้ของคุณได้ในหน้า**[การชําระเงิน>การเรียกเก็บเงิน&](https://go.microsoft.com/fwlink/p/?linkid=848039) **  

**หมายเหตุ**: หากไม่มีใบแจ้งหนี้ใดที่มองเห็น คุณอาจจําเป็นต้องปรับตัวกรองช่วงวันที่  โดยค่าเริ่มต้น 3 เดือนสุดท้ายของใบแจ้งหนี้จะแสดงขึ้น

ใบแจ้งหนี้จะถูกสร้างขึ้นเป็นรายเดือนหรือรายปีขึ้นอยู่กับความถี่ในการชําระเงินที่เลือกในระหว่างการซื้อ  หากชําระเงินเป็นประจําทุกปี คุณอาจได้รับใบเรียกเก็บเงินระยะกลางเมื่อเพิ่มหรือเอาสิทธิ์การใช้งานในการสมัครใช้งาน
 
- สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ทําความเข้าใจเกี่ยวกับใบแจ้งหนี้ของคุณสําหรับ Microsoft 365 สําหรับธุรกิจ](https://go.microsoft.com/fwlink/?linkid=2119101)
