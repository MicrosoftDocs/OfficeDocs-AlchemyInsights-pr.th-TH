# <a name="how-to-access-outlookcom-while-traveling"></a>วิธีเดินทางไปยังOutlook.comขณะเดินทาง

ในการเข้าถึงOutlook.comขณะเดินทาง โปรดตรวจสอบให้แน่ใจว่าข้อมูลความปลอดภัยบัญชี Microsoft ของคุณ ซึ่งเป็นที่อยู่อีเมลสํารองหรือหมายเลขโทรศัพท์ที่คุณสามารถรับรหัสความปลอดภัยได้ เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[วิธีการเข้าถึงOutlook.comเมื่อเดินทาง](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy)
