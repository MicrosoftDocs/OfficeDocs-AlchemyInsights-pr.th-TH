# <a name="export-a-notebook"></a>การส่งออกสมุดบันทึก

ในแอปเดสก์ท็อปของ OneNote 2016:

1. บนหน้าที่คุณต้องการส่งออก ให้คลิก**ไฟล์ > ส่งออก**
2. ภายใต้**ส่งออกปัจจุบัน**ให้คลิกสิ่งที่คุณต้องการส่งออก (หน้า ส่วน หรือสมุดบันทึก)
3. คลิก**ส่งออก**
 
ในการถ่ายโอนความเป็นเจ้าของไฟล์ที่เก็บอยู่ในระบบคลาวด์**Share****Move/Copy**  

- ที่มุมขวาบนของหน้าต่างแอป ให้คลิกหรือแตะ**แชร์**
- เมื่อต้องการย้ายสมุดบันทึก ให้ดูที่[ย้ายสมุดบันทึก OneNote สําหรับ Windows ที่คุณแชร์กับผู้อื่น](https://support.office.com/article/move-a-onenote-for-windows-notebook-that-you-ve-shared-with-others-56c7659e-1850-49a6-8874-e2db6b440cd4?ui=en-US&rs=en-US&ad=US)
