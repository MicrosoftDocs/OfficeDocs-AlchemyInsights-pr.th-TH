# <a name="the-new-outlook-on-the-web"></a>Outlook ใหม่บนเว็บ

คุณไม่สามารถกลับไปยัง Outlook แบบคลาสสิกบนประสบการณ์เว็บได้อีกต่อไป หากคุณต้องการให้คำติชมเกี่ยวกับประสบการณ์ใหม่ให้ไปที่[ฟอรัม Uservoice](https://go.microsoft.com/fwlink/?linkid=2103182) คุณสามารถลงคะแนนเสียงเกี่ยวกับแนวคิดหรือประเด็นที่คุณต้องการดูและเราจะตอบสนองโดยตรงกับความคิดเห็นและข้อเสนอแนะด้านบน

[การขอความช่วยเหลือเกี่ยวกับ Outlook ใหม่บนเว็บ](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
