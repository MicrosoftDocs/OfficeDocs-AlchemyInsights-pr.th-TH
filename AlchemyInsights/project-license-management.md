# <a name="project-license-management"></a>การจัดการสิทธิ์การใช้งานโครงการ

การจัดการสิทธิ์การใช้งาน Project สามารถเสร็จสมบูรณ์ในศูนย์ **การจัดการ Microsoft 365**

- [เปรียบเทียบตัวเลือก Project](https://www.microsoft.com/microsoft-365/project/compare-microsoft-project-management-software)เพื่อเรียนรู้เกี่ยวกับความแตกต่างระหว่าง **Project Plan 1, 3 และ 5**
- เมื่อต้องการ **ซื้อ** Project ให้ไปที่ **หน้าการเรียกเก็บเงิน> [สั่งซื้อบริการ](https://go.microsoft.com/fwlink/p/?linkid=868433)** ต่างๆ ใช้กล่องค้นหาเพื่อค้นหาบน "โครงการ" เลือกแผนและจํานวนผู้ใช้ ดูข้อมูลเพิ่มเติมได้ที่ [ซื้อการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/try-or-buy-microsoft-365#buy-a-different-subscription)อื่น
- เมื่อต้องการ **เปลี่ยน** แผน [ให้ดู อัปเกรดเป็นแผน](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/upgrade-to-different-plan)อื่น
- เมื่อต้องการ **กําหนด** สิทธิ์การใช้งาน Project ให้กับผู้ใช้ ให้ไปที่ **>[สิทธิ์การใช้งาน](https://go.microsoft.com/fwlink/p/?linkid=842264)** For more info, see [Assign licenses to users](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
- เมื่อต้องการ **ซื้อหรือลบ** สิทธิ์การใช้งาน Project ให้ไปที่หน้า **การเรียกเก็บเงิน> [ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)** ของคุณ ดูข้อมูลเพิ่มเติมได้ที่ [ซื้อหรือเอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses#add-or-remove-licenses-for-your-business-subscription)ออก
- เมื่อต้องการ **ยกเลิก** การสมัครใช้งาน Project ให้ไปที่ หน้า **การเรียกเก็บเงิน >[ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)** ของคุณ เลือกการสมัครใช้งาน และในส่วน **การตั้งค่าการสมัครใช้งานและการชเงิน****ให้เลือก** ยกเลิกการสมัครใช้งาน ดูข้อมูลเพิ่มเติมได้ที่ [ยกเลิกการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)ของคุณ

ดูข้อมูลเพิ่มเติมได้ที่ [ซื้อหรือเอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)ออก
