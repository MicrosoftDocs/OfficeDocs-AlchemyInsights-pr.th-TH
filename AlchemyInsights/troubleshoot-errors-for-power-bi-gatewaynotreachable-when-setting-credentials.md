# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable เมื่อตั้งค่าข้อมูลประจำตัว

คุณอาจพบ GatewayNotReachable เมื่อพยายามที่จะตั้งค่าข้อมูลประจำตัวสำหรับแหล่งข้อมูลภายในบริการ นี่อาจเป็นผลมาจากเกตเวย์ที่ล้าสมัย ติดตั้งเกตเวย์ล่าสุดแล้วลองอีกครั้ง
