# <a name="declined-credit-card"></a>บัตรเครดิตถูกปฏิเสธ

หากคุณชําระเงินด้วยบัตรเครดิต และบัตรเครดิตของคุณถูกปฏิเสธ คุณจะได้รับอีเมลแจ้งว่าเราไม่สามารถดําเนินการชําระเงินได้ ตรวจสอบอีกครั้งว่า[รายละเอียดบัตรเครดิต](https://go.microsoft.com/fwlink/p/?linkid=842054)- หมายเลขบัตรวันหมดอายุชื่อบนบัตรเครดิตและที่อยู่รวมทั้งเมืองรัฐและรหัสไปรษณีย์ - ปรากฏตรงตามที่พวกเขาทําในบัตรเครดิตและใบแจ้งยอดบัตรเครดิตของคุณ คุณสามารถอัปเดตข้อมูลบัตรเครดิตของคุณและส่งการชําระเงินได้ทันทีโดยใช้ตัวเลือก**ชําระยอดคงเหลือ**ในหน้า**[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)** 

หากต้องการเรียนรู้เพิ่มเติม ให้ดูที่ "จะเกิดอะไรขึ้นหากบัตรเครดิตของฉันหมดอายุและการชําระเงินของฉันเลยกําหนดชําระแล้ว ใน[ชําระเงินสําหรับการสมัครใช้งาน Microsoft 365 สําหรับธุรกิจ ของคุณ](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/pay-for-your-subscription#what-if-my-credit-card-was-declined-and-my-payment-is-past-due)
  
หากคุณยังคงเห็นข้อความ "ปฏิเสธ" อยู่ ให้ติดต่อธนาคารของคุณ เป็นไปได้ว่าบัตรเครดิตของคุณไม่ได้ใช้งาน เช่น หากคุณเพิ่งได้รับบัตรเครดิตในอีเมลที่มีวันหมดอายุที่อัปเดตแล้ว คุณอาจยังคงต้องเปิดใช้งานบัตรเครดิตดังกล่าว ธนาคารของคุณยังสามารถบอกคุณได้ว่าบัตรเครดิตของคุณไม่ได้รับการอนุมัติสําหรับการทําธุรกรรมออนไลน์ระหว่างประเทศหรือที่เกิดขึ้น
  
สําหรับข้อมูลเพิ่มเติม ให้ดูที่[จัดการวิธีการชําระเงิน](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/manage-payment-methods)
