# <a name="map-a-sharepoint-library-to-a-network-drive"></a>แผนผังไลบรารี SharePoint ไปยังไดรฟ์เครือข่าย

ไลบรารีเป็นไดรฟ์เครือข่ายที่แมปเป็นแบบชั่วคราว และได้รับการสนับสนุน โดย Internet Explorer เท่านั้น ในบางครั้งคุณต้องเปิดไซต์ SharePoint ใน Internet Explorer และเลือก "อยู่ลงชื่อเข้าใช้" เพื่อป้องกันไม่ให้เซสชันหมดอายุ แทน[ซิงค์แฟ้ม SharePoint ด้วยไคลเอ็นต์การซิงค์ OneDrive ใหม่](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)</a>ซึ่งเป็น[แฟ้มตามความต้องการ](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e) เข้าถึงแฟ้มของคุณทั้งหมดใน OneDrive โดยไม่ต้องใช้เนื้อที่เก็บข้อมูลภายในเครื่อง

ถ้าคุณเลือกที่จะแมปไดร์ฟแทนการ[ใช้ไคลเอ็นต์การซิงค์ OneDrive ใหม่](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)ให้แน่ใจว่า คุณทำตามขั้นตอนต่าง ๆ ในบทความนี้ 


**วิธีการตั้งค่าคอนฟิก และการแก้ไขปัญหาแมปไดรฟ์เครือข่าย**


- [ตั้งค่าคอนฟิก และการแก้ปัญหาแมปไดรฟ์เครือข่าย](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US)

หมายเหตุ: สำหรับ 10 Explorer อินเทอร์เน็ตกับ Windows 8 หรือ Windows 7 ที่ได้รับ "Access denied" หรือ "เส้นทางไม่สามารถเข้าถึงได้" เมื่อต้องการแมปไดรฟ์ ติดตั้งโปรแกรมแก้ไขด่วนนี้เพื่อแก้ไขปัญหานี้ ไปที่[ข้อผิดพลาดเมื่อคุณเปิดไลบรารีเอกสาร SharePoint ใน Windows Explorer หรือแมปไดรฟ์เครือข่ายไปยังไลบรารีหลังจากที่คุณติดตั้ง Internet Explorer 10](https://support.microsoft.com/help/2846960)
