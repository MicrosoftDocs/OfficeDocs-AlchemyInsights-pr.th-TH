# <a name="upload-a-folder-or-files-to-a-sharepoint-document-library"></a>อัปโหลดโฟลเดอร์หรือแฟ้มไปยังไลบรารีเอกสาร SharePoint

ในการอัปโหลดโฟลเดอร์, คุณจําเป็นต้องใช้ไมโครซอฟท์ขอบ, Google Chrome, หรือ Mozilla Firefox. คุณไม่สามารถอัปโหลดโฟลเดอร์ในประสบการณ์การใช้งานไลบรารีแบบคลาสสิก หรือใช้ Internet Explorer 10 หรือ 11
  
1. เปิดไลบรารีเอกสารที่คุณต้องการอัปโหลดโฟลเดอร์หรือแฟ้ม
    
2. เปิด File Explorer ในคอมพิวเตอร์ของคุณและเรียกดูโฟลเดอร์หรือไฟล์ที่คุณต้องการอัปโหลด
    
3. ลากโฟลเดอร์หรือแฟ้มไปยังไลบรารีเอกสาร ถ้าคุณไม่เห็นรายการที่คุณอัปโหลดจะปรากฏในไลบรารี ให้รีเฟรชหน้า 
    
ถ้าคุณไม่ต้องการอัปโหลดด้วยการลากรายการระหว่างหน้าต่าง**Upload**
