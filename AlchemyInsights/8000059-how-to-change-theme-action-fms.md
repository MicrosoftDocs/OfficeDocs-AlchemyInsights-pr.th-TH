# <a name="how-to-change-your-theme"></a>วิธีการเปลี่ยนชุดรูปแบบของคุณ

เมื่อต้องการเปลี่ยนชุดรูปแบบของคุณใน Outlook.com หรือ Outlook บนเว็บ ไปที่**การตั้งค่า**![ตั้งค่า](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png)และเลือกชุดรูปแบบภายใต้**ชุดรูปแบบ** เมื่อต้องการดูชุดรูปแบบที่พร้อมใช้งานทั้งหมด เลือก**มุมมองทั้งหมด**ภายใต้**ชุดรูปแบบ**
