# <a name="assign-or-remove-users-licenses"></a>มอบหมายหรือเอาสิทธิ์การใช้งานของผู้ใช้ออก

ในฐานะผู้ดูแลระบบ คุณสามารถปรับสิทธิ์การใช้งานของผู้ใช้ได้ตามต้องการ บทความด้านล่างจะแสดงวิธีจัดการใบอนุญาตของผู้ใช้รวมถึง:
  
- [กําหนดสิทธิ์การใช้งานให้กับผู้ใช้ใน Microsoft 365 สําหรับธุรกิจ](https://docs.microsoft.com//office365/admin/subscriptions-and-billing/assign-licenses-to-users)

- [ยกเลิกการมอบหมายสิทธิ์การใช้งานจากผู้ใช้ใน Microsoft 365 สําหรับธุรกิจ](https://docs.microsoft.com//office365/admin/subscriptions-and-billing/remove-licenses-from-users)

ไม่ได้ตอบคําถามของคุณ? ไม่มีปัญหา! ใช้ปุ่ม "ต้องการความช่วยเหลือ" เพื่อพูดคุยกับวิศวกรฝ่ายสนับสนุน
