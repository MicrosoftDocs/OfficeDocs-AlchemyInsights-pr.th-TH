# <a name="create-or-migrate-update-rings"></a>การสร้างหรือโยกย้ายการอัปเดตวงแหวน

ถ้าคุณกำหนดค่า Windows 10 การตั้งค่าการอัปเดตในพอร์ทัลของ Intune classic และคุณต้องการโยกย้ายการตั้งค่าไปยัง Intune ในพอร์ทัล Azure ให้ทำตามขั้นตอนต่อไปนี้:

1.  ไปที่พอร์ทัล Azure และเลือก**บริการทั้งหมด**
2.  ในเขตข้อมูล**ตัวกรอง**ให้พิมพ์**Intune**แล้วเลือก**Microsoft Intune**
3.  เลือกการ**อัปเดตซอฟต์แวร์**   >   **Windows 10 ปรับปรุง**การ   >   **สร้าง**วงแหวน
4.  ใส่ชื่อและคำอธิบายแล้วเลือก**กำหนดค่า**
5.  กำหนดค่าการตั้งค่าการอัปเดตซอฟต์แวร์สำหรับองค์กรของคุณ
6.  เลือก**ตกลง**  >  **สร้างวงแหวนอัปเดต**  >  **Create**
