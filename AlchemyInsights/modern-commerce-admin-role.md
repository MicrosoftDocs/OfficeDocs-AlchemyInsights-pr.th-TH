# <a name="modern-commerce-administrator-role"></a>บทบาทผู้ดูแลระบบการค้าสมัยใหม่

บทบาทนี้ถูกกําหนดให้กับผู้ใช้ที่ซื้อเพื่อให้สิทธิ์การเข้าถึงพอร์ทัลผู้ดูแลระบบที่จํากัดเพื่อจัดการการซื้อนั้น ผู้ใช้เหล่านี้ไม่ใช่ผู้ดูแลระบบ และไม่มีสิทธิ์ผู้ดูแล
