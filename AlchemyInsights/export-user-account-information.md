# <a name="export-user-account-information"></a>ส่งออกข้อมูลบัญชีผู้ใช้

เมื่อต้องการส่งออกข้อมูลบัญชีผู้ใช้ในศูนย์การจัดการ Microsoft 365 รวมถึงข้อมูลเกี่ยวกับสิทธิ์การใช้งาน นโยบายรหัสผ่าน บัญชีที่ถูกบล็อก สถานะการซิงค์ AAD และอื่นๆ ให้ไปที่เพจ**Users** > [ผู้ใช้ที่ใช้งานอยู่](https://go.microsoft.com/fwlink/p/?linkid=834822)ของผู้ใช้ แล้วเลือก**ส่งออกผู้ใช้**
