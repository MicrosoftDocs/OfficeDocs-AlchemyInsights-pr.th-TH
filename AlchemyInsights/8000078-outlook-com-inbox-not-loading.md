# <a name="inbox-not-loading-with-firefox"></a>ไม่ได้โหลด ด้วย Firefox กล่องขาเข้า

บาง add-on ของเบราว์เซอร์ Firefox อาจป้องกันกล่องขาเข้า Outlook.com จากการโหลด
  
เยี่ยมชมสนับสนุน Firefox เพื่อเรียนรู้วิธีการ[ปิดใช้งาน หรือเอา Add-on ของ](https://support.mozilla.org/kb/disable-or-remove-add-ons)
