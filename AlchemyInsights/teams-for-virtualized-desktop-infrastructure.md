# <a name="teams-for-virtualized-desktop-infrastructure"></a>ทีมสำหรับโครงสร้างพื้นฐานเดสก์ท็อปเสมือน

สำหรับข้อมูลรายละเอียดเกี่ยวกับทีมสำหรับสภาพแวดล้อมโครงสร้างพื้นฐานเดสก์ท็อปเสมือนให้ดูที่[ทีมงานโครงสร้างพื้นฐานเดสก์ท็อปเสมือน](https://docs.microsoft.com/en-us/microsoftteams/teams-for-vdi)จริง

ปัญหาที่ทราบและข้อจำกัดในสภาพแวดล้อม VDI มีรายละเอียดในส่วนของ[ปัญหาที่ทราบและข้อจำกัด](https://docs.microsoft.com/en-us/microsoftteams/teams-for-vdi#known-issues-and-limitations)ของบทความเดียวกันซึ่งรวมถึงหัวข้อต่อไปนี้:
 - การปรับใช้ไคลเอนต์การติดตั้งและการตั้งค่า
 - การโทรและการประชุม
