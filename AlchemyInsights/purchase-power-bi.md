# <a name="purchase-power-bi"></a>ซื้อ Power BI

Power BI สามารถซื้อได้ใน**ศูนย์การจัดการ Microsoft ๓๖๕**

1. ไปที่หน้า**บริการการ[ซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)> การเรียกเก็บเงิน**

2. ใช้กล่องค้นหาเพื่อค้นหา "Power BI"

3. เลือกแผน Power BI ที่คุณต้องการซื้อ

4. เลือกจำนวนผู้ใช้ที่จะใช้ Power BI

เมื่อต้องการกำหนดสิทธิ์การใช้งานให้กับผู้ใช้ให้ดูที่[กำหนดสิทธิ์การใช้งานให้กับผู้ใช้](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide)

เมื่อต้องการซื้อสิทธิ์การใช้งานเพิ่มเติมหรือเอาสิทธิ์การใช้งานออกให้ดูที่ [เพิ่ม/เอาสิทธิ์การ](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)ใช้งานออก
