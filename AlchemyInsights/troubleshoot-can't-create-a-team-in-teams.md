# <a name="what-to-do-if-you-cant-create-a-team-in-microsoft-teams"></a>สิ่งที่ต้องทำถ้าคุณไม่สามารถสร้างทีมในทีม Microsoft

ขอให้ผู้ดูแลระบบทีมของคุณดำเนินการดังต่อไปนี้:

1. สร้างหรือเปลี่ยนการตั้งค่ากลุ่มความปลอดภัยขององค์กรใน Microsoft ๓๖๕
2. เพิ่มคุณลงในกลุ่มความปลอดภัยนั้น
