# <a name="outlookcom-updates"></a>การปรับปรุงOutlook.com

**การปรับปรุงล่าสุด**

- รับข้อมูลเพิ่มเติมจากอีเมลของคุณด้วยOutlook.comใหม่ เราได้ปรับปรุงการค้นหาเพื่อให้การค้นหาง่ายขึ้นกว่าที่เคยค้นหาสิ่งที่คุณต้องการ คุณลักษณะใหม่ ๆ เช่น รูปภาพและตัวอย่างไฟล์แนบช่วยให้คุณเห็นรูปภาพและไฟล์ของคุณโดยไม่ต้องออกจากกล่องจดหมายของคุณ และตอนนี้คุณสามารถเพิ่มอิโมจิและ GIF ลงในข้อความของคุณได้แล้ว  [เรียนรู้เพิ่มเติมเกี่ยวกับOutlook.comใหม่](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)

**อีเมลอัพเดท**

- หากกล่องจดหมายOutlook.comของคุณไม่ได้รับการอัปเดต โปรดดู [แก้ไขปัญหาการซิงค์อีเมล Outlook.com](https://support.office.com/article/d39e3341-8d79-4bf1-b3c7-ded602233642?wt.mc_id=Office_Outlook_com_Alchemy)
