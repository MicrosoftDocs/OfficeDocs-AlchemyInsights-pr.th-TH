# <a name="export-user-account-information"></a>ส่งออกข้อมูลบัญชีผู้ใช้

เมื่อต้องการส่งออกข้อมูลบัญชีผู้ใช้ในศูนย์การจัดการ Microsoft ๓๖๕รวมถึงข้อมูลเกี่ยวกับสิทธิ์การใช้งานนโยบายรหัสผ่านบัญชีผู้ใช้ที่ถูกบล็อกสถานะการซิงค์ AAD และอื่นๆให้ไปที่หน้าผู้ใช้ที่ใช้งานอยู่ของ**ผู้ใช้**  >  [Active users](https://go.microsoft.com/fwlink/p/?linkid=834822)แล้วเลือก**ส่งออกผู้ใช้**
