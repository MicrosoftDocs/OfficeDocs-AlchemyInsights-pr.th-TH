<span data-ttu-id="0fe52-p101">ถ้าคุณกำลังใช้ Outlook บนเว็บ และคุณได้รับข้อผิดพลาดที่**ไม่สามารถพบกล่องจดหมายสำหรับ**บัญชีที่คุณใช้เพื่อเชื่อมต่อกับ Outlook บนเว็บ ไม่มีสิทธิ์การใช้งานการแลกเปลี่ยนแบบออนไลน์ และดังนั้น ไม่มีกล่องจดหมายถูกเชื่อมโยงกับลูกค้าองค์กร ผู้ดูแลระบบของคุณสามารถกำหนดสิทธิ์การใช้งานบัญชีของคุณ โดยทำตามขั้นตอนเหล่านี้:</span><span class="sxs-lookup"><span data-stu-id="0fe52-p101">If you're using Outlook on the web and you get a **Mailbox couldn't be found for** error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account. Your admin can assign a license to your account by following these steps:</span></span> 
  
1. <span data-ttu-id="0fe52-105">เปิด[Microsoft 365 admin ศูนย์](https://portal.office.com/adminportal/home#/homepage)และ**ผู้ใช้ที่ใช้งานอยู่**ใต้**แก้ไขผู้ใช้**</span><span class="sxs-lookup"><span data-stu-id="0fe52-105">Open the [Microsoft 365 admin center](https://portal.office.com/adminportal/home#/homepage) and under **Active users**, select **Edit a user**.</span></span>
    
2. <span data-ttu-id="0fe52-p102">ใน**การแก้ไขผู้ใช้**เพจที่เปิด เลือกผู้ใช้ ในหน้าคุณสมบัติของผู้ใช้ที่เปิด คลิก**แก้ไข**สำหรับ**สิทธิ์การใช้งานผลิตภัณฑ์**</span><span class="sxs-lookup"><span data-stu-id="0fe52-p102">In the **Edit a user** page that opens, select the user. In the user properties page that opens, click **Edit** for **Product licenses**.</span></span>
    
3. <span data-ttu-id="0fe52-p103">ในเพจ**สิทธิ์การใช้งานผลิตภัณฑ์**ที่เปิด เลือกค่า**ตำแหน่งที่ตั้ง**ที่เหมาะสม และกำหนดสิทธิ์การใช้งานที่ประกอบด้วยการแลกเปลี่ยนแบบออนไลน์ (ขยายสิทธิ์การใช้งานเพื่อดูรายละเอียดของ) เมื่อคุณทำเสร็จแล้ว คลิก**บันทึก**</span><span class="sxs-lookup"><span data-stu-id="0fe52-p103">In the **Product licenses** page that opens, select the appropriate **Location** value, and assign a license that contains Exchange Online (expand the license to see its details). When you're finished, click **Save**.</span></span>
