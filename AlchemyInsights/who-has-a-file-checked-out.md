# <a name="who-has-a-file-checked-out"></a>ใครมีไฟล์เช็คเอาท์

เมื่อต้องการดูว่าใครมีไฟล์ที่เช็คเอาท์ใน SharePoint คุณสามารถทำตามขั้นตอนใดขั้นตอนหนึ่งต่อไปนี้:
  
1. โฮเวอร์เหนือ **เช็คเอาท์** เพื่อดูคำแนะนำเครื่องมือที่บอกให้คุณทราบว่ามีไฟล์เช็คเอาท์แล้ว 
    
2. เพิ่มคอลัมน์ **เช็คเอาท์โดย** ใช้มุมมองเริ่มต้นของไลบรารี ด้วยวิธีนี้ทุกคนสามารถดูได้อย่างง่ายดายว่าใครเป็นผู้เช็คเอาท์เอกสาร 
    
เมื่อต้องการเรียนรู้วิธีการเพิ่มคอลัมน์เช็คเอาท์โดยใช้ใน SharePoint ให้ดูที่[สร้างคอลัมน์ในรายการหรือไลบรารี](https://go.microsoft.com/fwlink/?linkid=2019591) 
  
ดูคำถามที่ถามบ่อยเพิ่มเติมเกี่ยวกับการเช็คอินหรือเช็คเอาท์ดู[คำถามยอดนิยมเกี่ยวกับการเช็คเอาท์เช็คเอาท์](https://go.microsoft.com/fwlink/?linkid=2018786)
