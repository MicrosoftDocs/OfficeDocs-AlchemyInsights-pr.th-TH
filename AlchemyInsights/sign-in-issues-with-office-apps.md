# <a name="issues-signing-into-microsoft-365-apps"></a>ปัญหาในการลงชื่อเข้าใช้แอป Microsoft 365

เมื่อต้องการแก้ไขปัญหาการลงชื่อเข้าใช้ด้วยแอป Microsoft 365 ลองตัวเลือกต่อไปนี้บนเครื่องที่ได้รับผลกระทบ:

- เปิดแอป Office และ[ออกจากระบบ](https://go.microsoft.com/fwlink/?linkid=2114082)บัญชีผู้ใช้ที่มีอยู่
- การใช้การตั้งค่า Windows **>บัญชี>บัญชี&อีเมล**ลบบัญชีเก่าทั้งหมดหรือบัญชีที่ทํางานที่ไม่ได้ใช้
- การใช้การตั้งค่า Windows **> บัญชี> Access ทํางานหรือโรงเรียน**
- [รีเซ็ตสถานะการเปิดใช้งาน Office](https://docs.microsoft.com/office365/troubleshoot/activation/reset-office-365-proplus-activation-state)แล้ว[ลงชื่อเข้าใช้](https://support.office.com/article/sign-in-to-office-b9582171-fd1f-4284-9846-bdd72bb28426)แอป Microsoft 365 โดยใช้บัญชีผู้ใช้ที่ได้รับสิทธิ์การใช้งาน
- สําหรับ Mac ให้ดูที่[ไม่สามารถลงชื่อเข้าใช้แอป Office 2016 for Mac](https://docs.microsoft.com/office365/troubleshoot/authentication/sign-in-to-office-2016-for-mac-fail)ได้
