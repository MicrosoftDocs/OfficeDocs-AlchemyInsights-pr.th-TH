# <a name="having-issues-getting-outlook-to-connect"></a>มีปัญหาในการรับ Outlook ให้เชื่อมต่อใช่ไหม

เราตรวจพบว่า Outlook ไม่ได้กําหนดค่าให้ทํางานอย่างถูกต้องภายใต้โดเมนต่อไปนี้:
  
{สตริงเช็คค่าความสมบูรณ์}
  
ระเบียน NS หรือ CNAME ของการตั้งค่าโดเมน (DNS) ไม่ได้ตั้งค่าอย่างถูกต้อง คุณสามารถอ้างอิงถึง[สร้างระเบียน DNS ได้ที่ผู้ให้บริการโฮสต์ DNS สําหรับ](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)คําแนะนําโดยละเอียด 
  
หากต้องการแก้ไข โปรดไปที่หน้า[โดเมน](https://admin.microsoft.com/adminportal/home#/Domains)ในศูนย์การจัดการ Microsoft 365 และเราจะช่วยคุณได้
