# <a name="your-users-mailbox-is-being-created"></a>กล่องจดหมายของผู้ใช้ของคุณถูกสร้างขึ้น

อาจใช้เวลานานถึงหนึ่งชั่วโมงก่อนที่กล่องจดหมายใหม่จะพร้อมใช้งาน
  
ถ้ากล่องจดหมายของผู้ใช้ได้รับ "การเตรียม" มากกว่าหนึ่งชั่วโมงคุณสามารถลองลบและสร้างผู้ใช้ใหม่หรือใช้ตัวเลือก "อนุญาตให้เราติดต่อคุณ" เพื่อพูดคุยกับวิศวกรฝ่ายสนับสนุน
