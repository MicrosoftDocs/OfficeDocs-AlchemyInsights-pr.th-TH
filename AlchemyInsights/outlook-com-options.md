# <a name="how-to-find-options-in-outlookcom-settings"></a>วิธีการค้นหาตัวเลือกในการตั้งค่า Outlook.com

ที่ด้านบนของหน้า เลือกเกียร์ไอคอนเพื่อเปิดการตั้งค่า ที่ด้านล่างของบานหน้าต่างการตั้งค่า เลือก**ดูการตั้งค่า Outlook ทั้งหมด**

สิ่งนี้จะแสดงการทั่วไปของคุณ อีเมล ปฏิทิน และ (ถ้าเกี่ยวข้อง) ตัวเลือกและการตั้งค่าค่าจ้างพิเศษ
