# <a name="create-email-signatures"></a>สร้างลายเซ็นของเมล

หัวข้อเหล่านี้จะช่วยคุณสร้างลายเซ็นของเมล:
  
- สร้างลายเซ็นของจดหมายสำหรับข้อความของคุณเอง:[สร้างและเพิ่มลายเซ็นลงในข้อความ](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)หรือ[สร้างและเพิ่มลายเซ็นของจดหมายใน Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx)
    
- สร้างลายเซ็นของจดหมายส่วนกลางสำหรับข้อความทั้งหมดที่ส่งมาจากบริษัทของคุณ (หรือที่เรียกว่าการปฏิเสธความรับผิดชอบ): การปฏิเสธการรับรู้ของ[ทั้งองค์กรลายเซ็นท้ายกระดาษหรือส่วนหัวใน Office ๓๖๕](https://go.microsoft.com/fwlink/p/?linkid=391096)
