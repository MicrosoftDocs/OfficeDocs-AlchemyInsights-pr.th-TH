# <a name="cant-preview-all-of-an-attachment"></a>ไม่สามารถแสดงตัวอย่างสิ่งที่แนบมาทั้งหมดได้

เราทราบถึงปัญหาที่ลูกค้าบางรายไม่สามารถแสดงตัวอย่างสิ่งที่แนบมาทั้งหมดได้ ในฐานะที่เป็นวิธีแก้ไขปัญหาชั่วคราวโปรดเปิดสิ่งที่แนบมาในหน้าต่างใหม่
