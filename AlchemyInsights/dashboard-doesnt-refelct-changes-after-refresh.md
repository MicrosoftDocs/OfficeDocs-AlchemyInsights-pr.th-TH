# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>แดชบอร์ดไม่แสดงการเปลี่ยนแปลงหลังการรีเฟรช

ถ้าแดชบอร์ดไม่แสดงการเปลี่ยนแปลงหลังการรีเฟรชให้รอประมาณ10-15 นาทีเพื่อให้การรีเฟรชปรากฏในไทล์แดชบอร์ด ถ้ายังไม่ปรากฏขึ้นให้ปักหมุดการแสดงภาพไปยังแดชบอร์ดอีกครั้ง

สำหรับข้อมูลเพิ่มเติมให้ดู[การแก้ไขปัญหาชั่วคราวการรีเฟรช](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
