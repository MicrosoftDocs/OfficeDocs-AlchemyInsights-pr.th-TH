# <a name="manage-access-requests"></a>จัดการการร้องขอการเข้าถึง

เมื่อผู้ใช้พยายามเข้าถึงรายการใน SharePoint และไม่มีสิทธิ์เหล่านั้นจะเห็นข้อความที่พวกเขาจำเป็นต้องมีสิทธิ์ในการเข้าถึงรายการนั้น ผู้ใช้สามารถร้องขอการเข้าถึง (ในบางกรณีไปยังรายการที่เฉพาะเจาะจงหรือเมื่อต้องการเข้าร่วมกลุ่ม) และเจ้าของไซต์สามารถอนุมัติหรือปฏิเสธการร้องขอได้ ถ้าบุคคลที่ไม่ใช่เจ้าของไซต์แชร์ไซต์กับบุคคลอื่นเจ้าของไซต์จะได้รับการร้องขอการเข้าถึงสำหรับผู้ใช้ใหม่ เมื่อต้องการอนุมัติหรือปฏิเสธการร้องขอการเข้าถึง:
  
1. ไปที่ไซต์แล้วคลิกไอคอนการตั้งค่าที่มุมขวาบนแล้วคลิก**การตั้งค่าไซต์** (ถ้าคุณไม่เห็นการตั้งค่าไซต์ให้คลิก**สิทธิ์สำหรับไซต์**แล้วคลิก**การตั้งค่าสิทธิ์ขั้นสูง** ค้นหาแบนเนอร์ที่มีการเชื่อมโยงที่เรียกว่า "แสดงการร้องขอการเข้าถึงและการเชิญ")
    
2. ภายใต้**ผู้ใช้และสิทธิ์**ให้คลิกการ**ร้องขอการเข้าถึงและการเชิญ** (ถ้าไม่ปรากฏขึ้นแสดงว่ามีการร้องขอการเข้าถึงไม่ได้ทำ)
    
3. เมื่อต้องการให้สิทธิ์แก่ผู้ใช้แต่ละรายให้คลิก**อนุมัติ**หรือ**ปฏิเสธ** เมื่อต้องการเลือกระดับสิทธิ์ที่เฉพาะเจาะจงให้คลิกที่จุดไข่ปลา (...) ที่อยู่ถัดจากชื่อของบุคคลนั้น อีกวิธีหนึ่งคือเพิ่มผู้ใช้ลงในกลุ่มที่มีสิทธิ์ในการเข้าถึงไซต์ สำหรับไซต์การติดต่อสื่อสารและไซต์ทีมแบบคลาสสิกคุณสามารถทำสิ่งนี้ได้บนหน้าการตั้งค่าไซต์ สำหรับไซต์ที่เป็นสมาชิกของ Microsoft ๓๖๕กลุ่มคุณสามารถเพิ่มผู้ใช้ลงในกลุ่มได้
