# <a name="using-custom-policy-in-intune"></a>การใช้นโยบายแบบกำหนดเองใน Intune

Intune มีหลายการตั้งค่าสำหรับอุปกรณ์ Android, iOS, macOS และ Windows คุณสามารถสร้างส่วนกำหนดค่าแบบกำหนดเอง และจากนั้น ปรับใช้การตั้งค่าแบบกำหนดเองเหล่านี้ไปยังอุปกรณ์ของคุณโดยใช้ Intune ส่วนกำหนดค่าแบบกำหนดเองถูกออกแบบมาเพื่อเพิ่มการตั้งค่าอุปกรณ์และคุณลักษณะที่ไม่ได้อยู่ภายใน Intune

[การตั้งค่าแบบกำหนดเองสำหรับอุปกรณ์ Android ใน Intune](https://docs.microsoft.com/intune/custom-settings-android)

[การตั้งค่าแบบกำหนดเองสำหรับอุปกรณ์ Android องค์กรใน Intune](https://docs.microsoft.com/intune/custom-settings-android-for-work)

[การตั้งค่าแบบกำหนดเองสำหรับอุปกรณ์ iOS ใน Intune](https://docs.microsoft.com/intune/custom-settings-ios)

[การตั้งค่าแบบกำหนดเองสำหรับอุปกรณ์ macOS ใน Intune](https://docs.microsoft.com/intune/custom-settings-macos)

[การตั้งค่าแบบกำหนดเองสำหรับอุปกรณ์ Windows 10 ใน Intune](https://docs.microsoft.com/intune/custom-settings-windows-10)

[การตั้งค่าแบบกำหนดเองสำหรับ Windows Holographic สำหรับอุปกรณ์ธุรกิจใน Intune](https://docs.microsoft.com/intune/custom-settings-windows-holographic)
