# <a name="manage-who-can-create-microsoft-365-groups"></a>จัดการบุคคลที่สามารถสร้างกลุ่ม Microsoft ๓๖๕

ตามค่าเริ่มต้นใครก็ตามที่อยู่ในองค์กรของคุณสามารถสร้างกลุ่ม O365 ได้
  
ถ้าคุณต้องการเปลี่ยนแปลงที่คุณสามารถสร้างกลุ่มความปลอดภัยด้วย PowerShell ที่จำกัดการสร้างกลุ่มให้กับชุดย่อยของผู้ใช้
  
ทำตามคำแนะนำทีละขั้นตอนที่นี่: [จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
