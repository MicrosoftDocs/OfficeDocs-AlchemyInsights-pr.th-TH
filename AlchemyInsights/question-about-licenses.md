# <a name="questions-about-your-microsoft-365-license"></a>คําถามเกี่ยวกับสิทธิ์การใช้งาน Microsoft 365 ของคุณ

ถ้าคุณต้องการทราบว่าคุณได้ซื้อสิทธิ์การใช้งานจํานวนเท่าใด หรือจํานวนสิทธิ์การใช้งานที่ได้รับมอบหมายจากการสมัครใช้งาน:
  
1. ในศูนย์การจัดการ ให้ไปที่หน้า**การเรียกเก็บเงิน** \> **[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการทราบข้อมูลเพิ่มเติม คุณสามารถดูจํานวนใบอนุญาตที่คุณจ่ายและจํานวนใบอนุญาตที่มีให้

ถ้าคุณต้องการเพิ่มสิทธิ์การใช้งานเพิ่มเติม หรือถ้าคุณมีมากเกินไป ให้เลือก**เพิ่ม/เอาสิทธิ์การใช้งานออก**เพื่อปรับจํานวนเงินที่ซื้อ
