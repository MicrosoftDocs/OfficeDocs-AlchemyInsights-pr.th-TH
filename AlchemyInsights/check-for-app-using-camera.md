# <a name="check-for-app-using-camera"></a>ตรวจหาแอปโดยใช้กล้อง

1. เลือก **เริ่ม**จากนั้นเลือก **การตั้งค่า > กล้อง > ความเป็นส่วนตัว** (การตั้งค่า ms: ความเป็นส่วนตัว-เว็บแคม)
2. ในรายการแอปพลิเคชันให้ค้นหาแอปที่มี "กำลังใช้งานอยู่" และปิดแอปพลิเคชัน
