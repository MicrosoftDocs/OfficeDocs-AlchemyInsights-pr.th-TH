# <a name="microsoft-365-e1-subscription-and-license-management-faq"></a>ถามที่ถามบ่อยเกี่ยวกับการสมัครใช้งาน Microsoft 365 E1 และการจัดการสิทธิ์การใช้งาน

- เมื่อต้องการเปรียบเทียบ Microsoft 365 E1, E3, E5[ให้ดูเปรียบเทียบแผน Office 365 Enterprise](https://www.microsoft.com/microsoft-365/business/compare-more-office-365-for-business-plans)
- เมื่อต้องการ **ซื้อ** E1 ให้ไปที่ **หน้า>[บริการการซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)** ใช้กล่องค้นหาเพื่อค้นหาบน "E1"; เลือกแผนและจํานวนผู้ใช้ [ซื้อการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/try-or-buy-microsoft-365#buy-a-different-subscription)อื่น
- เมื่อต้องการ **เปลี่ยน** แผน [ให้ดู อัปเกรดเป็นแผน](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/upgrade-to-different-plan)อื่น
- เมื่อต้องการ **กําหนด** สิทธิ์การใช้งาน E1 ให้กับผู้ใช้ ให้ไปที่ **>[สิทธิ์การใช้งาน](https://go.microsoft.com/fwlink/p/?linkid=842264)** For more info, see [Assign licenses to users](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users).
- เมื่อต้องการ **ซื้อหรือลบ** สิทธิ์การใช้งาน E1 ให้ไปที่ หน้า **การเรียกเก็บเงิน > [ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)** ของคุณ ดูข้อมูลเพิ่มเติมได้ที่ [ซื้อหรือเอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)ออก
- เมื่อต้องการ **ยกเลิก** การสมัครใช้งาน E1 ให้ไปที่ **หน้าการเรียกเก็บเงิน >[ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)** ของคุณ เลือกการสมัครใช้งาน และ **ในส่วนการตั้งค่าการสมัครใช้งานและการชเงิน****เลือก** ยกเลิกการสมัครใช้งาน ดูข้อมูลเพิ่มเติมได้ที่ [ยกเลิกการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)ของคุณ
- เมื่อต้องการ **เปิดใช้งาน** การสมัครใช้งาน E1 ให้ดู [เปิดใช้งานการสมัครใช้งาน](https://docs.microsoft.com/alchemyinsights/activate-your-office-365-subscription)ของคุณ
- เมื่อต้องการ **เปิดใช้งานการสมัครใช้งาน** E1 ใหม่ [ให้ดู เปิดใช้งานการสมัครใช้งานของคุณ](https://docs.microsoft.com/alchemyinsights/reactivate-your-subscription)ใหม่
- To **extend** an E1 trial, see [Extend your trial](https://docs.microsoft.com/microsoft-365/commerce/extend-your-trial).

ดูข้อมูลเพิ่มเติมได้ที่ [ซื้อหรือเอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)ออก
