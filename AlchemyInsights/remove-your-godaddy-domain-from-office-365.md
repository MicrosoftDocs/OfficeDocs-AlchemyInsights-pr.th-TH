# <a name="remove-your-godaddy-domain-from-microsoft-365"></a>เอาโดเมน GoDaddy ของคุณจาก Microsoft ๓๖๕

เราไม่สามารถเอาโดเมน GoDaddy ของคุณจาก Microsoft ๓๖๕ให้คุณได้ ติดต่อ [ฝ่ายบริการลูกค้าของ GoDaddy](https://aka.ms/contact-godaddy) เพื่อขอความช่วยเหลือ
  
ถ้าชื่อองค์กรไม่มี "NETORG" หรือไม่ได้ซื้อจาก GoDaddy ให้เอาโดเมนของคุณโดยทำตามขั้นตอนต่อไปนี้ [เพื่อเอาโดเมนออกจาก Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/remove-a-domain)
