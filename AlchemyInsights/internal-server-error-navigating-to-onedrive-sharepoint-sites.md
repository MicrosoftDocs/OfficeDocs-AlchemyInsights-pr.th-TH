# <a name="internal-server-error-when-navigating-to-sharepoint-or-onedrive-sites"></a>ข้อผิดพลาดภายในเซิร์ฟเวอร์เมื่อนำทางไปยังไซต์ Sharepoint หรือ OneDrive

ผู้ใช้อาจได้รับข้อผิดพลาดภายในเซิร์ฟเวอร์๕๐๐เมื่อพยายามนำทางไปยังไซต์ SharePoint หรือ OneDrive 

โปรดลองล้างแคชของเบราว์เซอร์ก่อนที่จะนำทางไปยังไซต์


1. ในเบราว์เซอร์ Microsoft Edge ให้ไปที่เพิ่มเติม ... > การตั้งค่า

2. ภายใต้ล้างข้อมูลการเรียกดูให้เลือก **เลือกสิ่งที่ต้องการล้าง**

3. เลือกกล่องกาเครื่องหมายคุกกี้และข้อมูลเว็บไซต์ที่บันทึกไว้แล้วเลือก**ล้าง**

หมายเหตุ: ขั้นตอนเหล่านี้อาจแตกต่างกันเมื่อใช้เบราว์เซอร์อื่นเช่น Firefox หรือ Chrome

ถ้าวิธีนี้ไม่สามารถแก้ไขปัญหาได้คุณสามารถตรวจสอบว่ามีการบำรุงรักษาที่ใช้งานอยู่ในผู้เช่าของคุณโดยการนำทางไปยัง [ศูนย์ข้อความ](https://portal.office.com/adminportal/home#/MessageCenter)หรือไม่

สุดท้ายให้ตรวจสอบให้แน่ใจว่าคุณได้เยี่ยมชมหน้า [สถานภาพบริการ](https://portal.office.com/adminportal/home#/servicehealth) เพื่อตรวจสอบคำแนะนำ/กรณีปัญหาที่อาจเกิดขึ้น
