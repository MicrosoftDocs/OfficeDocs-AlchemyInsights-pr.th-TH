# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>สไลด์ PowerPoint ไม่แสดงใน Skype สําหรับธุรกิจบนอุปกรณ์ Android

หากคุณพบปัญหากับสไลด์ PowerPoint ไม่แสดงบนอุปกรณ์ Android อาจเป็นเพราะข้อผิดพลาดในเบราว์เซอร์ Chrome

ลองปิดใช้งาน แล้วถอนการติดตั้ง Chrome หลังจากที่ถอนการติดตั้ง Chrome แล้ว ให้เปิดใช้งาน Chrome อีกครั้งเพื่อให้เวอร์ชันของโรงงาน (74.0.3729.136) ซึ่งแก้ปัญหาได้
