# <a name="use-your-browser-to-check-spelling"></a>ใช้เบราว์เซอร์ของคุณเพื่อตรวจสอบการสะกด

แม้ว่าOutlook.comไม่มีฟังก์ชันการตรวจสอบการสะกด แต่เว็บเบราว์เซอร์จํานวนมากจะตรวจสอบการสะกดคําของคุณโดยอัตโนมัติ รวมทั้ง Microsoft Edge, Internet Explorer 10 และรุ่นที่ใหม่กว่า และ Firefox เวอร์ชันปัจจุบัน ตรวจสอบตัวเลือกสําหรับเว็บเบราว์เซอร์ของคุณเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีตรวจการสะกด
