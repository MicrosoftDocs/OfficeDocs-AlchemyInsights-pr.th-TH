# <a name="prepaid-microsoft-365-renewal"></a>การต่ออายุ Microsoft 365 แบบเติมเงิน

1. ในศูนย์การจัดการ ให้ไปที่หน้า**การเรียกเก็บเงิน** \> **[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการขยาย ในส่วน**การเรียกเก็บเงิน**ให้เลือก**ขยายวันที่สิ้นสุด**

3. ในบานหน้าต่าง**ขยายการบอกรับสมาชิก**ให้เลือก**ใช้คีย์ผลิตภัณฑ์ใหม่และหมายเลขผลิตภัณฑ์ที่ไม่ได้ใช้**แล้วเลือก**ถัดไป**

4. ป้อนหมายเลขผลิตภัณฑ์ แล้วเลือก**ถัดไป**

> [!NOTE]
> ถ้าคุณมีหมายเลขผลิตภัณฑ์มากกว่าหนึ่งหมายเลข คุณสามารถเลือก**เพิ่มหมายเลขผลิตภัณฑ์อื่น**เพื่อป้อนรหัสผลิตภัณฑ์เหล่านั้นได้

5. ตรวจสอบรายละเอียดคําสั่งซื้อของคุณ แล้วเลือก**แลกใช้**

เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[การเพิ่มสิทธิ์การใช้งานหรือขยายการสมัครใช้งานที่ชําระเงินสําหรับการใช้คีย์ผลิตภัณฑ์](https://docs.microsoft.com/microsoft-365/commerce/licenses/add-licenses-using-product-key)
