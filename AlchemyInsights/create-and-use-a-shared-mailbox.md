# <a name="create-and-use-a-shared-mailbox"></a>สร้าง และใช้กล่องจดหมายที่ใช้ร่วมกัน

กล่องจดหมายที่ใช้ร่วมกันช่วยให้ส่ง และรับอีเมลสำหรับแฝง เช่น sales@contoso.com โดยไม่มอบหมายสิทธิ์การใช้งานอื่นมากกว่าหนึ่งคน
  
- [ไปที่นี่เมื่อต้องการเริ่มต้นใช้งาน](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

คุณยังสามารถอ่านบทความ[สร้างกล่องจดหมายที่ใช้ร่วมกัน](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx)ซึ่งจะบอกให้คุณทราบทุกสิ่งที่คุณจำเป็นต้องทราบเกี่ยวกับกล่องจดหมายที่ใช้ร่วมกัน รวมทั้งวิธีการใช้
