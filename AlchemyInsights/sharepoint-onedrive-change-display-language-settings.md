# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>เปลี่ยนการตั้งค่าภาษาที่แสดงใน SharePoint หรือ OneDrive 

คุณไม่สามารถเปลี่ยนการตั้งค่าภาษาเริ่มต้นสำหรับ SharePoint หรือ OneDrive ได้เนื่องจากการตั้งค่าเหล่านั้นถูกกำหนดเมื่อคุณสร้างการสมัครใช้งาน Microsoft ๓๖๕ของคุณ อย่างไรก็ตามคุณสามารถเปลี่ยนการตั้งค่าภาษาที่แสดงของคุณได้

เมื่อต้องการเปลี่ยนการตั้งค่าส่วนบุคคลของคุณให้ดูที่[เปลี่ยนการตั้งค่าภาษาและภูมิภาคส่วนตัวของคุณ](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7)

เมื่อต้องการเปลี่ยนส่วนติดต่อผู้ใช้ของไซต์ให้ดูที่[เลือกภาษาที่คุณต้องการทำให้พร้อมใช้งานสำหรับส่วนติดต่อผู้ใช้ของไซต์](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8)

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการแก้ไขปัญหาการตั้งค่าภาษาให้ดูที่ [คุณไม่สามารถเปลี่ยนภาษาเริ่มต้นสำหรับไซต์คอลเลกชัน SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)ได้
