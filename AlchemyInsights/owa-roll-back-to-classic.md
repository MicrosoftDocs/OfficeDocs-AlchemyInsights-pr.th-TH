# <a name="the-new-outlook-on-the-web"></a>Outlook ใหม่บนเว็บ

ไม่สามารถย้อนกลับไปยัง Outlook แบบคลาสสิกบนเว็บได้อีกต่อไป หากคุณต้องการให้คําติชมเกี่ยวกับประสบการณ์ใหม่ ให้ไปที่[ฟอรัม UserVoice](https://go.microsoft.com/fwlink/?linkid=2103182) คุณสามารถโหวตในแนวคิดหรือประเด็นที่คุณต้องการเห็นและเราจะตอบความคิดเห็นและข้อเสนอแนะโดยตรง

[รับความช่วยเหลือเกี่ยวกับ Outlook ใหม่บนเว็บ](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
