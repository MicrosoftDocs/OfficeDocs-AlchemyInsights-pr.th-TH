# <a name="create-an-email-signature-in-outlook-2016"></a>สร้างลายเซ็นอีเมลใน Outlook 2016

ใน Outlook ลายเซ็นอีเมลของคุณสามารถประกอบด้วยข้อความ รูป นามบัตรอิเล็กทรอนิกส์ของคุณ โลโก้ หรือแม้แต่รูปลายเซ็นแบบลายมือของคุณ

1. ใน Outlook ให้เปิดข้อความอีเมลใหม่
2. **บนแท็บ** ข้อความ **ให้เลือก**  >  **ลายเซ็น**
3. **ภายใต้ เลือกลายเซ็น** ที่จะ **แก้ไข** ให้เลือก สร้าง แล้วพิมพ์ชื่อลายเซ็น
4. ภายใต้ **แก้ไข** ลายเซ็น ให้เขียนลายเซ็นของคุณ
