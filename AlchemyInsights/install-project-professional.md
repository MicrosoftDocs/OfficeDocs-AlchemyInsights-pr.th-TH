# <a name="install-project-professional"></a>ติดตั้งโปรเจ็กต

ดู[ติดตั้งโครงการ](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281)เพื่อเรียนรู้วิธีติดตั้งรุ่นสมัครสมาชิกของ Project Online และโครงการ 2019 รุ่นที่ไม่มีการบอกรับเป็นสมาชิก 

สําหรับโครงการ 2010 หรือ 2007 ให้ทําตามขั้นตอนใน[Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) หรือ [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657) 

โครงการจะขายเป็นโปรแกรมแบบสแตนด์อะโลนดังนั้นหากคุณยังไม่มีมัน [ซื้อหรือลองโครงการ](https://products.office.com/project)
