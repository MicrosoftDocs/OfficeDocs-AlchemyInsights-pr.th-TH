# <a name="the-new-outlook-on-the-web"></a>Outlook ใหม่บนเว็บ

คุณจะไม่สามารถกลับไปยัง Outlook แบบคลาสสิกบนประสบการณ์การท่องเว็บ ถ้าคุณต้องการส่งคำติชมเกี่ยวกับประสบการณ์ใหม่ ไปยัง[ฟอรัม UserVoice](https://outlook.uservoice.com/forums/313228--outlook-on-the-web-office-365) คุณสามารถลงคะแนนให้กับแนวคิดเกี่ยวกับ หรือการตัดสินค้าจากคลังที่คุณต้องการดูส่ง และเราจะตอบสนองกับข้อคิดเห็นด้านบนและคำแนะนำโดยตรง

[รับความช่วยเหลือเกี่ยวกับ Outlook ใหม่บนเว็บ](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
