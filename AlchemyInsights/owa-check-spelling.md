# <a name="use-your-browser-to-check-spelling"></a>ใช้เบราว์เซอร์ของคุณเพื่อตรวจสอบการสะกด

ไม่มีตัวเลือกการตรวจสอบการสะกดใน Outlook บนเว็บ เมื่อต้องการตรวจการสะกดของคุณคุณจำเป็นต้องใช้เว็บเบราว์เซอร์ของคุณ การตรวจการสะกดจะพร้อมใช้งานใน Microsoft Edge, Internet Explorer 10 และเวอร์ชันที่ใหม่กว่าและเวอร์ชันปัจจุบันของ Firefox, Chrome และ Safari ตรวจสอบตัวเลือกสำหรับเว็บเบราว์เซอร์ของคุณเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีการตรวจสอบการสะกด
