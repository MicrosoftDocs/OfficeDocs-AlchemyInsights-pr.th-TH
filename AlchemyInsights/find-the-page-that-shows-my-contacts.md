# <a name="find-the-page-that-shows-my-contacts"></a>ค้นหาหน้าที่แสดงที่ติดต่อของฉัน

คุณสามารถค้นหาผู้ติดต่อของคุณได้ด้วยวิธีเดียวกับที่คุณค้นหาอีเมลของคุณ
 
แตะไอคอนค้นหาแว่นขยายจากนั้นพิมพ์คำขอการค้นหาที่ติดต่อของคุณลงในกล่องที่ด้านบนของหน้าจอของคุณ (**Protip**: คุณสามารถแตะสองครั้งบนแว่นขยายเพื่อนำโฟกัสไปยังกล่องค้นหาได้) ผลลัพธ์ของคุณจะปรากฏในส่วนที่ติดต่อ
