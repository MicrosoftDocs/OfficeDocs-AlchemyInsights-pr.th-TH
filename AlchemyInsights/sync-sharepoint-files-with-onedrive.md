# <a name="troubleshoot-open-with-explorer-issues-in-sharepoint-online"></a>การแก้ไขปัญหา "เปิดด้วย Explorer" ใน SharePoint Online

เราขอแนะนำให้ [ซิงค์ไฟล์ SharePoint กับไคลเอ็นต์การซิงค์ OneDrive ใหม่](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) ที่มี [ไฟล์ตามความต้อง](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e) การเนื่องจากมีการเข้าถึงไฟล์ของคุณภายในเครื่องของคุณและมีประสิทธิภาพการทำงานที่ดีที่สุด

เมื่อต้องการแก้ไขปัญหาการเปิดด้วย Explorer ให้ทำตามขั้นตอนและแนวทางปฏิบัติที่ดีที่สุดในบทความต่อไปนี้:

- [วิธีใช้คำสั่ง "เปิดด้วย Explorer" เพื่อแก้ไขปัญหาใน SharePoint Online](https://docs.microsoft.com/sharepoint/support/lists-and-libraries/troubleshoot-issues-using-open-with-explorer)
- [คัดลอกหรือย้ายไฟล์ไลบรารีโดยใช้เปิดด้วย Explorer](https://support.office.com/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2)

> **บันทึกย่อ**
>- เปิดด้วย Explorer เท่านั้นที่ได้รับการสนับสนุนใน Internet Explorer 10 หรือ11 เปิดด้วย Explorer ไม่ทำงานใน Windows ด้วย Microsoft Edge, Google Chrome, Mozilla Firefox หรือบนแพลตฟอร์ม Mac เนื่องจากสาเหตุนี้ตัวเลือกมุมมอง Explorer อาจเป็นสีเทา
>
>- ปุ่มเปิดด้วย Explorer ไม่ปรากฏในประสบการณ์การใช้งานไลบรารีใหม่ เลือกเมนูดรอปดาวน์**มุมมอง**ที่มุมขวาบน (ชื่อของการเปลี่ยนแปลงแบบหล่นลงทั้งนี้ขึ้นอยู่กับมุมมองปัจจุบันของคุณ) จากนั้นเลือก**ดูใน File Explorer**
