# <a name="cant-sign-in-to-outlookcom"></a>ไม่สามารถลงชื่อเข้าใช้ Outlook.com

ถ้าคุณกำลังมีปัญหาในการลงชื่อเข้าใช้บัญชีผู้ใช้ของคุณให้ตรวจสอบให้แน่ใจว่าบัญชีผู้ใช้ของคุณไม่ได้ถูกปิดใช้งานเป็นเวลามากกว่า๓๖๕วัน หลังจาก๓๖๕วันที่ไม่ได้ใช้งานอีเมลของคุณจะถูกลบและไม่สามารถกู้คืนได้
  
บัญชีผู้ใช้ของคุณอาจถูกบล็อกชั่วคราวเนื่องจากเราสังเกตเห็นกิจกรรมการลงชื่อเข้าใช้ที่ผิดปกติ ถ้าบัญชีผู้ใช้ของคุณถูกบล็อกให้อ่านวิธีการ[ยกเลิกการบล็อกบัญชี Outlook.com ของฉัน](https://support.office.com/article/f4ad2701-d166-4d8b-8a6a-9af2a1f8a4c4?wt.mc_id=Office_Outlook_com_Alchemy)
  
ถ้าคุณสงสัยว่าบัญชีผู้ใช้ Outlook.com ของคุณถูกแฮ็กให้ดูที่[บัญชีผู้ใช้ Outlook.com ของฉันถูกแฮ็ก](https://support.office.com/article/35993ac5-ac2f-494e-aacb-5232dda453d8?wt.mc_id=Office_Outlook_com_Alchemy)
  
ถ้าคุณลืมหรือทำรหัสผ่านหายคุณสามารถ [ตั้งค่ารหัสผ่านของคุณใหม่](https://go.microsoft.com/fwlink/p/?LinkID=242804)ได้
  
อ่านเพิ่มเติมเกี่ยวกับ [เมื่อคุณไม่สามารถลงชื่อเข้าใช้บัญชี Microsoft ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=837479)ได้
