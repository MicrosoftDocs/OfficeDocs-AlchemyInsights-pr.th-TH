# <a name="changes-to-external-user-access"></a>การเปลี่ยนแปลงการเข้าถึงของผู้ใช้ภายนอก

เริ่มตั้งแต่23มีนาคม๒๐๑๘ผู้ใช้ภายนอกจะไม่สามารถเข้าถึงเนื้อหาที่มีการใช้ร่วมกันกับทุกคนผู้ใช้ที่รับรองความถูกต้องทั้งหมดหรือผู้ใช้แบบฟอร์มทั้งหมดแม้ว่าจะได้รับเชิญหรือเพิ่มกลุ่มที่อนุญาต ผู้ใช้ภายนอกจะสามารถเข้าถึงเนื้อหาที่มีการใช้ร่วมกันกับผู้ใช้หรือกลุ่มที่ผู้ใช้เป็นสมาชิกอยู่เท่านั้น หากคุณต้องการที่จะเก็บการตั้งค่าที่มีอยู่ของคุณโปรดดู:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
