# <a name="how-to-create-an-email-signature"></a>วิธีการสร้างลายเซ็นอีเมล

ไป[หน้าเรียบเรียงและตอบการตั้งค่า](https://go.microsoft.com/fwlink/?linkid=2006164)ของคุณ (**การตั้งค่า** \> **ดูการตั้งค่า Outlook ทั้งหมด** \> **เรียบเรียงและตอบ**) 
  
 **หมายเหตุ:** คุณสามารถมีลายเซ็นเดียวเท่านั้นสำหรับแต่ละบัญชี 
  
เรียนรู้เพิ่มเติมเกี่ยวกับวิธีการ[สร้าง และเพิ่มลายเซ็นอีเมลของที่ใน Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001404&amp;clcid=0x409)
