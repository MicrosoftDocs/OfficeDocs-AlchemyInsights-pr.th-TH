# <a name="sharepoint-publishing-infrastructure"></a>ประกาศโครงสร้างพื้นฐานของ SharePoint


<p>SharePoint มีชุดของคุณลักษณะที่ช่วยให้คุณสามารถสร้าง และเผยแพร่เว็บเพจที่ rich สำหรับอินทราเน็ตของคุณ แบบคลาสสิก เรียกใช้ SharePoint ประกาศโครงสร้างพื้นฐานในระดับไซต์คอลเลกชันเพื่อเปิดใช้งาน</p>  <p>สำหรับข้อมูลเพิ่มเติมโปรดเยี่ยมชมลิงค์ด้านล่าง</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">เปิดใช้งานคุณลักษณะการประกาศ</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">กำลังประกาศคุณลักษณะ</a></li>  </ul>  <p>หมายเหตุ:</p>  <ul>  <li>แม่แบบไซต์ที่เผยแพร่คลาสสิกรวมคุณลักษณะการประกาศ โดยค่าเริ่มต้น</li>  <li>กำลังประกาศคุณลักษณะแบบคลาสสิกไม่พร้อมใช้งานบนไซต์สมัยใหม่ ไซต์แบบสมัยใหม่ให้เป็นอีกทางเลือกหนึ่งที่ใหม่กว่า โดยค่าเริ่มต้น</li>  </ul>
