# <a name="prevent-files-from-being-downloaded-from-sharepoint-or-onedrive"></a>ป้องกันไม่ให้มีการดาวน์โหลดแฟ้มจาก SharePoint หรือ OneDrive

มีวิธีการต่างๆ ที่อาจใช้เพื่อป้องกันหรือบล็อกการดาวน์โหลดเกิดขึ้น กรุณาเยี่ยมชมลิงก์ด้านล่างสําหรับวิธีการที่เกี่ยวข้อง:

- [บล็อกการดาวน์โหลดแฟ้มด้วยการเข้าถึงแบบมีเงื่อนไขของ Azure AD](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad#create-a-block-download-policy-for-unmanaged-devices)

- [ควบคุมการเข้าถึงจากอุปกรณ์ที่ไม่มีการจัดการ](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)

- [กําหนดค่านโยบาย IRM บนไลบรารีเอกสาร SharePoint เพื่อจํากัดการดาวน์โหลดเนื้อหา](https://docs.microsoft.com/office365/securitycompliance/set-up-irm-in-sp-admin-center)
