# <a name="too-many-licenses"></a>สิทธิ์การใช้งานมากเกินไปหรือไม่

ถ้าคุณคิดว่า คุณกำลังมีการออกบิลสำหรับสิทธิ์การใช้งานมากเกินไป โดยทั่วไปเนื่องจากไม่ได้ซื้อสิทธิ์การใช้งานจาก นั้นคุณจำเป็นต้อง
  
 **ลดจำนวนของสิทธิ์การใช้งานที่ซื้อ**
  
1. ในศูนย์ดูแล ไปหน้าการ[บอกรับเป็นสมาชิก](https://go.microsoft.com/fwlink/p/?linkid=842054)หรือเลือก**การเรียกเก็บเงิน** \> **บอกรับเป็นสมาชิก**
    
2. ทางด้านซ้าย เลือกการบอกรับเป็นสมาชิกที่คุณต้องการเอาสิทธิ์การใช้งาน
    
3. เลือก**เพิ่ม/เอาออกสิทธิ์การใช้งาน**
    
4. ป้อนจำนวนของผู้ใช้ที่จำเป็น ตัวอย่างเช่น ถ้าคุณอยู่ในขณะนี้มีสิทธิ์การใช้งาน 25 และคุณต้องการเอา 5 ป้อน 20 ในฟิลด์นี้
    
5. เลือก**ส่ง**
