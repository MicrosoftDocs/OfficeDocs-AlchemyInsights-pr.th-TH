# <a name="create-users-in-hybrid-deployments"></a>สร้างผู้ใช้ในการปรับใช้ไฮบริดสลี

เมื่อองค์กรของคุณมีการปรับใช้ไฮบริดสลีระหว่างมีสภาพแวดล้อมในสถานที่และ Office 365 คุณจำเป็นต้องสร้างวัตถุผู้ใช้ทั้งหมดในไดเรกทอรีที่ใช้งานอยู่ในสถานของคุณ คุณสามารถสร้างวัตถุผู้ใช้ในหน่วยงานที่เหมาะสม (OU) เพื่อที่จะซิงโครไนส์ลงในองค์กรของคุณ Office 365
