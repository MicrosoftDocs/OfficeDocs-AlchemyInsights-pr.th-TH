# <a name="control-creation-of-microsoft-365-groups"></a>การสร้างตัวควบคุมของกลุ่ม Microsoft 365

โดยค่าเริ่มต้น ผู้ใช้ทั้งหมดภายในบริษัทสามารถสร้างกลุ่มได้ ถ้าจําเป็น คุณสามารถเปลี่ยนแปลงและมอบหมายสิทธิ์ในการสร้างกลุ่มให้กับกลุ่มของบุคคลภายในบริษัทของคุณ โปรดสังเกตว่า นี้จะมีผลกระทบต่อการสร้างกลุ่มจากบริการใด ๆ ของ Microsoft 365 (เช่น Outlook, ทีม, วางแผนและอื่น ๆ)
  
คําแนะนํา:[จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
