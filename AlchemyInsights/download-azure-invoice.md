# <a name="download-azure-invoice"></a>ดาวน์โหลดใบแจ้งหนี้ Azure

คุณต้องมีเจ้าของผู้สนับสนุนโปรแกรมอ่านหรือบทบาทผู้จัดการใบแจ้งหนี้ในโปรไฟล์การเรียกเก็บเงินเพื่อดาวน์โหลดใบแจ้งหนี้ในพอร์ทัล Azure ถ้าคุณมีบทบาทของเจ้าของผู้สนับสนุนหรือผู้อ่านในบัญชีการเรียกเก็บเงินคุณสามารถดาวน์โหลดใบแจ้งหนี้ของโปรไฟล์การเรียกเก็บเงินทั้งหมดที่เป็นส่วนหนึ่งของบัญชีผู้ใช้การเรียกเก็บเงินได้

1. ลงชื่อเข้าใช้[พอร์ทัล Azure](https://portal.azure.com/)
2. ค้นหา **การจัดการต้นทุน + การเรียกเก็บเงิน**
3. เลือก **โปรไฟล์การเรียกเก็บเงิน** จากด้านซ้าย
4. จากรายการโปรไฟล์การเรียกเก็บเงินให้เลือกโปรไฟล์การเรียกเก็บเงินที่คุณต้องการดาวน์โหลดใบแจ้งหนี้ Azure
5. คลิก **ดาวน์โหลดใบแจ้งหนี้** เพื่อดูสำเนาใบแจ้งหนี้ PDF ของคุณ ถ้า **ไม่มี** ให้ดู [ที่ทำไมฉันจึงไม่เห็นใบแจ้งหนี้สำหรับช่วงเวลาการเรียกเก็บเงินล่าสุด](https://docs.microsoft.com/azure/cost-management-billing/manage/download-azure-invoice-daily-usage-date)

คุณยังสามารถดูการใช้งานประจำวันของคุณได้โดยการเลือกช่วงเวลาการเรียกเก็บเงิน เมื่อต้องการรับ PDF ของใบแจ้งหนี้ของคุณและสำเนาของไฟล์การใช้งานรายละเอียดของคุณ ( CSV):[รับข้อมูลใบแจ้งหนี้และการใช้งาน](https://docs.microsoft.com/azure/cost-management-billing/manage/download-azure-invoice-daily-usage-date)
