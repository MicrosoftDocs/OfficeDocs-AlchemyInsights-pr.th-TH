# <a name="add-or-remove-an-email-address-for-a-user"></a>เพิ่มหรือลบที่อยู่ e-mail สำหรับผู้ใช้

[ทำตามขั้นตอนเหล่านี้](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)ในศูนย์ดูแลของ Microsoft ๓๖๕เพื่อเริ่มต้นใช้งาน

 **หมายเหตุ**: คุณสามารถมีที่อยู่ e-mail มากกว่าหนึ่งที่สำหรับผู้ใช้แต่ชื่อแทนของ*email*เพิ่มเติมสามารถใช้สำหรับการรับเมลได้เท่านั้น ถ้าคุณต้องการส่งเมลด้วยนามแฝงให้สร้าง[กล่องจดหมายที่ใช้ร่วมกัน](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9) ด้วยวิธีนี้คุณจะได้รับนามแฝงในการส่งและรับจดหมาย
