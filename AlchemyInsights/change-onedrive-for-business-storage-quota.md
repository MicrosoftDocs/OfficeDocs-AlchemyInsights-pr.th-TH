# <a name="change-the-default-onedrive-storage-space-for-your-users"></a>เปลี่ยนพื้นที่การจัดเก็บ OneDrive เริ่มต้นสำหรับผู้ใช้ของคุณ

1. ไป[หน้าการจัดเก็บของศูนย์ admin OneDrive](https://admin.onedrive.com/?v=StorageSettings)
    
2. ป้อนยอดเงินเก็บข้อมูลเริ่มต้น (ในกิกะไบต์) ในกล่องที่**เก็บเริ่มต้น**และจากนั้น คลิก**บันทึก**
    
การตั้งค่าพื้นที่จัดเก็บข้อมูลนี้นำไปใช้กับผู้ใช้ทั้งหมดที่คุณไม่ได้ตั้งค่าขีดจำกัดการจัดเก็บเฉพาะ เมื่อต้องการเปลี่ยนพื้นที่การจัดเก็บสำหรับผู้ใช้บางคน คุณจำเป็นต้องใช้ Microsoft PowerShell สำหรับข้อมูลเกี่ยวกับวิธีการทำเช่นนี้ ดู[พื้นที่จัดเก็บ OneDrive ของผู้ใช้เปลี่ยนแปลงโดยใช้ PowerShell](https://go.microsoft.com/fwlink/?linkid=866402)
