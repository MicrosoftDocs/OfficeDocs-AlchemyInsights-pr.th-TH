# <a name="how-to-select-all-in-outlookcom"></a>วิธี "เลือกทั้งหมด" ใน Outlook.com

คุณสามารถเลือกทั้งหมดได้ถ้าข้อความทั้งหมดสามารถมองเห็นได้ นี่คือการป้องกันไม่ให้มีการดำเนินการโดยไม่ตั้งใจบนข้อความที่คุณไม่เห็น

**CTRL + คลิก** เลือกแต่ละข้อความหลายข้อความ

**SHIFT + คลิก** เลือกข้อความทั้งหมดระหว่างข้อความแรกและข้อความสุดท้ายที่คุณเลือก
