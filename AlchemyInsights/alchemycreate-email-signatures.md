# <a name="create-email-signatures"></a>สร้างลายเซ็นอีเมล

ดูหัวข้อต่อไปนี้เพื่อเรียนรู้วิธีสร้างลายเซ็นอีเมลใน Outlook
  
- สร้างลายเซ็นอีเมลสําหรับข้อความของคุณเอง:[สร้างและเพิ่มลายเซ็นลงในข้อความ](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)หรือ[สร้างและเพิ่มลายเซ็นอีเมลใน Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx)
    
- สร้างลายเซ็นอีเมลส่วนกลางสําหรับข้อความทั้งหมดที่ส่งจากบริษัทของคุณ (หรือที่เรียกอีกอย่างว่าคํา[ปฏิเสธความรับผิดชอบ):](https://go.microsoft.com/fwlink/p/?linkid=391096)
