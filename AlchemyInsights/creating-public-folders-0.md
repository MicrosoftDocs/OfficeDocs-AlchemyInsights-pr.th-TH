# <a name="creating-public-folders"></a>การสร้างโฟลเดอร์สาธารณะ

ก่อนที่ผู้ใช้ของคุณจะสามารถใช้โฟลเดอร์สาธารณะได้คุณจะต้องสร้างกล่องจดหมายโฟลเดอร์สาธารณะอย่างน้อยหนึ่งกล่อง เมื่อต้องการสร้าง:
  
1. ในศูนย์การจัดการ Microsoft ๓๖๕ให้ไปที่**Admin centers** \> **Exchange**ศูนย์การจัดการ

2. นำทางไปยังกล่องจดหมายโฟลเดอร์**สาธารณะโฟลเดอร์สาธารณะ** \> **Public folder mailboxes**แล้วคลิก**เพิ่ม**

3. ใน **กล่องจดหมายโฟลเดอร์สาธารณะ**ให้ใส่ชื่อสำหรับกล่องจดหมายโฟลเดอร์สาธารณะ

4. คลิก **บันทึก**

เมื่อมีการสร้างกล่องจดหมายแล้วคุณสามารถเริ่มสร้างแต่ละโฟลเดอร์หรือแจ้งให้ผู้ใช้ของคุณ [เริ่มสร้างและใช้โฟลเดอร์สาธารณะใน Outlook](https://support.office.com/article/Create-and-share-a-public-folder-in-Outlook-a2835011-d524-4a5c-a207-05c159bb2a97)ได้
