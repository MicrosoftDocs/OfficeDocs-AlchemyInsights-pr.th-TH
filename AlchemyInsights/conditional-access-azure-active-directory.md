# <a name="conditional-access"></a>การเข้าถึงแบบมีเงื่อนไข

การเข้าถึงตามเงื่อนไขคือความสามารถของ Azure Active Directory ด้วยการเข้าถึงแบบมีเงื่อนไขคุณสามารถใช้การตัดสินใจเกี่ยวกับการควบคุมการเข้าถึงโดยอัตโนมัติสำหรับการเข้าถึงแอป cloud ของคุณที่ยึดตามเงื่อนไข สำหรับข้อมูลเพิ่มเติมโปรดไป[ที่การกำหนดค่านโยบายที่ยึดตามตำแหน่งที่ตั้ง](https://docs.microsoft.com/azure/active-directory/conditional-access/overview)

ในฐานะผู้ดูแลระบบ IT คุณยังสามารถควบคุมการเข้าถึงทรัพยากร SharePoint และ OneDrive โดยยึดตามตำแหน่งที่ตั้งเครือข่ายที่กำหนดหรืออุปกรณ์ที่คุณเชื่อถือได้ สำหรับข้อมูลเพิ่มเติมโปรดไปที่ลิงก์ด้านล่าง

- [การควบคุมการเข้าถึงข้อมูล SharePoint Online และ OneDrive โดยยึดตามตำแหน่งที่ตั้งเครือข่าย](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)

- [การควบคุมการเข้าถึงจากอุปกรณ์ที่ไม่มีการจัดการ](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
