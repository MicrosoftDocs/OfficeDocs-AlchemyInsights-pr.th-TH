# <a name="teams-and-cloud-video-interop-cvi"></a>ทีมและการทํางานร่วมกันของวิดีโอคลาวด์ (CVI)

Cloud Video Interop (CVI) เป็นโซลูชันของบริษัทอื่นที่มีคุณสมบัติเหมาะสมของ Microsoft ที่เปิดใช้งานห้องประชุม (TelePresence) และอุปกรณ์วิดีโอส่วนบุคคล (VTCs) เพื่อเข้าร่วมการประชุม Microsoft Teams

- สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การทํางานร่วมกันของวิดีโอบนระบบคลาวด์สําหรับ Microsoft Teams](https://docs.microsoft.com/microsoftteams/cloud-video-interop)
