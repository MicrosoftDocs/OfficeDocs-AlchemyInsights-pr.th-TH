# <a name="outlook-on-the-web-account-hacked"></a>Outlook บนบัญชีเว็บที่ถูกแฮ็ก

หากบัญชีของคุณถูกแฮ็ก ให้ทําตามคําแนะนําใน[วิธีการรักษาความปลอดภัยและคืนค่าฟังก์ชันอีเมลไปยังบัญชี Microsoft 365 ที่น่าสงสัยและกล่องจดหมายที่ถูกบุกรุก](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account#how-to-secure-and-restore-email-function-to-a-suspected-compromised-office-365-account-and-mailbox)

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับขั้นตอนเพิ่มเติมที่คุณสามารถใช้เพื่อป้องกันไม่ให้บัญชีของคุณถูกบุกรุก ให้ดูที่[การตอบสนองต่อบัญชีอีเมลที่ถูกบุกรุกใน Microsoft 365](https://docs.microsoft.com/office365/securitycompliance/responding-to-a-compromised-email-account)
