# <a name="open-and-use-a-shared-mailbox-in-outlook"></a>เปิดและใช้กล่องจดหมายที่แชร์ใน Outlook

กล่องจดหมายที่แชร์จะอนุญาตให้กลุ่มบุคคลสามารถตรวจสอบและส่งอีเมลจากที่อยู่อีเมลที่ใช้กันทั่วไป 

**หมายเหตุ**: ผู้ดูแลระบบ Microsoft ๓๖๕ของคุณจำเป็นต้องสร้างกล่องจดหมายที่แชร์และเพิ่มคุณเป็นสมาชิก

ดูสิ่งต่อไปนี้เพื่อเรียนรู้วิธีการเปิดและใช้กล่องจดหมายที่แชร์:

- [เพิ่มกล่องจดหมายที่แชร์ไปยัง Outlook บนเว็บ](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)

- [เปิดและใช้กล่องจดหมายที่แชร์ใน Outlook สำหรับ Windows](https://support.office.com/article/open-and-use-a-shared-mailbox-in-outlook-d94a8e9e-21f1-4240-808b-de9c9c088afd)

- [เปิดโฟลเดอร์หรือกล่องจดหมายที่แชร์ใน Outlook for Mac](https://support.office.com/article/Open-a-shared-folder-or-mailbox-in-Outlook-for-Mac-6ecc39c5-5577-4a1d-b18c-bbdc92972cb2)

- [เพิ่มกล่องจดหมายที่แชร์ลงใน Outlook mobile (Outlook สำหรับ iOS & Android)](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f)
