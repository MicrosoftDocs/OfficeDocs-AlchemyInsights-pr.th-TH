# <a name="use-teamviewer-to-remotely-administer-intune-devices"></a>ใช้ TeamViewer เพื่อจัดการอุปกรณ์ Intun

อุปกรณ์ที่จัดการโดย Intunine สามารถจัดการจากระยะไกลโดยใช้[TeamViewer](https://www.teamviewer.com/)

เมื่อต้องการจัดการ Intuner โดยใช้ TeamViewer ใช้ขั้นตอนเหล่านี้: 

เริ่มต้น โดยการรับข้อมูลประจําตัวจาก TeamViewer เพื่อตั้งค่าตัวเชื่อมต่อ TeamViewer บน Intun การดําเนินการเพียงครั้งเดียวเพื่อสร้างการเชื่อมโยงระหว่าง Intunen และ TeamViewer บริการ

**ส่วนที่ 1: เริ่มเซสชันด้วยอุปกรณ์ระยะไกล**

1. ภายใต้**อุปกรณ์ทั้งหมด**ให้เลือกอุปกรณ์ที่คุณต้องการเริ่มเซสชันระยะไกลด้วย
2. จาก **... เพิ่มเติม**ให้เลือก**เซสชันความช่วยเหลือระยะไกลใหม่**
3. เลือก**ใช่**เพื่อรับทราบว่าคุณต้องการสร้างเซสชันระยะไกล
    หลังจากการร้องขอ "การเริ่มต้นเซสชันระยะไกลใหม่" ได้รับการยอมรับโดยบริการ TeamViewer คุณจะเห็นตัวเลือกในการ**เริ่มความช่วยเหลือระยะไกล**ภายใต้รายละเอียดของบานหน้าต่างภาพรวม (หรือสิ่งจําเป็นสําหรับ) สําหรับอุปกรณ์ เลือก**ดูเพิ่มเติม**เพื่อขยายบานหน้าต่างและแสดงสถานะ 'ความช่วยเหลือระยะไกล'
4. เลือก**เริ่มเซสชันระยะไกล**เพื่อเริ่มเซสชันบนด้านผู้ดูแลระบบ
5. เลือกเพื่อดาวน์โหลดไบนารี TeamViewer (Windows)**และเลือก**<br/>
    **หมายเหตุ** คุณสามารถละเว้นหน้าเว็บใดๆ ที่เปิดไปยังเว็บไซต์ TeamViewer ได้

6. ยอมรับคําขอสําหรับโปรแกรมประยุกต์ TeamViewer เพื่อทําการเปลี่ยนแปลงบนอุปกรณ์ (Windows เท่านั้น)
7. แอพลิเคชัน TeamViewer เริ่มต้น และรวมถึงรหัสเซสชันการรับรองความถูกต้องการเชื่อมต่อกับอุปกรณ์ระยะไกล

**ส่วนที่ 2: บนอุปกรณ์ที่ถูกกําหนดเป้าหมายสําหรับเซสชันระยะไกล**

1. เปิดพอร์ทัลบริษัท Intun
2. มองหาธงการแจ้งเตือน: "ผู้ดูแลระบบ IT ของคุณร้องขอการควบคุมอุปกรณ์นี้สําหรับเซสชันความช่วยเหลือระยะไกล" และเลือกการแจ้งเตือน
3. เลือกดาวน์โหลดแอปพลิเคชัน TeamViewer หรือรับทราบการดาวน์โหลดแอป TeamViewer จาก App Store แล้วเลือก**เรียกใช้**
    **หมายเหตุ** คุณสามารถละเว้นหน้าเว็บใดๆ ที่เปิดไปยังเว็บไซต์ TeamViewer ได้

4. ยอมรับคําขอสําหรับโปรแกรมประยุกต์ TeamViewer เพื่อทําการเปลี่ยนแปลงบนอุปกรณ์ (Windows เท่านั้น)
5. แอพลิเคชัน TeamViewer เริ่มต้น และรวมถึงรหัสเซสชันการรับรองความถูกต้องการเชื่อมต่อกับอุปกรณ์ระยะไกล
6. ป๊อปอัปจะถามว่าคุณต้องการอนุญาตให้เซสชันเริ่มทํางานหรือไม่

**หมายเหตุ** รหัสเซสชันที่สร้างขึ้นโดยบริการ TeamViewer จะใช้ครั้งเดียวเท่านั้น ถ้าคุณสูญเสียการเชื่อมต่อ คุณต้อง:

1. ปิดอินสแตนซ์ของโปรแกรมประยุกต์ TeamViewer บนอุปกรณ์ระยะไกล และ บนเวิร์กสเตชันของผู้ดูแลระบบ
2. ปิดพอร์ทัลบริษัทบนอุปกรณ์ระยะไกล
3. เริ่มต้น "เซสชันความช่วยเหลือระยะไกลใหม่" จากพอร์ทัลผู้ดูแลระบบ
4. เปิดพอร์ทัลบริษัทบนอุปกรณ์ระยะไกลอีกครั้งเพื่อรับการแจ้งเตือนใหม่
5. ดาวน์โหลดและเปิดแอป TeamViewer บนอุปกรณ์ระยะไกลและเวิร์กสเตชันผู้ดูแลระบบเช่นเดียวกับก่อนหน้านี้
