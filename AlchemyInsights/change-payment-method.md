# <a name="change-payment-method"></a>เปลี่ยนวิธีการชําระเงิน

คุณสามารถเปลี่ยนวิธีการชําระเงินจาก**ศูนย์การจัดการ Microsoft 365**
  
1. ไปที่หน้า**การเรียกเก็บเงิน**\>**[สินค้าของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการเปลี่ยนวิธีการชําระเงิน คลิก**แก้ไข**ถัดจากวิธีการชําระเงิน

3. ทําตามขั้นตอนบนหน้าจอเพื่อทําการเปลี่ยนแปลงให้เสร็จสมบูรณ์

เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[การเปลี่ยนวิธีการชําระเงินของคุณสําหรับ Microsoft 365 สําหรับธุรกิจ](https://docs.microsoft.com/microsoft-365/commerce/billing-and-payments/change-payment-method)
