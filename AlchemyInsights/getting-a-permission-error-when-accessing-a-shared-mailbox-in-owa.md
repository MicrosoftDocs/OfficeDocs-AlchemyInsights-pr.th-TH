# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>ได้รับข้อผิดพลาดสิทธิ์เมื่อเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บหรือไม่

ถ้าคุณได้รับข้อผิดพลาด**บางอย่างผิดพลาด คุณไม่มีสิทธิ์ในการเปิดกล่องจดหมายนี้** เมื่อคุณพยายามที่จะเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บ บัญชีของคุณอาจไม่ได้รับอนุญาตให้เข้าถึงกล่องจดหมายที่ใช้ร่วมกัน ผู้ดูแลระบบองค์กรของคุณสามารถมอบหมายสิทธิ์ของบัญชีผู้ใช้ของคุณไปยังกล่องจดหมายที่ใช้ร่วมกันโดยทําตามคําแนะนําในหัวข้อ[Create a shared mailbox](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)
