# <a name="credit-request-due-to-a-service-outage"></a>คำขอสินเชื่อเนื่องจากกระแสตกการบริการ

เมื่อต้องการนำไปใช้กับคำขอสินเชื่อเนื่องจากการตกเป็นบริการโปรดเปิดคำขอบริการ

โปรดตรวจสอบให้แน่ใจว่าคุณมี ID เหตุการณ์ที่พบใน [หน้าสถานภาพบริการ](https://docs.microsoft.com/office365/enterprise/view-service-health) ในศูนย์การจัดการของ Microsoft ๓๖๕

โปรดทราบว่าเมื่อมีการประมวลผลการร้องขอของคุณจะได้รับการตรวจสอบเพื่อสังเกตผลกระทบของกระแสไฟ
