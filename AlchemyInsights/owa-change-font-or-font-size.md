# <a name="change-font-or-font-size"></a>เปลี่ยนแบบอักษรหรือขนาดแบบอักษร

เมื่อต้องการอ่านอีเมล คุณสามารถใช้ฟังก์ชันซูมของเบราว์เซอร์ของคุณเพื่อให้ข้อความบนหน้าจอของคุณใหญ่ขึ้น หรือเล็กลง
  
เมื่อต้องการเปลี่ยนแบบอักษรหรือขนาดแบบอักษรเมื่อเขียนข้อความ:
  
1. ที่ด้านล่างของบานหน้าต่างเรียบเรียง เลือกไอคอน**แบบอักษร**หรือ**ขนาดแบบอักษร**
    
2. เลือกแบบอักษรหรือขนาดแบบอักษรที่คุณต้องการใช้
    
ถ้าคุณไม่เห็นเมนูตัวเลือกการจัดรูปแบบที่ด้านล่างของบานหน้าต่างเรียบเรียง เลือกไอคอนที่**ตัวเลือกการจัดรูปแบบ**
  
ดูข้อมูลเพิ่มเติม[เปลี่ยนขนาดแบบอักษรใน Outlook บนเว็บ](https://support.office.com/article/43a2137f-8c3c-46df-af4a-73a12c9bb86e)
