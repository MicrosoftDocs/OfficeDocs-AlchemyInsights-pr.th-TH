# <a name="spam-filter-not-working"></a>ตัวกรองสแปมไม่ทำงาน

ถ้าคุณเชื่อว่าข้อความที่คุณได้รับใน Outlook.com คือการหลอกลวงฟิชชิ่งหรือการหลอกลวงให้เลือกข้อความแล้วเลือกฟิชชิ่ง**อีเมลขยะ** \> **Phishing**เพื่อทำเครื่องหมายว่าเป็นการหลอกลวงแบบฟิชชิ่ง เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับการใช้งานที่ไม่เหมาะสมฟิชชิ่งและการหลอกลวงให้[อ่านสิ่งนี้](https://support.office.com/article/0d882ea5-eedc-4bed-aebc-079ffa1105a3?wt.mc_id=Office_Outlook_com_Alchemy) เมื่อต้องการบล็อกที่อยู่อีเมลให้เพิ่มที่อยู่หรือโดเมนลงในรายชื่อผู้ส่งที่ปลอดภัยของคุณหรือนำตัวเลือกอีเมลขยะอื่นๆไปใช้ให้เลือก**ตัวเลือกอีเมลขยะ**
