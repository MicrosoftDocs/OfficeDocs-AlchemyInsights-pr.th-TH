# <a name="links-in-outlookcom-dont-work-in-firefox"></a>ลิงก์ใน Outlook.com ไม่ทำงานใน Firefox

ลิงก์ในข้อความอีเม Outlook.com ไม่ทำงานเมื่อพวกเขากำลังดูในเบราว์เซอร์ Firefox Mozilla กำลัง [ทำงานบนการแก้ไข](https://go.microsoft.com/fwlink/p/?linkid=2001502&amp;clcid=0x409) สำหรับ Firefox แต่จนกว่าจะ:
  
- โปรดใช้ [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) หรือเว็บเบราว์เซอร์อื่น
- ดูอีเมลในบานหน้าต่างการอ่าน สำหรับข้อมูลเพิ่มเติมให้อ่านวิธีการ[เปลี่ยนลักษณะที่ปรากฏของกล่องจดหมาย Outlook.com ของคุณ](https://support.office.com/article/b41c2ecb-f23c-42b3-b7f8-659646d5e58c?wt.mc_id=Office_Outlook_com_Alchemy)
