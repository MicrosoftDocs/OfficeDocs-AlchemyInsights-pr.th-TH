# <a name="microsoft-teams---guest-access"></a>ทีมงาน Microsoft - การเข้าถึงของแขก


ถ้าคุณจำเป็นต้องเปิดใช้งาน หรือการตั้งค่าคอนฟิกบทบาทการเข้าถึงของแขกในทีม Microsoft เรามีคำแนะนำคำแนะนำทีละขั้นตอนสำหรับคุณ:[รายการตรวจสอบการเข้าถึงของแขก](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
  
คุณอาจต้องรอถึง 24 ชั่วโมงสำหรับการเปลี่ยนแปลงการตั้งค่าคอนฟิกกลุ่มคนมีผลบังคับใช้
