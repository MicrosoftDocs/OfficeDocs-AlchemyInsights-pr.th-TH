# <a name="set-up-information-management-policies"></a>ตั้งค่านโยบายการจัดการข้อมูล

1. ถ้าไซต์ SharePoint มีไซต์ย่อย ให้เรียกดูไซต์ระดับบนสุด
    
2. คลิก**การตั้งค่า**แล้วคลิก**การตั้งค่าไซต์** ถ้าไซต์เป็นไซต์ที่เชื่อมต่อกับกลุ่ม Microsoft 365 ให้คลิก**การตั้งค่า**คลิก**เนื้อหาของไซต์**แล้วคลิก**การตั้งค่าไซต์**
    
3. ภายใต้**การดูแลไซต์คอลเลกชัน**ให้คลิก**คุณลักษณะไซต์คอลเลกชัน**
    
4. ภายใต้**การจัดการระเบียนในตําแหน่ง**ให้คลิก**เปิดใช้งาน**
    
สําหรับข้อมูลเพิ่มเติม ให้ดูที่[บทนําสู่นโยบายการจัดการข้อมูล](https://go.microsoft.com/fwlink/?linkid=404239)และ[สร้างและใช้นโยบายการจัดการข้อมูล](https://go.microsoft.com/fwlink/?linkid=2003916)
