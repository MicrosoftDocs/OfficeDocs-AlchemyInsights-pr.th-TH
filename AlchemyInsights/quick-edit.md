# <a name="quick-edit"></a>แก้ไขด่วน

การแก้ไขรายการในรายการ SharePoint มีสองวิธี คุณสามารถแก้ไขข้อมูลเดียวในมุมมองรายการ ซึ่งเป็นวิธีการเริ่มต้น คุณสามารถแก้ไขรายการต่างๆ ในมุมมองแก้ไขด่วนได้ คุณต้องมีสิทธิ์ระดับสนับสนุนหรือสูงกว่าเพื่อแก้ไขรายการ ดู[แก้ไขข้อมูลในรายการ ใน SharePoint Online](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7)
