# <a name="remove-a-hub-site"></a>เอาไซต์ฮับ

ถอนการลงทะเบียนไซต์ฮับในศูนย์กลางการดูแล SharePoint ใหม่
  
1. ในศูนย์การดูแล SharePoint ใหม่ภายใต้**ไซต์**คลิก**ไซต์ที่ใช้งานอยู่**

2. เลือกไซต์ คลิก**ฮับไซต์**และจากนั้น คลิก**Unregister เป็นฮับไซต์**

3. คลิก **OK**
