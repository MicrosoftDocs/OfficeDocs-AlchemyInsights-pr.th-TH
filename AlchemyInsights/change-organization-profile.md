# <a name="change-organization-profile"></a>เปลี่ยนโปรไฟล์องค์กร

เปลี่ยนชื่อบริษัท ที่อยู่ ข้อมูลติดต่อด้านเทคนิค และอื่นๆ ของคุณในศูนย์การจัดการในหน้า[โปรไฟล์องค์กร](https://go.microsoft.com/fwlink/p/?linkid=2067339)**การตั้งค่า** > 

ตรวจสอบให้แน่ใจว่าได้กรอกข้อมูลในฟิลด์ที่จําเป็นทั้งหมดที่มีเครื่องหมาย * เพื่อเปิดใช้งานการบันทึกการเปลี่ยนแปลงของคุณ

[ดูข้อมูลเพิ่มเติมเกี่ยวกับที่อยู่หรือการเปลี่ยนแปลงที่อยู่สําหรับเรียกเก็บเงิน](https://docs.microsoft.com/office365/admin/manage/change-address-contact-and-more)
