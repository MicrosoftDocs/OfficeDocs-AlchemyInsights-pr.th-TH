# <a name="credential-messages-in-sharepoint-or-onedrive"></a>ข้อความข้อมูลประจำตัวใน SharePoint หรือ OneDrive

<p>ผู้ใช้อาจจะเป็นระยะ ๆ กำลังรับการพร้อมท์ โดย Office SharePoint แบบออนไลน์หรือ OneDrive เมื่อพยายามเปิดเอกสาร</p> <p>ต่อไปนี้คือ ตัวอย่างของข้อมูลประจำตัวของข้อความ:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>ลงชื่อเข้าใช้ ด้วยบัญชีขององค์กรของคุณ</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>ข้อมูลประจำตัวจำเป็น</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>เราจะไม่สามารถเชื่อมต่อในขณะนี้ การตรวจสอบการเชื่อมต่อเครือข่ายของคุณ</p> <p>หากคุณพบอาการเหล่านี้ กรุณาลองขั้นตอนที่อยู่ใน<a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">บทความนี้</a></p> <p>อีกวิธีหนึ่งคือ ถ้าคุณกำลังใช้สู่ระบบแบบครั้งเดียว (SSO) เพื่อรับรองความถูกต้อง โปรดดูที่<a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">บทความนี้</a>แทน</p> <p>&nbsp;</p>
