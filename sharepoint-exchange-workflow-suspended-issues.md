# <a name="workflows-in-sharepoint"></a>เวิร์กโฟลว์ใน SharePoint

<p>ถ้าเวิร์กโฟลว์ SharePoint ไม่ได้กำลังส่งอีเมล์ องค์กรของคุณอาจจะพบขีดจำกัดผู้ส่งอัตราแลกเปลี่ยนแบบออนไลน์&nbsp;</p> <p>ข้อความแสดงข้อผิดพลาด 'ลำดับงานหยุดการทำงาน' จะเกิดขึ้นถ้าคุณมีตัวเลือกอย่างใดอย่างหนึ่งต่อไปนี้:</p> <ul> <li>คุณมีลำดับงานใน SharePoint แบบออนไลน์ที่กำลังใช้ SharePoint 2010 หรือชนิดแพลตฟอร์มเวิร์กโฟลว์ SharePoint 2013 ในส่วน</li> <li>เวิร์กโฟลว์ถูกกำหนดค่าให้ส่งข้อความอีเมลที่กำหนดเองแก่ผู้ใช้มากกว่า 200 ในแต่ละครั้ง ผู้รับมากกว่า 10000 ต่อวัน หรือข้อความมากกว่า 30 ต่อนาที</li> <li>เมื่อคุณเรียกใช้เวิร์กโฟลว์ ไม่ได้ส่งข้อความอีเมล และคุณพบข้อผิดพลาด<strong>ตั้งค่าสถานะภายในถูกเลื่อนออกไป</strong>หรือ<strong>ไม่สามารถส่งไปยังผู้รับ</strong>จะแสดงขึ้น</li> </ul> <p>สำหรับข้อมูลเพิ่มเติม โปรดดูที่<a href="https://support.office.com/en-us/article/-daily-email-limit-has-exceeded-and-your-workflow-has-been-suspended-or-unable-to-send-to-a-recipient-error-in-a-sharepoint-online-workflow-89d02169-5fa6-4259-affc-73edb6ca9fb6?ui=en-US&amp;rs=en-US&amp;ad=US">บท</a>ความต่อไปนี้</p>
