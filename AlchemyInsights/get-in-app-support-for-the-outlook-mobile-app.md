# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>รับการสนับสนุนในแอปสําหรับ Outlook สําหรับอุปกรณ์เคลื่อนที่

1. ในแอป Outlook สําหรับอุปกรณ์เคลื่อนที่ ให้เปิด**เมนู**

2. เลือกเครื่องหมายคําถามเพื่อเปิดหน้าจอ**คําติชมของวิธีใช้&amp;**

3. แตะ**ติดต่อฝ่ายสนับสนุน**และอธิบายปัญหาของคุณ

4. แตะ**ส่ง**แล้วคุณจะได้รับการแจ้งเตือนในแอปเมื่อตัวแทนฝ่ายสนับสนุนติดต่อคุณ

สําหรับคําแนะนําโดยละเอียด ให้ดูที่[รับความช่วยเหลือในแอปสําหรับ Outlook สําหรับ iOS และ Android](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support)
