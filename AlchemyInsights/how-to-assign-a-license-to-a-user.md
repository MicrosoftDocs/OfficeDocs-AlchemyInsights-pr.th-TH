# <a name="how-to-assign-a-license-to-a-user"></a><span data-ttu-id="d9628-102">วิธีการกำหนดสิทธิ์การใช้งานให้กับผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="d9628-102">How to assign a license to a user</span></span>

<span data-ttu-id="d9628-103">ผู้ใช้ต้องมีใบอนุญาตการเข้าถึงบริการ</span><span class="sxs-lookup"><span data-stu-id="d9628-103">A user needs a license to have access to services.</span></span> <span data-ttu-id="d9628-104">ต่อไปนี้เป็นขั้นตอนเพื่อเพิ่ม เปลี่ยนแปลง หรือเอาสิทธิ์การใช้งานออกจากผู้ใช้</span><span class="sxs-lookup"><span data-stu-id="d9628-104">Here are steps to add, change, or remove a license from a user.</span></span>
  
1. <span data-ttu-id="d9628-105">ในศูนย์ดูแล ไปหน้า[ผู้ใช้ที่ใช้งานอยู่](https://go.microsoft.com/fwlink/p/?linkid=834822)หรือเลือก**ผู้ใช้** \> **ผู้ใช้ที่ใช้งานอยู่**</span><span class="sxs-lookup"><span data-stu-id="d9628-105">In the Admin center, go to the [Active Users](https://go.microsoft.com/fwlink/p/?linkid=834822) page, or choose **Users** \> **Active users**.</span></span>
    
2. <span data-ttu-id="d9628-106">เลือกกล่องอยู่ถัดจากชื่อของผู้ใช้ที่คุณต้องการกำหนด เปลี่ยนแปลง หรือเอาสิทธิ์การใช้งาน</span><span class="sxs-lookup"><span data-stu-id="d9628-106">Select the box next to the name of the user to whom you want to assign, change, or remove a license.</span></span>
    
3. <span data-ttu-id="d9628-107">บนด้านขวา ในแถว**ใบอนุญาตผลิตภัณฑ์**เลือก**แก้ไข**เท่านั้น</span><span class="sxs-lookup"><span data-stu-id="d9628-107">On the right, in the **Product licenses** row, choose **Edit**.</span></span>
    
4. <span data-ttu-id="d9628-108">ในบานหน้าต่าง**ใบอนุญาตผลิตภัณฑ์**สลับสลับไปยังตำแหน่ง**บน**สำหรับสิทธิ์การใช้งานที่คุณต้องการกำหนดให้กับผู้ใช้รายนี้</span><span class="sxs-lookup"><span data-stu-id="d9628-108">In the **Product licenses** pane, switch the toggle to the **On** position for the license that you want to assign to this user.</span></span> 
    
    <span data-ttu-id="d9628-109">โดยค่าเริ่มต้น บริการทั้งหมดที่สัมพันธ์กับสิทธิ์การใช้งานนั้นจะถูกกำหนดให้กับผู้ใช้โดยอัตโนมัติ</span><span class="sxs-lookup"><span data-stu-id="d9628-109">By default, all services associated with that license are automatically assigned to the user.</span></span> <span data-ttu-id="d9628-110">เมื่อต้องการจำกัดบริการใดพร้อมใช้งานสำหรับผู้ใช้ สลับสลับเพื่อ**ปิด**ตำแหน่งสำหรับบริการที่คุณไม่ต้องการให้ผู้ใช้ดังกล่าวได้</span><span class="sxs-lookup"><span data-stu-id="d9628-110">To limit which services are available to the user, switch the toggles to the **Off** position for the services that you don't want that user to have.</span></span> 
    
    <span data-ttu-id="d9628-111">สลับการสลับไปยังตำแหน่ง**ปิด**เพื่อเอาสิทธิ์การใช้งาน</span><span class="sxs-lookup"><span data-stu-id="d9628-111">Switch the toggle to the **Off** position to remove a license.</span></span> 
    
5. <span data-ttu-id="d9628-112">ที่ด้านล่างของบานหน้าต่าง**สิทธิ์การใช้งานผลิตภัณฑ์**เลือก**บันทึก** \> **ปิด** \> **ปิด**</span><span class="sxs-lookup"><span data-stu-id="d9628-112">At the bottom of the **Product licenses** pane, choose **Save** \> **Close** \> **Close**.</span></span>
