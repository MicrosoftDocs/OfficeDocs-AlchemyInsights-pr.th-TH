# <a name="error-creating-o365-groups-the-classification-field-cant-be-empty"></a>ข้อผิดพลาดในการสร้างกลุ่ม O365: เขตข้อมูลการจัดประเภทไม่สามารถว่างเปล่าได้

เมื่อสร้างกลุ่ม O365 ใหม่ใน OWA คุณอาจได้รับข้อผิดพลาดต่อไปนี้: "เขตข้อมูลการจัดประเภทไม่สามารถว่างได้"  หรือคุณอาจได้รับข้อผิดพลาดเมื่อแก้ไขการตั้งค่ากลุ่ม O365 ที่มีอยู่ข้อผิดพลาด: "เลือกการจัดประเภทที่ถูกต้อง"   คุณอาจเห็นตัวเลือกป้ายชื่อความ **ลับ** สำหรับกลุ่ม Microsoft ๓๖๕แม้ว่าฟีเจอร์นี้ยังไม่ได้ถูกเปิดใช้งานสำหรับกลุ่ม

การแก้ไขสำหรับปัญหานี้ได้รับการระบุและอยู่ในกระบวนการของการจัดวาง  ในฐานะที่เป็นวิธีแก้ไขปัญหาชั่วคราวคุณสามารถสร้างได้โดยใช้ [ศูนย์การจัดการ Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/create-groups/create-groups?view=o365-worldwide) หรือพอร์ทัลผู้ดูแลระบบ Azure
