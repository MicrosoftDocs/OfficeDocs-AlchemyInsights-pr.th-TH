# <a name="desktop-app-for-sharepoint"></a>แอปเดสก์ท็อปสำหรับ SharePoint

องค์กรของคุณจะถูกใส่ในโปรแกรมการแสดงตัวอย่างส่วนตัวสำหรับแอปเดสก์ท็อปสำหรับ SharePoint ในตอนนี้ Microsoft ยังไม่สนับสนุนแอปเดสก์ท็อปสำหรับ SharePoint (การแสดงตัวอย่างส่วนตัว) โปรดใช้คู่ค้าแสดงตัวอย่างของ Microsoft แบบส่วนตัวเพื่อรับความช่วยเหลือเกี่ยวกับแอปเดสก์ท็อปสำหรับ SharePoint
