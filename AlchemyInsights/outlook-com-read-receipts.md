# <a name="read-receipts"></a>ตอบรับการอ่าน

คุณไม่สามารถร้องขอการตอบรับการอ่านใน Outlook.com แต่คุณสามารถร้องขอการตอบรับการอ่านสำหรับข้อความของ Outlook.com ส่งโดยใช้ Outlook สำหรับ Windows เมื่อต้องการเรียนรู้วิธีการ ดู[เพิ่มและอ่านการแจ้งเตือนการจัดส่งการรับและการร้องขอ](https://support.office.com/article/a34bf70a-4c2c-4461-b2a1-12e4a7a92141)
  
เมื่อต้องการเลือกวิธีการ Outlook.com ตอบสนองต่อการร้องขอการตอบรับการอ่าน ไปที่[อ่านการตั้งค่าใบเสร็จ](https://outlook.live.com/mail/options/mail/handling/readReceipts)
