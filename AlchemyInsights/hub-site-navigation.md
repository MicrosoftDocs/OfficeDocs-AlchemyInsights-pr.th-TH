# <a name="hub-site-navigation"></a>การนำทางไซต์ฮับ

ที่ด้านบนของฮับไซต์และไซต์ใด ๆ ที่เกี่ยวข้อง เหนือการนำทางไซต์ มีแถบการนำทางของไซต์ฮับ ถ้าคุณเป็นเจ้าของไซต์ฮับ คุณสามารถกำหนด[แถบนำทาง](https://support.office.com/en-us/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav) 

**หมายเหตุ** เมื่อต้องการเลือกไซต์การสื่อสาร หรือไซต์ทีมโดยใช้ต้นแบบสมัยใหม่เป็นฮับไซต์ของคุณมีพื้นที่เพียงพอ ถ้าคุณใช้ไซต์ทีมคลาสสิก การนำทางของฮับจะปรากฏเฉพาะบนหน้าสมัยใหม่ และการตั้งค่าไซต์ฮับจะปรากฏเฉพาะบนหน้าสมัยใหม่
