# <a name="change-the-font-size-in-a-message"></a>เปลี่ยนขนาดฟอนต์ในข้อความ

1. ถ้าคุณได้พิมพ์ข้อความแล้วให้เลือกข้อความ
2. เลือกไอคอน  **รูปแบบ** ที่ด้านล่างของข้อความ (เป็นไอคอนแรก)
3. เลือกไอคอน  **ขนาดฟอนต์**  เหนือข้อความ (เป็นไอคอนที่เจ็ด)
4. เลือกขนาด
