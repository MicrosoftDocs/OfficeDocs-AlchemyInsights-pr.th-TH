# <a name="cant-add-default-2010-approval-workflow"></a>ไม่สามารถเพิ่มเวิร์กโฟลว์การอนุมัติการอนุมัติ 2010 เริ่มต้น

ในไซต์คอลเลกชัน Microsoft SharePoint คุณไม่สามารถเพิ่มเวิร์กโฟลว์ที่นํามาใช้ใหม่ได้แบบโกลบอล (เช่น "การอนุมัติ - SharePoint 2010") ลงในรายการหรือไลบรารี เวิร์กโฟลว์การอนุมัติอาจหายไป เมื่อต้องการแก้ไขปัญหานี้ โปรดดู[ไม่สามารถเพิ่มเวิร์กโฟลว์ที่นํามาใช้ใหม่ได้แบบโกลบอลลงในรายการ SharePoint หรือไลบรารี](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library)

สําหรับข้อมูลเพิ่มเติมเกี่ยวกับการอนุมัติเวิร์กโฟลว์ ให้ดูที่[ทั้งหมดเกี่ยวกับเวิร์กโฟลว์การอนุมัติ](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742) 
 
เรียนรู้เพิ่มเติมเกี่ยวกับ[การอนุมัติสมัยใหม่ในโฟล](https://flow.microsoft.com/blog/introducing-modern-approvals)
