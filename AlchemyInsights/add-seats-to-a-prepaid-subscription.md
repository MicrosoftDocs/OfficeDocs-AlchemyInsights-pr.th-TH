# <a name="add-seats-to-a-prepaid-subscription"></a>เพิ่มที่นั่งไปยังการสมัครใช้งานแบบชำระเงินล่วงหน้า

สิทธิ์การใช้งานแบบชำระเงินล่วงหน้าจะถูกนำออกให้คุณเป็นรหัสตัวอักษรและตัวเลข25อักขระ หลังจากที่คุณซื้อสิทธิ์การใช้งานที่คุณต้องการแล้วคุณสามารถเพิ่มสิทธิ์การใช้งานลงในการสมัครใช้งานของคุณได้ 

1. ในศูนย์การจัดการให้ไปที่หน้าการ**เรียกเก็บเงิน**  >  **[ของผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการเพิ่มสิทธิ์การใช้งาน ในส่วน**สิทธิ์**การใช้งานให้เลือกเพิ่มสิทธิ์การใช้งาน**เพิ่มเติม**

3. ในบานหน้าต่างเพิ่มสิทธิ์การใช้งาน**เพิ่มเติม**ให้เลือก**ใช้คีย์ผลิตภัณฑ์ใหม่และยัง**ไม่ได้ใช้งานจากนั้นเลือก**ถัดไป**

4. ใส่คีย์ผลิตภัณฑ์จากนั้นเลือก**ถัดไป**

    > [!NOTE]
    > ถ้าคุณมีคีย์ผลิตภัณฑ์มากกว่าหนึ่งรายการคุณสามารถเลือก **เพิ่มคีย์ผลิตภัณฑ์อื่น** เพื่อใส่ลงไปได้

5. ตรวจทานรายละเอียดการสั่งซื้อของคุณจากนั้นเลือก **แลก**ใช้

เมื่อต้องการเรียนรู้เพิ่มเติมให้ดู[เพิ่มสิทธิ์การใช้งานหรือขยายการสมัครใช้งานที่ชำระเงินสำหรับการใช้คีย์ผลิตภัณฑ์](https://docs.microsoft.com/microsoft-365/commerce/licenses/add-licenses-using-product-key)
