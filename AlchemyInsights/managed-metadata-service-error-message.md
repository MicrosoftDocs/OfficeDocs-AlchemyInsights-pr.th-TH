# <a name="managed-metadata-service-error-message"></a>ข้อความข้อผิดพลาดบริการ Metadata ที่มีการจัดการ

เมื่อคุณเปิดที่เก็บคำคุณอาจเห็นข้อความแสดงข้อผิดพลาด: "บริการ Metadata ที่มีการจัดการหรือการเชื่อมต่อในขณะนี้ยังไม่พร้อมใช้งาน พูลโปรแกรมประยุกต์หรือบริการเว็บ Metadata ที่มีการจัดการอาจยังไม่ได้เริ่มต้นใช้งาน โปรดติดต่อผู้ดูแลระบบของคุณ "
  
ถ้ามีการส่งผ่าน24ชั่วโมงหลังจากที่คุณเห็นปัญหานี้ก่อนแล้วโปรดเข้าสู่ระบบกรณีสนับสนุน เราเข้าใจว่า24ชั่วโมงอาจดูเหมือนเป็นเวลานาน ในหลายกรณีเรากำลังดำเนินการแก้ไขปัญหาดังนั้นโปรดให้เราอย่างน้อย24ชั่วโมง
