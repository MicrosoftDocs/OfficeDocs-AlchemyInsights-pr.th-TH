# <a name="questions-about-your-microsoft-365-license"></a>คําถามเกี่ยวกับใบอนุญาตใช้งาน Microsoft 365 ของคุณ

ถ้าคุณต้องการทราบจํานวนสิทธิ์การใช้งานที่คุณซื้อ หรือจํานวนสิทธิ์การใช้งานที่มอบหมายจากการสมัครใช้งาน ให้ทําดังนี้
  
1. ไปที่หน้า[บริการ&ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)**การเรียกเก็บเงิน**\>

2. ค้นหาการสมัครใช้งานที่คุณต้องการทราบข้อมูลเพิ่มเติม คุณสามารถดูจํานวนใบอนุญาตที่คุณจ่ายและจํานวนที่พร้อมให้ได้รับมอบหมาย

ถ้าคุณต้องการเพิ่มสิทธิ์การใช้งานเพิ่มเติม หรือถ้าคุณมีสิทธิ์การใช้งานมากเกินไป**ให้เลือก**
