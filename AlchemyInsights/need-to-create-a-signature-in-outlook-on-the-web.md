# <a name="need-to-create-an-email-signature"></a>ต้องการสร้างลายเซ็นอีเมลใช่หรือไม่

ถ้าคุณมีบัญชี Outlook หรือ Outlook บนเว็บ (ที่ทํางานหรือที่โรงเรียน) โปรดทราบว่าลายเซ็นอีเมลของคุณจะแตกต่างกันใน Outlook และ Outlook บนเว็บ เมื่อต้องการสร้างลายเซ็นอีเมลของคุณ ให้ดูที่[สร้างและเพิ่มลายเซ็นลงในข้อความใน Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)หรือ[สร้างและเพิ่มลายเซ็นอีเมลใน Outlook บนเว็บ](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)
