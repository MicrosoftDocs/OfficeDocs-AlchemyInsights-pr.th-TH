# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>ฉันไม่เห็นปฏิทินที่ฉันคาดหวังเมื่อเลือกความพร้อม

ถ้าคุณไม่เห็นปฏิทินที่คุณคาดไว้ ให้ไปที่ ปฏิทิน แตะปุ่มเมนูที่มุมบนซ้ายของหน้าจอ และตรวจดูให้แน่ใจว่าปฏิทินที่คุณคาดว่าจะเห็นถูกเลือก เฉพาะปฏิทินที่เปิดใช้งานในเมนูนี้เท่านั้นที่จะปรากฏเมื่อเลือกเวลาสําหรับ "ส่งความพร้อมใช้งาน"
