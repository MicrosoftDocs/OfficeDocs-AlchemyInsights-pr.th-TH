# <a name="no-option-to-install-office-visio-or-project"></a>ไม่มีตัวเลือกในการติดตั้ง Office, Visio หรือโครงการ

เมื่อต้องการติดตั้งแอปไคลเอ็นต์ Office โปรดตรวจสอบให้แน่ใจว่าคุณมีการสมัครใช้งาน Office 365 หรือ Microsoft 365 ที่มี[แอปไคลเอ็นต์ Office](https://support.office.com/article/office-for-home-and-office-for-business-plans-28cbc8cf-1332-4f04-9123-9b660abb629e)เช่น Microsoft 365 Business Premium, Microsoft 365 Apps สําหรับธุรกิจ หรือแอป Microsoft 365 สําหรับธุรกิจ

**หมายเหตุ**: [Office Home และธุรกิจ](https://products.office.com/home-and-business)เป็น[Office สําหรับผลิตภัณฑ์ที่บ้าน](https://support.office.com/article/28cbc8cf-1332-4f04-9123-9b660abb629e?wt.mc_id=Alchemy_ClientDIA)และไม่ได้เป็นส่วนหนึ่งของการสมัครใช้งานธุรกิจ สิ่งจําเป็นสําหรับโครงการออนไลน์ไม่รวมรุ่นเดสก์ท็อปของโครงการจึงไม่จําเป็นต้องติดตั้ง ผู้ใช้ Visio Plan 1 สามารถเข้าถึง Visio บนเว็บได้ เนื่องจากแผนดังกล่าวไม่มีแอป Visio บนเดสก์ท็อป แอพเดสก์ท็อปโครงการและ Visio ไม่พร้อมใช้งานบน Mac

ถ้าคุณมีการสมัครใช้งานที่มีแอป Microsoft 365 คุณจะไม่เห็นตัวเลือกในการติดตั้ง เว้นแต่ว่าคุณมี[สิทธิ์การใช้งานที่กําหนด](https://support.office.com/article/what-office-365-business-product-or-license-do-i-have-f8ab5e25-bf3f-4a47-b264-174b1ee925fd?wt.mc_id=scl_installoffice_home) ถ้าคุณเป็นผู้ดูแลระบบ Office 365 ที่รับผิดชอบในการกําหนดสิทธิ์การใช้งาน ให้ดูที่[มอบหมายสิทธิ์การใช้งานให้กับผู้ใช้](https://support.office.com/article/assign-licenses-to-users-in-office-365-for-business-997596b5-4173-4627-b915-36abac6786dc?wt.mc_id=scl_installoffice_home)


มีผู้ใช้แต่ละราย:

1. ไปที่[หน้าพอร์ทัล](https://portal.office.com/OLS/MySoftware.aspx)

2. ลงชื่อเข้าใช้ด้วยบัญชีที่ทํางานหรือที่โรงเรียนซึ่งมีการกําหนดสิทธิ์การใช้งาน Office ไว้

3. ภายใต้ส่วน Office เลือกภาษา เลือกรุ่น 32 บิตหรือ 64 บิต

4. คลิก**ติดตั้ง**

สําหรับขั้นตอนโดยละเอียดและการแก้ไขปัญหาเมื่อติดตั้ง Office ให้ดูที่[ดาวน์โหลดและติดตั้งหรือติดตั้ง Office บนพีซีหรือ Mac](https://support.office.com/article/4414eaaf-0478-48be-9c42-23adc4716658?wt.mc_id=Alchemy_ClientDIA) สําหรับคําแนะนําเฉพาะในการติดตั้ง Visio หรือ Project โปรดดูที่[การติดตั้ง Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710)หรือ[ติดตั้งโครงการ](https://support.office.com/article/7059249b-d9fe-4d61-ab96-5c5bf435f281)
