# <a name="activate-visio"></a>เปิดใช้งาน Visio

Admin จำเป็นเพื่อมอบหมาย[สิทธิ์การใช้งาน Visio](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800)ไปยังผู้ใช้ก่อน Visio เดสก์ท็อปแอพลิเคชันเดสก์ท็อปที่มาพร้อมกับ 2 แผนออนไลน์ของ Visio คุณสามารถติดตั้ง และเปิดใช้งาน (Visio ออนไลน์แผน 1 ไม่มี Visio รุ่นเดสก์ท็อป)

1. [ติดตั้ง Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1800)ถ้าไม่ได้ติดตั้งไว้แล้ว
2. เมื่อติดตั้งเสร็จสมบูรณ์ เปิดในโปรแกรมประยุกต์ ในหลายกรณี โปรแกรมประยุกต์จะเปิดใช้งานโดยอัตโนมัติ และผู้ใช้สามารถเริ่มต้นการใช้ Visio ทันที
    ถ้าได้รับพร้อมท์ให้เข้าสู่ระบบ ผู้ใช้ต้องลงชื่อเข้าใช้ ด้วยการทำงานเดียวกันหรือบัญชีโรงเรียนซึ่งมีการกำหนดสิทธิ์การใช้งาน Visio
