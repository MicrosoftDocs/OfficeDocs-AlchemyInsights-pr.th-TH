# <a name="address-book-in-outlookcom"></a>สมุดรายชื่อใน Outlook.com

สมุดรายชื่อของ Outlook.com เรียกว่าบุคคล

เมื่อต้องการไปยังเพจของคุณบุคคล เลือก**บุคคล**ที่มุมด้านล่าง ของหน้า หรือ จากตัวเลือกโปรแกรมประยุกต์ที่มุมบนของหน้า

เรียนรู้วิธีการ[สร้าง มุม มอง และแก้ไขผู้ติดต่อใน Outlook.com](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01)
