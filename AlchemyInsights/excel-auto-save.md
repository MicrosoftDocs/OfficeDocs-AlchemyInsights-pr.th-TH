# <a name="what-is-autosave"></a>'การบันทึกอัตโนมัติ' คืออะไร

การบันทึกอัตโนมัติเป็นคุณลักษณะที่มีอยู่ใน Excel, Word และ PowerPoint สําหรับสมาชิก Microsoft 365 ที่บันทึกแฟ้มของคุณโดยอัตโนมัติทุกสองสามวินาทีในขณะที่คุณทํางาน 

สําหรับคําถามที่ถามบ่อย ให้ดูที่['การบันทึกอัตโนมัติ' คืออะไร](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
