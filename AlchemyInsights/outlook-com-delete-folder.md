# <a name="delete-a-folder-in-outlookcom"></a>ลบโฟลเดอร์ใน Outlook.com

1. ในบานหน้าต่างด้านซ้าย คลิกขวาโฟลเดอร์หรือโฟลเดอร์ย่อยที่คุณต้องการลบออก และ**ลบโฟลเดอร์**ที่เลือก
2. เลือก**ตกลง**เพื่อยืนยัน

เมื่อต้องการเรียนรู้เพิ่มเติม ดู[การทำงานกับโฟลเดอร์ข้อความใน Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)
