# <a name="activating-a-new-device-running-windows-10"></a>การเปิดใช้งานอุปกรณ์ใหม่ที่รัน Windows 10

หากคุณซื้ออุปกรณ์ใหม่ที่ใช้ Windows 10 ระบบจะเปิดใช้งานโดยอัตโนมัติเมื่อคุณเชื่อมต่ออุปกรณ์ของคุณกับอินเทอร์เน็ต ถ้าเซิร์ฟเวอร์การเปิดใช้งานไม่ว่างคุณอาจต้องรอสักครู่เพื่อให้ Windows เปิดใช้งาน

เพื่อตรวจสอบสถานะการเปิดใช้งานให้เลือกการ**เริ่ม****การตั้งค่าการ** > **ปรับปรุง &** > การ**เปิดใช้งาน**ความปลอดภัย

**หมายเหตุ:** ในบางกรณีคุณอาจต้องป้อนหมายเลขผลิตภัณฑ์ของ Windows 10 ที่มาพร้อมกับบรรจุภัณฑ์ของอุปกรณ์หรือรวมเป็นบัตรหรือบนใบรับรองความเป็นผลิตภัณฑ์ของแท้ (COA) ที่แนบมากับอุปกรณ์

[เรียนรู้เพิ่มเติมเกี่ยวกับการเปิดใช้งานใน Windows 10](https://support.microsoft.com/help/12440)
