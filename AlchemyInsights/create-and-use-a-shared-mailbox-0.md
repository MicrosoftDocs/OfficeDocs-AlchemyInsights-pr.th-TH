# <a name="create-and-use-a-shared-mailbox"></a>สร้าง และใช้กล่องจดหมายที่ใช้ร่วมกัน

กล่องจดหมายที่ใช้ร่วมกันช่วยให้ส่ง และรับอีเมลสำหรับแฝง เช่น**sales@contoso.com**โดยไม่มอบหมายสิทธิ์การใช้งานอื่นมากกว่าหนึ่งคน
  
[สร้างกล่องจดหมายที่ใช้ร่วมกัน](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9)ที่อธิบายถึงวิธีการ: 
  
- สร้างกล่องจดหมายที่ใช้ร่วมกัน
    
- ให้สิทธิ์ในการใช้กล่องจดหมายที่ใช้ร่วมกันของผู้ใช้ของคุณ
    
- การเข้าถึงกล่องจดหมายที่ใช้ร่วมกัน ใน Outlook หรือ Outlook บนเว็บ
