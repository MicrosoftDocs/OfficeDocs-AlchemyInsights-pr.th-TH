# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>การติด ตั้ง ติดตั้ง และซิงค์แฟ้ม OneDrive หรือ SharePoint บน Mac 

การติดตั้ง ติดตั้งและซิงค์ OneDrive หรือ SharePoint แฟ้มบน Mac ยังมีคำแนะนำจาก[ซิงค์แฟ้มกับไคลเอนต์ซิงค์ OneDrive บน Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f)

ถ้ามีการติดตั้งโปรแกรมประยุกต์ OneDrive และการตั้งค่าบน Mac เป็น แต่ผู้ใช้จะประสบปัญหาการซิงค์ ดู[ซิงค์แก้ไขปัญหา ด้วย OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756?ui=en-US&amp;rs=en-US&amp;ad=US)

> [!NOTE]
> คุณลักษณะเรียกว่าการย้ายโฟลเดอร์ซิงค์ OneDrive สำหรับ Windows จะไม่พร้อมใช้งานสำหรับ Mac.
