# <a name="cant-delete-files-or-photos-from-files"></a>ไม่สามารถลบแฟ้มหรือรูปภาพจากแฟ้ม

แฟ้มแสดงแฟ้มและรูปภาพที่แนบไปกับข้อความในกล่องจดหมายของคุณ คุณไม่สามารถลบแฟ้มหรือรูปถ่ายโดยตรงจากแฟ้ม เมื่อต้องการลบแฟ้มและรูปภาพ คุณจำเป็นต้องลบข้อความอีเมลที่แนบไป:

สำหรับข้อมูลเพิ่มเติม อ่าน[วิธีการลบแฟ้มหรือรูปถ่ายใน Outlook.com](https://support.office.com/article/bae0531f-040f-4c42-90b9-786ca718c16d.aspx)
