นโยบายเซสชันการรักษาความปลอดภัยของแอปบนระบบคลาวด์ช่วยให้คุณสามารถจํากัดเซสชันตามสถานะอุปกรณ์ได้ เมื่อต้องการควบคุมเซสชันโดยใช้อุปกรณ์เป็นเงื่อนไข ให้สร้างทั้งนโยบายการเข้าถึงแบบมีเงื่อนไขและนโยบายเซสชัน
