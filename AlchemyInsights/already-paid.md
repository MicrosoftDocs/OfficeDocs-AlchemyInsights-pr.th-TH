# <a name="already-paid"></a>ชำระเงินแล้วใช่หรือไม่

ถ้าคุณได้ส่งการชำระเงินแล้วคุณสามารถแจ้งให้เราทราบ:
  
1. ในศูนย์การจัดการให้ไปที่หน้าการสมัครใช้งานการ**เรียกเก็บเงิน** \> [Subscriptions](https://go.microsoft.com/fwlink/p/?linkid=842054)

2. เลือกการสมัครใช้งานและในส่วนใบ**เรียกเก็บเงินของฉัน**ให้เลือก**ดูใบเรียกเก็บเงิน**

3. บนหน้า **รายละเอียดใบเรียกเก็บ** เงินให้เลือก **ชำระเงินแล้วใช่หรือไม่** คุณจะถูกขอให้ใส่ข้อมูลที่เกี่ยวข้องกับการชำระเงินที่ส่ง

4. เมื่อใส่ข้อมูลที่จำเป็นทั้งหมดแล้วให้เลือก**ส่ง**
