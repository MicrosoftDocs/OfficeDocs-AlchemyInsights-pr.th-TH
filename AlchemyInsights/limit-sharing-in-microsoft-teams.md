# <a name="limit-sharing-in-microsoft-teams"></a>จำกัดการแชร์ในทีม Microsoft

เมื่อต้องการทำความเข้าใจเกี่ยวกับวิธีการจำกัดการแชร์ในทีมและ Microsoft ๓๖๕โดยทั่วไปรีวิวการ[แชร์ขีดจำกัดใน Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-limit-sharing?view=o365-worldwide)

เมื่อต้องการทำความเข้าใจเกี่ยวกับวิธีการควบคุมการเข้าถึงผ่านทางนโยบายการเข้าถึงตามเงื่อนไขให้ดูที่:

- [การควบคุมการเข้าถึงจากอุปกรณ์ที่ไม่มีการจัดการ](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
- [บทช่วยสอน: การดาวน์โหลดข้อมูลที่เป็นความลับของบล็อก](https://docs.microsoft.com/cloud-app-security/use-case-proxy-block-session-aad)
- [ตั้งค่านโยบายการเข้าถึงตามเงื่อนไข](https://docs.microsoft.com/microsoft-365/business/set-up-conditional-access-policies?view=o365-worldwide)
