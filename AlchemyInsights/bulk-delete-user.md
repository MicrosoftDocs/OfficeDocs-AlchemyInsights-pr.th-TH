# <a name="bulk-delete-user"></a>ผู้ใช้ลบจํานวนมาก

คุณสามารถลบผู้ใช้จํานวนมากออกจากกลุ่มโดยใช้ไฟล์ค่าที่คั่นด้วยเครื่องหมายจุลภาค (CSV) เพื่อลบผู้ใช้จํานวนมาก เมื่อต้องการลบผู้ใช้จํานวนมากโดยใช้พอร์ทัล Azure AD ดู[ผู้ใช้จํานวนมากลบในไดเรกทอรีที่ใช้งานอยู่ของ Azure](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete)
