# <a name="create-a-new-folder-in-outlookcom"></a>สร้างโฟลเดอร์ใหม่ใน Outlook.com

1. ภายใต้**โฟลเดอร์**ในบานหน้าต่างนำทางให้เลือก**โฟลเดอร์ใหม่**

2. พิมพ์ชื่อสำหรับโฟลเดอร์และกด Enter

หากต้องการเรียนรู้เพิ่มเติมโปรดดู[สร้างโฟลเดอร์ใน Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy)
