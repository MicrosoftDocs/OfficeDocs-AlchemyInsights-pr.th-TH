# <a name="get-the-a1-plan"></a>รับแผน A1

ถ้าสถาบันของคุณมีคุณสมบัติสําหรับ Microsoft 365 Education อยู่แล้ว คุณสามารถเพิ่มการสมัครใช้งาน A1 จาก**ศูนย์การจัดการ Microsoft 365** 

1. ไปที่**ผลิตภัณฑ์>การเรียกเก็บเงิน[& บริการ](https://go.microsoft.com/fwlink/p/?linkid=868433)**

2. ใช้กล่องค้นหาเพื่อค้นหาใน "A1"

3. เลือกจํานวนผู้ใช้ที่จะใช้แผน A1

ถ้าสถาบันของคุณยังไม่ได้ตั้งค่า Microsoft 365 Education แต่คุณกําลังหาเพิ่ม Office 365 ฟรี คุณสามารถ[เริ่มต้นใช้งาน](https://www.microsoft.com/education/products/office)ได้โดยให้อีเมลของคุณ
