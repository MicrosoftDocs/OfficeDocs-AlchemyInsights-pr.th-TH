# <a name="need-to-create-an-email-signature"></a>ต้องการสร้างลายเซ็นอีเมลหรือไม่

ถ้าคุณมีบัญชีผู้ใช้ Outlook หรือบัญชีผู้ใช้ outlook บนเว็บ (ที่ทำงานหรือที่โรงเรียน) คุณสามารถสร้างลายเซ็นและเพิ่มลงในข้อความขาออกทั้งหมดโดยอัตโนมัติหรือเลือกเพิ่มลงในข้อความขาออกที่เฉพาะเจาะจงได้ โปรดสังเกตว่าลายเซ็นอีเมลของคุณจะแตกต่างกันใน Outlook และ Outlook บนเว็บ เมื่อต้องการสร้างลายเซ็นอีเมลของคุณให้ดูที่[สร้างและเพิ่มลายเซ็นลงในข้อความใน outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx)หรือ[สร้างและเพิ่มลายเซ็นอีเมลใน outlook บนเว็บ](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx)
