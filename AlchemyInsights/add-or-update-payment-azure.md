# <a name="update-billing-info-under-azure-enterprise-agreement"></a>อัปเดตข้อมูลการเรียกเก็บเงินภายใต้ข้อตกลงขององค์กร Azure

การเปลี่ยนแปลงข้อมูลการชำระเงินของคุณสำหรับการสมัครใช้งาน Azure enterprise ไม่ได้รับการสนับสนุน
