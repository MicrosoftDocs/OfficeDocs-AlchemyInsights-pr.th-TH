# <a name="view-your-users-microsoft-365-apps-activity"></a>ดูกิจกรรมแอป Microsoft ๓๖๕ของผู้ใช้ของคุณ

[รายงานการเปิดใช้งาน Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide)จะให้มุมมองของผู้ใช้ที่ติดตั้งและเปิดใช้งานแอป Microsoft ๓๖๕ของพวกเขาบนอุปกรณ์อย่างน้อยหนึ่งเครื่อง

นอกจากนี้คุณยังสามารถรับมุมมองในกิจกรรมแอป Microsoft ๓๖๕ของผู้ใช้ของคุณโดยใช้[รายงานการใช้งานแอป microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)
