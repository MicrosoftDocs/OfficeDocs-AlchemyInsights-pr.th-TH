# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>โปรไฟล์ของฉันจะเปลี่ยนแปลงการซิงค์กับโปรแกรมประยุกต์ส่วนกำหนดค่าผู้ใช้ของ SharePoint เมื่อใด

SharePoint แบบออนไลน์ใช้งานตัวจับเวลาการนำเข้าไดเรกทอรีที่ใช้งานอยู่ (การนำเข้าโฆษณา) เพื่อนำเข้าผู้ใช้และกลุ่มไปยังโปรแกรมประยุกต์ส่วนกำหนดค่าผู้ใช้ 
  
1. นำเข้าโฆษณาซิงค์การเปลี่ยนแปลงจากที่เก็บไดเรกทอรีแบบออนไลน์ของ SharePoint ไปยังโปรแกรมประยุกต์ส่วนกำหนดค่าผู้ใช้ การเปลี่ยนแปลงเหล่านี้จะถูกประมวลผลเป็นชุดงาน
    
2. งานตัวจับเวลาจะรันจนกว่าจะมีการซิงค์การเปลี่ยนแปลง
    
> [!NOTE]
> เวลาที่ใช้งานในการรันจะขึ้นอยู่กับจำนวนของการเปลี่ยนแปลงที่จะประมวลผล จำนวนมากของการเปลี่ยนแปลงจะใช้เวลานานกว่า ข้อตกลงระดับการให้บริการ (SLA) ระบุว่าการเปลี่ยนแปลงผู้ใช้ในไดเรกทอรีแบบออนไลน์ของ SharePoint จะมีผลในแอพลิเคชันส่วนกำหนดค่าผู้ใช้ใน24ชั่วโมง
