# <a name="restore-a-user"></a>คืนค่าผู้ใช้

คืนค่าผู้ใช้โดยใช้ขั้นตอนเหล่านี้:
  
1. ไปที่[ \> ผู้ใช้ที่ถูกลบ](https://admin.microsoft.com/adminportal/home#/deletedusers)

2. เลือกผู้ใช้ แล้วเลือก**กู้คืน**

3. ทําตามพร้อมท์เพื่อตั้งรหัสผ่าน

4. คลิก**ส่งอีเมล และปิด**และคุณเสร็จสิ้นแล้ว!

ไม่ง่ายเหรอ? สําหรับรายละเอียดเพิ่มเติมและขั้นตอนกับภาพหน้าจอให้ตรวจสอบบทความนี้:[คืนค่าผู้ใช้](https://docs.microsoft.com/microsoft-365/admin/add-users/restore-user) ถ้าคุณตระหนักว่า คุณจําเป็นต้องคืนค่ากล่องจดหมายของผู้ใช้ ให้ดูที่[ลบ หรือคืนค่ากล่องจดหมายของผู้ใช้ใน Exchange แบบออนไลน์](https://docs.microsoft.com/exchange/recipients-in-exchange-online/delete-or-restore-mailboxes)
