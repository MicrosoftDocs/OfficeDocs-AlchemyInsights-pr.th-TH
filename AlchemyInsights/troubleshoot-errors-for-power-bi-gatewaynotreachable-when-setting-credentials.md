# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable เมื่อการตั้งค่าข้อมูลประจำตัว

คุณอาจพบ GatewayNotReachable เมื่อพยายามตั้งค่าข้อมูลประจำตัวสำหรับแหล่งข้อมูลภายในบริการพลังงาน BI นี่อาจเป็นผลของเกตเวย์ล้าสมัย เกตเวย์ล่าสุดติดตั้ง แล้วลองอีกครั้ง
