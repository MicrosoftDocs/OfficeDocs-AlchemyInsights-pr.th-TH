# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>ได้รับข้อผิดพลาดสิทธิ์เมื่อเปิดกล่องจดหมายที่แชร์ใน Outlook บนเว็บหรือไม่

ถ้าคุณได้รับข้อผิดพลาด **บางอย่างไม่ถูกต้อง คุณไม่มีสิทธิ์ในการเปิดกล่องจดหมายนี้** เมื่อคุณพยายามเปิดกล่องจดหมายที่แชร์ใน Outlook บนเว็บบัญชีผู้ใช้ของคุณอาจไม่มีสิทธิ์ในการเข้าถึงกล่องจดหมายที่แชร์ ผู้ดูแลระบบองค์กรของคุณสามารถกำหนดสิทธิ์ของบัญชีผู้ใช้ของคุณให้กับกล่องจดหมายที่แชร์ได้โดยทำตามคำแนะนำในหัวข้อนี้เพื่อ[สร้างกล่องจดหมายที่แชร์](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)
