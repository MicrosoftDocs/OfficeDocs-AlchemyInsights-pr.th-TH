# <a name="web-only-limited-access"></a>Web-Only การเข้าถึงที่จำกัด

Web-Only การเข้าถึงแบบจำกัดคือตัวแปรของระดับสิทธิ์ ' การเข้าถึงที่จำกัด ' ซึ่งจะช่วยให้ผู้ใช้สามารถเข้าถึงเว็บวัตถุได้เท่านั้น

สำหรับข้อมูลเพิ่มเติมให้ดูที่[ระดับสิทธิ์เริ่มต้น](https://docs.microsoft.com/sharepoint/understanding-permission-levels#default-permission-levels)
