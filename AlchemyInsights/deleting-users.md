# <a name="deleting-users"></a>การลบผู้ใช้

ถ้าคุณต้องการบล็อกการเข้าถึงของผู้ใช้ในทันที และจําเป็นต้องเก็บข้อมูลของพวกเขา โปรดดูที่ บล็อกการเข้าถึงข้อมูล[Microsoft 365 ของพนักงานเก่า](https://docs.microsoft.com/office365/admin/add-users/remove-former-employee#block-a-former-employees-access-to-office-365-data)
  
หากคุณต้องการลบผู้ใช้และไม่ได้เกี่ยวข้องกับข้อมูลใดๆ ของพวกเขา โปรดดูที่[ลบผู้ใช้ออกจากองค์กรของคุณ](https://docs.microsoft.com/office365/admin/add-users/delete-a-user) ทันทีนี้จะเพิ่มใบอนุญาตของพวกเขา
