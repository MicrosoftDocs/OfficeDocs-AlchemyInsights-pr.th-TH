# <a name="change-domain-name-in-sharepoint-online"></a>เปลี่ยนชื่อโดเมนใน SharePoint Online

ขออภัยคุณไม่สามารถเปลี่ยนชื่อโดเมน SharePoint Online สำหรับองค์กรของคุณได้ ตัวอย่างเช่นถ้าชื่อขององค์กรของคุณเปลี่ยนจาก "Contoso" เป็น "กาแฟที่สี่" คุณจะไม่สามารถเปลี่ยน contoso.sharepoint.com เป็น fourthcoffee.sharepoint.com ได้
  
เมื่อต้องการใช้ชื่อโดเมน fourthcoffee.sharepoint.com คุณจำเป็นต้องซื้อการสมัครใช้งานใหม่และย้ายอีเมลทั้งหมดไฟล์และข้อมูลอื่นๆที่คุณต้องการเก็บไว้ในการสมัครใช้งานใหม่
  
 **หมายเหตุ**: ตัวอย่างเช่นเมื่อต้องการเปลี่ยนชื่อของไซต์จากให้ https://contoso.sharepoint.com/sites/sample1 https://contoso.sharepoint.com/sites/sample2 ทำตามคำแนะนำใน[บทความนี้](https://docs.microsoft.com/sharepoint/change-site-address) 
  
สำหรับข้อมูลเกี่ยวกับการย้ายรายการในไลบรารีเอกสารให้ดูที่[ย้ายหรือคัดลอกรายการจากไลบรารีเอกสาร SharePoint](https://go.microsoft.com/fwlink/?linkid=2025831)
