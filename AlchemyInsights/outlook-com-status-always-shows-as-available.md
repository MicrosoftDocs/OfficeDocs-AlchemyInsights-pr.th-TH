# <a name="my-status-always-shows-as-available"></a>สถานะของฉันแสดงเป็นพร้อมใช้งานเสมอ

แต่ละครั้งที่คุณเข้าสู่ระบบ Outlook.com สถานะบัญชีของคุณจะถูกตั้งค่าให้พร้อมใช้งาน คุณสามารถป้องกันไม่ให้สถานะของคุณเปลี่ยนแปลงโดยไม่มีการเซ็นชื่อจาก Outlook.com การเข้าสู่ระบบในเมื่อปิดเบราว์เซอร์ของคุณ เลือกกล่อง**ให้ฉันลงชื่อเข้าใช้**บนหน้าการลงชื่อเข้าใช้ใน Outlook.com
