# <a name="solutions-for-unsupported-operating-system"></a>วิธีแก้ไขปัญหาสำหรับ "ระบบปฏิบัติการที่ไม่สนับสนุน"



เมื่อคุณพยายามติดตั้ง Office 365 คุณอาจเห็นข้อผิดพลาดของ**ระบบปฏิบัติการที่ไม่สนับสนุน**อุปกรณ์ของคุณไม่เป็นไปตาม[ข้อกำหนดของระบบ](https://products.office.com/office-system-requirements)
  
ต้องใช้ office 365:
  
- พีซี: Windows 10, Windows 8.1, Windows 8, 7 Service Pack Windows 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 หรือ Windows Server 2008 R2
    
    OR
    
- ระบบปฏิบัติการ Mac X 10.10
    
ถ้าคุณได้รับข้อความนี้ขณะพยายามติดตั้ง Office บนอุปกรณ์เคลื่อนที่หรือเป็น Chromebook ดู[วิธีการติดตั้ง และเรียกใช้ Microsoft Office บน Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)
