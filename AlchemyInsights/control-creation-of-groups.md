# <a name="control-creation-of-microsoft-365-groups"></a>ตัวควบคุมการสร้างกลุ่ม Microsoft ๓๖๕

ตามค่าเริ่มต้นผู้ใช้ทั้งหมดภายในบริษัทจะสามารถสร้างกลุ่มได้ ถ้าจำเป็นคุณสามารถเปลี่ยนการตั้งค่านี้และกำหนดสิทธิ์ในการสร้างกลุ่มเป็นชุดของบุคคลภายในบริษัทของคุณได้ โปรดทราบว่าสิ่งนี้จะมีผลกระทบต่อการสร้างกลุ่มจากบริการ Microsoft ๓๖๕ใดๆ (เช่น Outlook, ทีม, การวางแผนและอื่นๆ)
  
คำแนะนำ: [จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
