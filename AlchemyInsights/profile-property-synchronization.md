# <a name="profile-property-synchronization"></a>การซิงโครไนซ์คุณสมบัติโปรไฟล์

คุณไม่สามารถแมปคุณสมบัติเพิ่มเติมสำหรับการนำเข้าโฆษณาเพื่อซิงค์จาก Azure Active Directory ไปยังแอปพลิเคชันโปรไฟล์ผู้ใช้ได้ การนำเข้าโฆษณาถูกจำกัดไว้ที่ชุดคุณสมบัติที่กำหนดไว้ล่วงหน้าเพื่อรับประกันประสิทธิภาพการทำงานของตัวจับเวลาที่สอดคล้องกัน
  
[เรียนรู้เพิ่มเติมเกี่ยวกับคุณสมบัติของโปรไฟล์ที่ซิงค์](https://go.microsoft.com/fwlink/?linkid=875671)
