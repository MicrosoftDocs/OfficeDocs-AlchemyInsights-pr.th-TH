# <a name="use-your-browser-to-check-spelling"></a>ใช้เบราว์เซอร์ของคุณเพื่อตรวจสอบการสะกดคำ

ไม่มีตัวเลือกการตรวจสอบการสะกดใน Outlook บนเว็บ ในการตรวจสอบการสะกดของคุณคุณจะต้องใช้เว็บเบราว์เซอร์ของคุณ การตรวจสอบการสะกดสามารถใช้ได้ใน Microsoft Edge, Internet Explorer 10 และรุ่นที่ใหม่กว่าและ Firefox, Chrome และ Safari รุ่นปัจจุบัน ตรวจสอบตัวเลือกสำหรับเว็บเบราว์เซอร์ของคุณเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีตรวจสอบการสะกดคำ
