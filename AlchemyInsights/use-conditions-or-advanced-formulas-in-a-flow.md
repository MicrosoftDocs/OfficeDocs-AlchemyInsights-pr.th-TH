# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>ใช้เงื่อนไขหรือสูตรขั้นสูงในโฟลว์

ใน Microsoft Flow คุณสามารถกําหนดโฟลว์เองเพื่อให้ทํางานเฉพาะเมื่อเงื่อนไขเป็นจริง สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การเพิ่มเงื่อนไขลงในโฟลว์](https://go.microsoft.com/fwlink/?linkid=872112)
  
เมื่อต้องการสร้างสูตรขั้นสูง ให้ดูที่[เค้าร่างภาษาของข้อกําหนดของเวิร์กโฟลว์สําหรับแอปตรรกะ Azure](https://aka.ms/logicexpressions)
