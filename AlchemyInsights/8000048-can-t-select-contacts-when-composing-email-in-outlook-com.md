# <a name="cant-select-contacts-when-composing-email"></a>ไม่สามารถเลือกผู้ติดต่อเมื่อเขียนอีเมล

เมื่อสร้างการอีเมลใหม่ และคลิกปุ่ม**ถึง**หรือ**สำเนาถึง**ลูกค้าบางรายอาจไม่เห็นรายชื่อผู้ติดต่อของพวกเขาได้ เมื่อต้องการเลือกชื่อ เริ่มพิมพ์ใน**การ**หรือเขตข้อมูล **'สำเนาถึง'** และผู้ติดต่อที่เกี่ยวข้องจะแสดงขึ้น
