# <a name="how-to-reactivate-a-subscription"></a>วิธีการเปิดใช้งานการสมัครใช้งาน

ถ้ามีการสมัครใช้งานของคุณได้หมดอายุแล้ว ยกเลิก หรือปิดใช้งาน โดย Microsoft คุณสามารถใน Microsoft 365 admin ศูนย์ในภายหลัง
  
1. ในศูนย์ดูแล ไป**การเรียกเก็บเงิน**\>หน้า[บริการ & ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)

2. ภายใต้**การเรียกเก็บเงิน**อยู่ติดกับ**Renews บน**หรือ**Expiring บน**เลือก**แก้ไข**

3. ในหน้าต่าง**ตั้งค่าการเรียกเก็บเงินเกิดซ้ำ**เลือกตัวเลือกที่คุณต้องการ หากเรียกเก็บเงินที่ที่เกิดซ้ำแล้วปิด เลือก**รายเดือน**หรือ**หนึ่งครั้ง**เพื่อเปิดใช้งาน

4. เลือก**ส่งการเปลี่ยนแปลง**

สำหรับข้อมูลเพิ่มเติม ดู:[ต่ออายุ Office 365 สำหรับธุรกิจ](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/renew-your-subscription)
