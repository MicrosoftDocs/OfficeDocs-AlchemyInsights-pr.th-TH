# <a name="microsoft-stream-licensing-overview"></a>ภาพรวมของการให้สิทธิ์การใช้งาน Microsoft Stream

Microsoft Stream คือบริการ cloud ที่ได้รับสิทธิ์การใช้งานสำหรับการสมัครใช้งานต่อผู้ใช้แต่ละราย พร้อมใช้งานเป็นส่วนหนึ่งของการสมัครใช้งาน Microsoft ๓๖๕ที่มีสิทธิ์ ให้ดูที่การกำหนดราคา Microsoft ๓๖๕สำหรับข้อมูลเกี่ยวกับชนิดการสมัครใช้งานแต่ละชนิด
