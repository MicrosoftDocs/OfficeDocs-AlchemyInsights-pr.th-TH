# <a name="search-and-hybrid"></a>การค้นหาและการไฮบริด

- คุณต้องการข้อมูลเกี่ยวกับการค้นหาแบบไฮบริดและที่ติดต่อกับภายนอกใน SharePoint Online หรือไม่ ถ้าเป็นเช่นนั้นให้ดู[บทความนี้](https://docs.microsoft.com/sharepoint/hybrid/hybrid-search-in-sharepoint)

- คุณต้องการข้อมูลเกี่ยวกับการวางแผนการค้นหาแบบไฮบริดที่ติดต่อกับภายนอกหรือไม่  ถ้าเป็นเช่นนั้นให้ดู[บทความนี้](https://docs.microsoft.com/sharepoint/hybrid/plan-hybrid-federated-search)
