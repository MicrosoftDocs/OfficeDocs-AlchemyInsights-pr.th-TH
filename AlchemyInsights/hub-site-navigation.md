# <a name="sharepoint-hub-site-navigation"></a>การนำทางไซต์ฮับ SharePoint

ที่ด้านบนของไซต์ฮับและไซต์ใดๆที่เกี่ยวข้องเหนือการนำทางไซต์เป็นแถบนำทางของฮับไซต์ ถ้าคุณเป็นเจ้าของไซต์ฮับคุณสามารถกำหนด[แถบนำทาง](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav)เองได้ 

**หมายเหตุ:** ขอแนะนำให้เลือกไซต์การสื่อสารหรือไซต์ทีมโดยใช้แม่แบบที่ทันสมัยเป็นฮับไซต์ของคุณ ถ้าคุณใช้ไซต์ทีมคลาสสิกการนำทางฮับจะปรากฏเฉพาะบนเพจที่ทันสมัยและการตั้งค่าไซต์ฮับจะปรากฏบนเพจที่ทันสมัยเท่านั้น
