# <a name="types-of-calendar-sharing"></a>ชนิดของปฏิทินที่ใช้ร่วมกัน

ได้แก่ชนิดของปฏิทินที่ใช้ร่วมกันที่มีอยู่และวิธีการตั้งค่าคอนฟิกเหล่านี้:
  
- [สร้างกล่องจดหมายที่ใช้ร่วมกัน](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9.aspx)
    
- [กล่องจดหมายของห้องและอุปกรณ์](https://support.office.com/article/9f518a6d-1e2c-4d44-93f3-e19013a1552b.aspx)
    
- [สร้างปฏิทินกลุ่ม](https://support.office.com/article/8385667b-d758-4489-a53f-f542dd01e6ff.aspx)
    
- [ปฏิทิน Outlook ที่มีการใช้ร่วมกันกับผู้อื่น](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88.aspx)
    
หากคุณไม่สามารถดูข้อมูลว่าง/ไม่ว่างปฏิทินในปฏิทินอื่น หรือบุคคลอื่นไม่สามารถดูข้อมูลว่าง/ไม่ว่างในปฏิทินของคุณ ใช้การ[สนับสนุนของ Microsoft และช่วยกู้คืนสำหรับ Office 365](https://diagnostics.office.com/)
