# <a name="update-dns-records"></a>อัปเดตระเบียน DNS

- [เพิ่มหรือแก้ไขเรกคอร์ด DNS ที่กำหนดเอง](https://docs.microsoft.com/office365/admin/dns/add-or-edit-custom-dns-records)
