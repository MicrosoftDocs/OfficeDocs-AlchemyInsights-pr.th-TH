# <a name="how-to-install-office-on-an-iphone"></a>วิธีการติดตั้ง Office บน iPhone ผิด


- มีผู้ใช้[เคาะเชื่อมโยงดาวน์โหลด](https://support.office.com/article/9df6d10c-7281-4671-8666-6ca8e339b628?wt.mc_id=Alchemy_ClientDIA)จากอุปกรณ์เคลื่อนที่ของตน สำหรับ Word, Excel, PowerPoint และ OneNote 
    
- ผู้ใช้เริ่มต้นโปรแกรมประยุกต์ เคาะ**เข้าสู่ระบบ**และป้อนการทำงานของตน หรือโรงเรียนข้อมูลประจำตัวได้
