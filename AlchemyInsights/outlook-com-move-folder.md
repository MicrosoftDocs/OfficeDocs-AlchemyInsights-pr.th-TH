# <a name="move-a-folder-in-outlookcom"></a>ย้ายโฟลเดอร์ในOutlook.com

คุณสามารถเรียงลําดับโฟลเดอร์ใหม่ภายใต้**รายการโปรด**แต่คุณไม่สามารถเรียงลําดับโฟลเดอร์ใหม่ภายใต้**โฟลเดอร์**หรือสร้างโฟลเดอร์ย่อยให้โฟลเดอร์ ถ้าคุณย้ายโฟลเดอร์ไปยังโฟลเดอร์อื่นโดยไม่ได้ตั้งใจ ให้สร้างโฟลเดอร์ใหม่และคัดลอกอีเมลทั้งหมดของคุณจากโฟลเดอร์ย่อยไปยังโฟลเดอร์ใหม่
  
เมื่อต้องการเรียนรู้เพิ่มเติม ให้อ่าน[การทํางานกับโฟลเดอร์ข้อความใน Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)
