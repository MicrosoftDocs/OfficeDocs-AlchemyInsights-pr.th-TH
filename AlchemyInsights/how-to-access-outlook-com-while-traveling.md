# <a name="how-to-access-outlookcom-while-traveling"></a>วิธีการเข้าถึง Outlook.com ระหว่างการเดินทาง

เข้าถึง Outlook.com ระหว่างการเดินทาง ทำให้แน่ใจว่า มีข้อมูลล่าสุดของ Microsoft บัญชีข้อมูลความปลอดภัย -การสำรองอีเมลที่อยู่หรือหมายเลขโทรศัพท์ที่คุณสามารถรับรหัสความปลอดภัย- เมื่อต้องการเรียนรู้เพิ่มเติม ดู[วิธีการเข้าถึง Outlook.com เมื่อเดินทาง](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e)
