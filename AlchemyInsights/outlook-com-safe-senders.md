# <a name="stop-messages-from-going-into-your-junk-email-folder"></a>หยุดข้อความไม่ให้เข้าไปในโฟลเดอร์เมลขยะของคุณ

มีบางครั้งที่คุณไม่ต้องการให้ข้อความจากบุคคลหรือโดเมนที่ระบุสิ้นสุดลงในโฟลเดอร์เมลขยะของคุณใน Outlook.com ข้อความจากที่อยู่หรือโดเมนในรายชื่อผู้ส่งที่ปลอดภัยจะไม่ถูกย้ายไปยังโฟลเดอร์เมลขยะของคุณ

1. เปิด[การตั้งค่าผู้ส่งที่ปลอดภัย](https://go.microsoft.com/fwlink/?linkid=2035804)
2. ในส่วนผู้**ส่งและโดเมนที่ปลอดภัย**ให้ป้อนที่อยู่หรือโดเมนที่คุณต้องการเพิ่มแล้วเลือก**เพิ่ม**
3. หากต้องการเพิ่มรายชื่อเมลให้กับผู้ส่งที่ปลอดภัยให้ป้อนรายชื่อจดหมายใต้**รายการส่งเมล์ที่ปลอดภัย**และเลือก**เพิ่ม**
4. เลือก**บันทึก**

อ่านเพิ่มเติมได้ที่[บล็อกหรือเลิกบล็อกผู้ส่งใน Outlook.com](https://support.office.com/article/afba1c94-77bb-4f50-8b85-057cf52f4d5e?wt.mc_id=Office_Outlook_com_Alchemy)
