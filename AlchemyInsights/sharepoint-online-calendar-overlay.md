# <a name="sharepoint-online-calendar-overlay"></a>การวางซ้อนปฏิทินของ SharePoint Online

คุณสามารถวางซ้อนปฏิทิน SharePoint อื่นได้ตราบใดที่ปฏิทิน SharePoint ทั้งสองอยู่ในไซต์คอลเลกชันเดียวกัน ความสามารถในการใช้การวางซ้อนปฏิทินใน SharePoint Online หรือ SharePoint Server ปฏิทินที่ถูกดึงมาจาก Exchange Online ไม่ได้รับการสนับสนุน

SharePoint Online มีประสบการณ์การใช้งานที่ทันสมัยด้วย Microsoft ๓๖๕กลุ่มปฏิทิน เมื่อต้องการเรียนรู้เพิ่มเติมให้ดู[ที่ใช้ web part สำหรับปฏิทินกลุ่ม](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce)
