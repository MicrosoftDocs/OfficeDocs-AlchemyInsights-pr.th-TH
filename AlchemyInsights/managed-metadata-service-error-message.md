# <a name="managed-metadata-service-error-message"></a>ข้อผิดพลาดบริการ Metadata ที่มีการจัดการ

เมื่อคุณเปิดเก็บคำ คุณอาจเห็นข้อผิดพลาด: "บริการ Metadata ที่มีการจัดการหรือการเชื่อมต่อในขณะนี้ใช้งานไม่ได้ พูลโปรแกรมประยุกต์หรือบริการเว็บข้อมูลเมตาที่ที่มีจัดการอาจไม่ได้ถูกเริ่มต้น กรุณาติดต่อผู้ดูแลระบบของคุณ"
  
ถ้ามากกว่า 24 ชั่วโมงได้ตั้งแต่คุณเห็นปัญหานี้ขั้นแรก โปรดเข้าสู่ระบบสนับสนุนกรณีและปัญหา เราเข้าใจว่า 24 ชั่วโมงอาจเหมือนเป็นเวลานาน ในหลายกรณี เรากำลังทำงานอยู่บนโซลูชันดังนั้นโปรดให้กับเราอย่างน้อย 24 ชั่วโมง
