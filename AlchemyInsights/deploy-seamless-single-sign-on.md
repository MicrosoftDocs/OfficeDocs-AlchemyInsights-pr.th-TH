# <a name="enable-seamless-sso"></a>เปิดใช้งาน SSO ที่ไม่มีรอยต่อ

Azure ที่ใช้งานอยู่ไดเรกทอรี (Azure AD) การเข้าสู่ระบบแบบครั้งเดียวอย่างราบรื่น (SSO ไม่มีรอยต่อ) โดยอัตโนมัติในผู้ใช้เมื่อพวกเขาอยู่บนเดสก์ท็อปของบริษัทที่เชื่อมต่อกับเครือข่ายขององค์กรของคุณ
  
[ทำตามขั้นตอนในบทความนี้เพื่อเปิดใช้งาน SSO แบบไม่มีรอยต่อ](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
