# <a name="refresh-using-web-connector-doesnt-work-properly"></a>การรีเฟรชโดยใช้ตัวเชื่อมต่อเว็บทำงานไม่ถูกต้อง

ถ้าคุณมีสคริปต์ตัวเชื่อมต่อเว็บที่ใช้ฟังก์ชัน[เว็บเพจ](https://msdn.microsoft.com/library/mt260924.aspx)และคุณได้ปรับปรุงชุดข้อมูลหรือรายงานของคุณหลังจากวันที่18พฤศจิกายน๒๐๑๖คุณจำเป็นต้องใช้เกตเวย์เพื่อให้รีเฟรชทำงานได้อย่างถูกต้อง
