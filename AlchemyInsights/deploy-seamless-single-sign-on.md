# <a name="enable-seamless-sso"></a>เปิดใช้งาน SSO อย่างราบรื่น

Azure Active Directory (Azure AD) การลงชื่อเข้าระบบครั้งเดียวอย่างราบรื่น (SSO ที่ราบรื่น) จะลงชื่อเข้าใช้โดยอัตโนมัติเมื่อพวกเขาอยู่บนเดสก์ท็อปขององค์กรที่เชื่อมต่อกับเครือข่ายขององค์กรของคุณ
  
[ทำตามขั้นตอนในบทความนี้เพื่อเปิดใช้งาน SSO อย่างราบรื่น](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
