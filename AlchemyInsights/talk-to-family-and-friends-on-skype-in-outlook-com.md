# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>พูดคุยกับครอบครัวและเพื่อนบน Skype ใน Outlook.com

ลูกค้า Outlook.com สามารถพูดคุยกับเพื่อนและครอบครัวด้วยคุณลักษณะการแชท, เสียง, และวิดีโอใน Skype. เมื่อคุณ[ผสานบัญชี Microsoft และ Skype ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409)คุณจะมีสิทธิ์เข้าถึง Skype บนเว็บใน Outlook.com โดยอัตโนมัติ
  
เรียนรู้เพิ่มเติมเกี่ยวกับวิธีการ[พูดคุยกับครอบครัวและเพื่อนบน Skype ใน Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409)
