# <a name="connect-classic-sharepoint-team-sites-to-new-microsoft-365-groups"></a>เชื่อมต่อไซต์ทีม SharePoint แบบคลาสสิกกับกลุ่ม Microsoft ๓๖๕ใหม่

เมื่อต้องการประเมินว่าไซต์คอลเลกชัน SharePoint ของคุณพร้อมที่จะเชื่อมต่อกับกลุ่ม Microsoft ๓๖๕หรือไม่ให้ใช้[เครื่องมือสแกนเนอร์ของ sharepoint](https://go.microsoft.com/fwlink/?linkid=873066) เครื่องมือนี้สามารถช่วยคุณวางแผนกลยุทธ์ของเว็บไซต์ของคุณและจะสร้างรายงานในเชิงลึกรวมถึงพื้นที่ที่ต้องการความสนใจ
  
ถ้าคุณต้องการอนุญาตให้ผู้ดูแลไซต์คอลเลกชันเชื่อมต่อไซต์ทีมแบบคลาสสิกไปยังกลุ่มใหม่จากคำสั่งเมนูใหม่ที่เรากำลังเพิ่มอยู่คุณไม่จำเป็นต้องทำอะไรเลย อย่างไรก็ตามถ้าคุณต้องการปิดใช้งานตัวเลือกนี้คุณจะต้องเปลี่ยนการตั้งค่าเริ่มต้น [เรียนรู้วิธี](https://go.microsoft.com/fwlink/?linkid=2004316)การ โปรดสังเกตว่าผู้ดูแลระบบส่วนกลางและผู้ดูแล SharePoint จะสามารถเชื่อมต่อไซต์กับกลุ่มใหม่ได้เสมอโดยใช้ cmdlet PowerShell หรือ API
