# <a name="copy-and-paste-in-outlookcom"></a>คัดลอกและวางในOutlook.com

ไม่มีปุ่มคัดลอกและวางในOutlook.com แต่คุณสามารถใช้แป้นพิมพ์ลัดได้

**บนพีซี:** CTRL+C เพื่อคัดลอก CTRL+X เพื่อตัด และ CTRL+V เพื่อวาง

**บน Mac:** CMD+ C เพื่อคัดลอก, CMD + X เพื่อตัด, และ CMD + V เพื่อวาง
