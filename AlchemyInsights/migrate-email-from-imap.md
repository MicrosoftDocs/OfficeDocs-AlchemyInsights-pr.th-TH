# <a name="when-you-get-a-not-an-accepted-domain-error"></a>เมื่อคุณได้รับข้อผิดพลาด "ไม่ใช่โดเมนที่ยอมรับ"

เมื่อพยายามเพิ่มที่อยู่อีเมลบนผู้ใช้หรือกลุ่ม หรือระหว่างการย้ายกล่องจดหมาย Microsoft 365 คุณอาจได้รับข้อผิดพลาดต่อไปนี้:**คุณไม่สามารถใช้โดเมนได้เนื่องจากโดเมนดังกล่าวยังไม่ได้รับการยอมรับสําหรับ Exchange System**
  
ข้อผิดพลาดนี้เป็นผลมาจากชื่อโดเมน *(เช่น contoso.com)* ไม่ได้ถูกติดตั้งไว้อย่างสมบูรณ์สําหรับองค์กรของคุณ
  
โปรดตรวจสอบว่า โดเมนมีการตั้งค่าทั้งหมด โดยทําตามคําแนะนําใน[เพิ่มโดเมนไปยัง Microsoft 365](https://docs.microsoft.com/office365/admin/setup/add-domain)
