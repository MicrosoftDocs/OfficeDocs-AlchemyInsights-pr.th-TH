# <a name="how-to-reactivate-a-subscription"></a>วิธีการเปิดใช้งานการสมัครใช้งาน

ถ้ามีการสมัครใช้งานของคุณได้หมดอายุแล้ว ยกเลิก หรือปิดใช้งาน โดย Microsoft คุณสามารถใน Microsoft 365 admin ศูนย์ในภายหลัง
  
1. ในศูนย์ดูแล ไป**การเรียกเก็บเงิน**\>หน้า[บริการ & ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)

2. ค้นหาการบอกรับเป็นสมาชิกที่คุณต้องการเปิดใช้งานใหม่ ในส่วนของ**การเรียกเก็บเงิน**เลือก**เปิดใช้งานใหม่**

    ถ้าคุณไม่เห็น**เปิดใช้อีกครั้ง**เป็นการดำเนินการที่พร้อมใช้งาน โปรดติดต่อฝ่ายสนับสนุนการเปิดใช้งานการสมัครใช้งาน

3. ยืนยันรายละเอียดการชำระเงินของคุณ คุณสามารถปรับปรุงข้อมูลของคุณที่มีอยู่การชำระเงินที่นี่ เมื่อคุณเสร็จสิ้นการป้อนวิธีการชำระเงิน เลือก**เปิดใช้งานใหม่**

คุณกำลังใช้ศูนย์ดูแลแบบเก่าหรือไม่ สำหรับขั้นตอนเดิม หรือเมื่อต้อง การเรียนรู้เพิ่มเติม ดู[เปิดใช้งานการสมัครใช้งาน](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/reactivate-your-subscription)
