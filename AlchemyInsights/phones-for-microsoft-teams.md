# <a name="are-you-looking-for-phones-that-are-compatible-with-teams"></a>คุณกำลังมองหาโทรศัพท์ที่เข้ากันได้กับทีมหรือไม่

ในฐานะผู้ดูแลระบบคุณจะ [จัดการอุปกรณ์ทั้งหมด](https://docs.microsoft.com/microsoftteams/device-management) ที่ใช้กับทีมในองค์กรของคุณจากศูนย์การจัดการ Microsoft ๓๖๕ สำหรับข้อมูลเพิ่มเติม: 

- ตรวจทานรายชื่อของ[โทรศัพท์สำหรับทีม Microsoft](https://docs.microsoft.com/microsoftteams/phones-for-teams) 
- ตรวจทานรายการโทรศัพท์ IP ที่ได้รับการ[รับรองในปัจจุบัน](https://docs.microsoft.com/microsoftteams/teams-ip-phones#currently-certified-ip-phones) 
- ตรวจสอบรายการของ[อุปกรณ์ที่มีข้อจำกัดและปัญหาที่ทราบแล้ว](https://support.office.com/article/control-calls-using-a-headset-in-teams-65d6e104-444d-4013-b8c2-f11317dd69a8)
