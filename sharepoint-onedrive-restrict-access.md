# <a name="restrict-access-in-sharepoint-or-onedrive"></a>จำกัดการเข้าถึงใน SharePoint หรือ OneDrive

<p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">มีหลายวิธีในการจำกัดการเข้าถึงบริการ ออนไลน์/OneDrive ของ SharePoint วิธีเข้าถึงข้อจำกัดต่าง ๆ จะถูกทำเค้าร่างด้านล่าง</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">การจำกัดสิทธิ์</span></u></strong></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">ใน SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ เราจำกัดการเข้าถึงรายการอย่างเช่นไซต์ แฟ้ม และโฟลเดอร์ โดยเฉพาะ สิทธิการเข้าถึงกลุ่ม/ผู้ใช้เหล่านั้นควรมีการเข้าถึง</span></p> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"><a href="https://support.office.com/en-us/article/Customize-permissions-for-a-SharePoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782">กำหนดสิทธิ์สำหรับรายการ SharePoint หรือไลบรารี</a></span></li> </ul> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"><a href="https://docs.microsoft.com/en-us/sharepoint/customize-sharepoint-site-permissions">กำหนดสิทธิ์ของไซต์ SharePoint</a></span></li> </ul> <ul> <li><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"><a href="https://support.office.com/en-us/article/Change-the-permissions-on-a-subfolder-5427BD7C-F20A-4F75-8CF2-5359DD45A1A6">เปลี่ยนแปลงสิทธิ์บนโฟลเดอร์ย่อย</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">ควบคุมการเข้าถึงจากอุปกรณ์ไม่มีการจัดการ</span></u></strong></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">เป็นผู้ดูแลส่วนกลางใน Office 365 หรือ SharePoint คุณสามารถ<a href="https://docs.microsoft.com/en-us/sharepoint/control-access-from-unmanaged-devices">บล็อกหรือจำกัดการเข้าถึงเนื้อหา SharePoint และ OneDrive จากอุปกรณ์ไม่มีการจัดการ</a>(เหล่าไฮบริโฆษณาที่นำมารวมกัน หรือเข้ากันได้ใน Intune) ได้</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">ข้อจำกัดของตำแหน่งที่ตั้งเครือข่าย</span></u></strong></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">เป็นผู้ดูแลระบบ IT ผิด คุณสามารถควบคุมการเข้าถึงทรัพยากร SharePoint และ OneDrive โดยยึดตามตำแหน่งที่ตั้งเครือข่ายที่กำหนดที่คุณเชื่อถือ นี่คือหรือที่เรียกอีกอย่างหนึ่งว่านโยบายที่อ้างอิงตำแหน่งที่ตั้ง สำหรับข้อมูลเพิ่มเติม โปรดดูที่<a href="https://docs.microsoft.com/en-us/sharepoint/control-access-based-on-network-location">ควบคุมการเข้าถึงแบบออนไลน์ของ SharePoint และข้อมูล OneDrive โดยยึดตามตำแหน่งที่ตั้งเครือข่าย</a></span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">จำกัดผู้ใช้จากการสร้างไซต์หรือไซต์ย่อย</span></u></strong></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">เป็นผู้ดูแลระบบ SharePoint หรือผู้ดูแลส่วนกลางของ Office 365 คุณสามารถให้ผู้ใช้ของคุณสร้าง และการดูแลไซต์ SharePoint ของตนเอง กำหนดชนิดของไซต์ที่พวกเขาสามารถสร้าง และระบุที่ตั้งของไซต์ได้ สำหรับข้อมูลเพิ่มเติม โปรดดู<a href="https://docs.microsoft.com/en-us/sharepoint/manage-site-creation">การสร้างไซต์การจัดการใน SharePoint แบบออนไลน์</a></span></p>
