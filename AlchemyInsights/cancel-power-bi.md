# <a name="cancel-power-bi"></a>ยกเลิก Power BI

Power BI สามารถยกเลิกได้ในศูนย์การจัดการ Microsoft ๓๖๕

1. ในศูนย์การจัดการให้ไปที่หน้าการ**เรียกเก็บเงิน >[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**
2. บนแท็บ **ผลิตภัณฑ์** ให้ค้นหาการสมัครใช้งานที่คุณต้องการยกเลิก เลือก **การกระทำเพิ่มเติม** (จุดสามจุด) แล้วเลือก **ยกเลิกการสมัคร**ใช้งาน
3. ในบานหน้าต่าง **ยกเลิกการสมัคร** ใช้งานให้เลือกเหตุผลที่คุณกำลังยกเลิก อีกทางหนึ่งคือให้คำติชม
4. เลือก**บันทึก**

**การขอคืนเงิน: เครดิตตามสัดส่วนจะถูกส่งกลับไปยังคุณภายในรอบการเรียกเก็บเงินครั้งถัดไป**

> [!NOTE]
> ถ้าคุณไม่เห็นตัวเลือกในการยกเลิกให้เลือกการสมัครใช้งานเพื่อดูหน้ารายละเอียด ในส่วนการ **ตั้งค่าการสมัครใช้งานและการชำระเงิน** ให้เลือก **ยกเลิกการสมัคร**ใช้งาน

เมื่อต้องการเรียนรู้เพิ่มเติมให้ดู[ยกเลิกการสมัครใช้งานของคุณ](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)
