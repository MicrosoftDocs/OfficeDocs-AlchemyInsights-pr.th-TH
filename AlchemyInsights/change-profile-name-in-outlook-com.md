# <a name="change-your-profile-name"></a>เปลี่ยนชื่อโปรไฟล์ของคุณ

1. ไปที่[ข้อมูลบัญชี Microsoft ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=860841)
    
2. เลือก**แก้ไขชื่อ** 
    
3. ป้อนชื่อและนามสกุล แล้วเลือก**บันทึก** 
    
โปรดรอได้ถึง 24 ชั่วโมงเพื่อให้การเปลี่ยนแปลงชื่อโปรไฟล์ของคุณมีผล
  
หากต้องการเรียนรู้เพิ่มเติม โปรดดูที่[เปลี่ยนรูปหรือชื่อโปรไฟล์ของคุณใน Outlook.com](https://go.microsoft.com/fwlink/?linkid=873110)
