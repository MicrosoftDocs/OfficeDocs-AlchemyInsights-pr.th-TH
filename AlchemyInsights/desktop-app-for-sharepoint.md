# <a name="desktop-app-for-sharepoint"></a>แอปเดสก์ท็อปสำหรับ SharePoint

องค์กรของคุณจะถูกป้อนในโปรแกรมแสดงตัวอย่างส่วนตัวสำหรับแอปเดสก์ท็อปสำหรับ SharePoint ขณะนี้ Microsoft ไม่สนับสนุนโปรแกรมประยุกต์สำหรับเดสก์ท็อปสำหรับ SharePoint (ตัวอย่างส่วนตัว) กรุณาใช้คู่ค้าของคุณในการแสดงตัวอย่างแบบส่วนตัวของ Microsoft เพื่อขอความช่วยเหลือเกี่ยวกับแอปเดสก์ท็อปสำหรับ SharePoint
