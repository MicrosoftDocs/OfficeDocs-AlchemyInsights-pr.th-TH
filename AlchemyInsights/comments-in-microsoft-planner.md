# <a name="comments-in-microsoft-planner"></a>ความคิดเห็นในไมโครซอฟท์วางแผน

ข้อคิดเห็นสําหรับงานภายในแผนจะถูกเก็บไว้ในกล่องจดหมาย Exchange แบบออนไลน์สําหรับกลุ่ม Microsoft 365 ที่เชื่อมโยงกับแผน  เมื่อคุณโพสต์ความคิดเห็นในงาน การแจ้งเตือนทางอีเมลจะถูกส่งไปยังกล่องจดหมายของกลุ่ม และคุณจะได้รับอีเมลสําหรับความคิดเห็นที่ตามมาแต่ละรายการที่เกิดขึ้นในงานนั้น

ต่อไปนี้เป็นคําตอบสําหรับปัญหาทั่วไปที่เกี่ยวข้องกับความคิดเห็น

- **ผู้ใช้ไม่ได้รับอีเมล**- ความคิดเห็นจะถูกส่งไปยังกล่องจดหมายของกลุ่มสําหรับกลุ่มที่แผนเป็นของ สําหรับผู้ใช้ที่ได้รับอีเมลกลุ่ม กลุ่มควรได้รับการกําหนดค่าให้ส่งการสนทนากลุ่มไปยังกล่องจดหมายของสมาชิก

- **ข้อคิดเห็นจะไม่ได้รับการบันทึก**- ผู้ใช้เพิ่มข้อคิดเห็นไม่มีสิทธิ์ในการส่งอีเมลไปยังกลุ่ม Microsoft 365 อ่าน[วิธีการทํางานของ Microsoft วางแผน](https://techcommunity.microsoft.com/t5/planner-blog/how-microsoft-planner-works/ba-p/1214736)สําหรับข้อมูลเพิ่มเติมเกี่ยวกับสถานการณ์จําลองนี้

- ข้อผิดพลาด**คุณไม่มีการเข้าถึง**จะแสดงขึ้น หรือ**ผู้ใช้ที่เป็นแขกไม่สามารถเพิ่มข้อคิดเห็น**- ผู้ใช้ที่เป็นแขกที่ไม่สามารถส่งอีเมลไปยังกล่องขาเข้าของกลุ่มอาจเห็นข้อความนี้ เมื่อต้องการแก้ไข ให้ตรวจสอบว่าผู้ใช้ที่เป็นแขกมีที่อยู่อีเมลที่ถูกต้อง

- **ผู้ใช้ถูกลบออกจะได้รับอีเมล**- หากความคิดเห็นของผู้ใช้ในงานก่อนที่จะถูกเอาออกจากแผน, ด้ายอีเมลรวมถึงผู้ใช้สําหรับความคิดเห็นที่ทําในงานแต่ละ

สําหรับข้อมูลรายละเอียดเกี่ยวกับข้อคิดเห็นที่มี Microsoft Planner ดูวิธีการทํางานของ[Microsoft Planner](https://techcommunity.microsoft.com/t5/planner-blog/how-microsoft-planner-works/ba-p/1214736)และ[ข้อคิดเห็นเกี่ยวกับงานใน Microsoft Planner](https://support.microsoft.com/office/fd4aedde-7785-4cd0-96ee-122fbc9140e1)
