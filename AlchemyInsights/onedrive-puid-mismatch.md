# <a name="onedrive-puid-mismatch"></a>วันไดรฟ์ PUID ไม่ตรงกัน
ปรากฏว่าคุณกําลังประสบกับ[PUID ไม่ตรงกัน](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business#when-accessing-a-onedrive-site)ใน OneDrive สําหรับธุรกิจ โปรดทําตามขั้นตอนด้านล่างเพื่อให้เราสามารถช่วยคุณโดยเร็วที่สุด:

1. คลิก**ติดต่อฝ่ายสนับสนุน**แล้วคลิก**คําขอบริการใหม่**
2. ปล่อยชื่อเรื่องและคําอธิบายเป็น**PUID ไม่ตรงกันบนไซต์ OneDrive**
3. เมื่อตั๋วถูกส่งไปแล้ว โปรดอัปเดตด้วยข้อมูลต่อไปนี้:
    - URL ของไซต์ OneDrive ที่ได้รับผลกระทบ
    - ผู้ใช้หลักชื่อ (UPN) ของผู้ใช้ที่ได้รับผลกระทบ
