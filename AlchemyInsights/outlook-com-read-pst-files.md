# <a name="pst-data-files-in-outlookcom"></a>แฟ้มข้อมูล PST ใน Outlook.com

ไม่สามารถนำเข้า Outlook แฟ้มข้อมูล PST บัญชี Outlook.com ของคุณโดยใช้เบราว์เซอร์

การนำเข้าอยู่ แฟ้มข้อมูล PST คุณต้องใช้แอพลิเคชันเดสก์ท็อป Outlook เมื่อต้องการเรียนรู้เพิ่มเติม อ่าน[นำเข้าอีเมล ผู้ติดต่อ และปฏิทินจาก Outlook .pst](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
