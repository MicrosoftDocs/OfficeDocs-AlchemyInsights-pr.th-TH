# <a name="add-remove-or-prevent-users-from-changing-profile-photos"></a>เพิ่ม ลบ หรือป้องกันไม่ให้ผู้ใช้เปลี่ยนรูปโปรไฟล์

- **การเพิ่มรูปโปรไฟล์:** รูปโปรไฟล์สามารถเพิ่มโดยผู้ดูแลระบบใน[ศูนย์การจัดการ Microsoft 365 ผู้ใช้ที่ใช้งานอยู่](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users)หรือ[การจัดการผู้ใช้ไดเรกทอรีที่ใช้งานอยู่ของ Azure](https://portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/AllUsers)  หากคุณไม่เห็นตัวเลือก "เปลี่ยนรูปภาพ" โปรดตรวจสอบว่าได้กําหนดสิทธิ์การใช้งานให้กับผู้ใช้นั้น รูปภาพสามารถเพิ่มหรือเปลี่ยนแปลงได้โดยผู้ใช้ในโปรไฟล์ผู้ใช้จากบริการ Microsoft 365 ใด ๆ โดยคลิกที่ชื่อย่อ / รูปภาพที่มุมขวาบนของหน้าจอ สําหรับข้อมูลเพิ่มเติมเกี่ยวกับการเพิ่มรูปถ่ายโปรไฟล์ ให้ดูที่[การเพิ่มรูปโปรไฟล์ของคุณลงใน Microsoft 365](https://support.office.com/article/add-your-profile-photo-to-office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7)

- **การลบรูปโปรไฟล์:** รูปโปรไฟล์สามารถถูกเอาออก โดยผู้ดูแลระบบในการจัดการ[ผู้ใช้ไดเรกทอรีที่ใช้งานอยู่ของ Azure](https://portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/AllUsers)หรือ โดยผู้ใช้ในโพรไฟล์ผู้ใช้ของ Microsoft Teams

- **การบล็อกการเปลี่ยนแปลงรูปโปรไฟล์:** การเปลี่ยนแปลงรูปถ่ายสามารถถูกบล็อกสําหรับ Microsoft 365* ทั้งหมดโดยการเพิ่มนโยบาย Outlook Web App ต่อบทความ[การล็อกรูปถ่ายหรือการจํากัดสิทธิ์ในการเปลี่ยนรูปถ่ายโปรไฟล์ Microsoft 365](https://answers.microsoft.com/msoffice/forum/msoffice_o365admin-mso_manage/locking-photos-or-restricting-permissions-to/1d19ae4f-de5d-4c3d-a0ad-4b8b8ac32e3d)

*โปรดทราบว่า Microsoft Teams ไม่สนับสนุนนโยบาย Outlook Web App เพื่อบล็อกการเปลี่ยนแปลงของรูปถ่าย แต่กําลังวางแผนที่จะเพิ่มการสนับสนุนสําหรับคุณลักษณะนี้ในช่วงต้นปี 2020
