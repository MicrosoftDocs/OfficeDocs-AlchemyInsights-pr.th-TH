# <a name="47500-access-denied-please-try-again-later"></a>ปฏิเสธการเข้าถึง 4.7.500 โปรดลองอีกครั้งในภายหลัง

ข้อผิดพลาดนี้เกิดขึ้นเมื่อปริมาณของปริมาณการใช้งานอีเมลจากอยู่ IP ของแหล่งข้อมูลเกินขีดจำกัดตามชื่อเสียง (หรือไม่มีชื่อเสียง) ของที่อยู่ IP ของแหล่งที่มา

บล็อกอีเมลจากอยู่ IP ของแหล่งที่มาจะหมดอายุภายในหนึ่งชั่วโมง ถ้าอยู่ IP ของแหล่งที่มาเป็นเซิร์ฟเวอร์อีเมลในสถานที่ที่เป็นของคุณ ตรวจสอบการกำหนดค่าของตัวเชื่อมต่อการไหลของจดหมาย ถ้าลักษณะการทำงานมีความยาวมากกว่า หนึ่งชั่วโมง ติดต่อฝ่ายสนับสนุนเพื่อขอข้อยกเว้นสำหรับอยู่ IP ของแหล่งที่มา
