# <a name="project-license-management"></a>การจัดการลิขสิทธิ์โครงการ

การจัดการสิทธิ์การใช้งานโครงการสามารถเสร็จสมบูรณ์ใน**ศูนย์การจัดการ Microsoft 365**

- [เปรียบเทียบตัวเลือกโครงการ](https://products.office.com/project/compare-microsoft-project-management-software?tab=1&OCID=AID2000748_SEM_5j2j5X4B&MarinID=5j2j5X4B%7C78821275986631%7C%2Bproject%20%2Bo365%7Cbb%7Cc%7C%7C1261139959949905%7Ckwd-78821311481635:loc-190&lnkd=Bing_O365SMB_App&msclkid=185eccc165db1d3da290924720afcaa4&ef_id=XoY8vgAAAUTu0Bj8:20200402200513:s)เพื่อเรียนรู้เกี่ยวกับความแตกต่างระหว่าง**แผนโครงการ 1, 3 และ 5**
- หากต้องการ**ซื้อ**Project ให้ไปที่**[บริการเรียกเก็บเงิน](https://go.microsoft.com/fwlink/p/?linkid=868433)> ซื้อ** ใช้ช่องค้นหาเพื่อค้นหาใน "โครงการ"; เลือกแผนและจํานวนผู้ใช้ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ซื้อการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/buy-another-subscription?view=o365-worldwide)
- เมื่อต้องการ**สลับแผน**ให้ดูที่[สลับแผน Microsoft 365 สําหรับธุรกิจด้วยตนเอง](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-plans-manually?view=o365-worldwide)
- เมื่อต้องการ**มอบหมาย**สิทธิ์การใช้งานโครงการ ให้ไปที่**สิทธิ์การใช้งาน >[การเรียกเก็บเงิน](https://go.microsoft.com/fwlink/p/?linkid=842264)** สําหรับข้อมูลเพิ่มเติม โปรดดูที่[มอบหมายสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide)
- เมื่อต้องการ**เพิ่ม/เอา**สิทธิ์การใช้งานของโครงการออก ให้ไปที่**การเรียกเก็บเงิน > ซื้อ[บริการ](https://go.microsoft.com/fwlink/p/?linkid=868433)** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การเพิ่ม/เอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)ออก
- เมื่อต้องการ**ยกเลิก**สิทธิ์การใช้งานโครงการ ให้ไปที่ **> การเรียกเก็บเงิน &[ผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)** ให้เลือก**การตั้งค่า& การดําเนินการ**และคลิก**ยกเลิกการสมัครใช้งาน** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ยกเลิกการสมัครใช้งานของคุณ](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/cancel-your-subscription)

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[จัดการสิทธิ์การใช้งานการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)
