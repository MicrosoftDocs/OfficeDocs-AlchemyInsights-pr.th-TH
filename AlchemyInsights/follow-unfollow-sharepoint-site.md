# <a name="follow-or-un-follow-a-sharepoint-site"></a>ติดตามหรือยกเลิกการติดตามไซต์ SharePoint

มีหลายวิธีในการติดตามไซต์ที่คุณสนใจใน SharePoint วิธีนี้จะช่วยให้คุณสามารถค้นหาไซต์ได้อย่างง่ายดายในรายการของคุณและช่วยให้คุณสามารถดูกิจกรรมจากเว็บไซต์ได้ 

ดูวิดีโอเพื่อติด[ตามไซต์ SharePoint](https://support.office.com/article/Video-Follow-a-SharePoint-site-33DB6FA5-9528-45D7-BCC7-F9C1FAAACAE0) 

หากคุณไม่สามารถติดตามหรือยกเลิกการติดตามเว็บไซต์และรู้สึกว่าผลลัพธ์ไม่ได้เป็นสิ่งที่คาดหวังให้ดูว่าปัญหาเกี่ยวข้องกับผู้ใช้คนเดียวหรือผู้ใช้ทั้งหมดภายในองค์กรของคุณ หากพบปัญหาของผู้ใช้ทั้งหมดอาจมีปัญหาการบริการแบบถาวร 

ตรวจสอบ[แดชบอร์ดความสมบูรณ์ของบริการ](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)เพื่อดูว่าองค์กรของคุณได้รับผลกระทบหรือไม่
