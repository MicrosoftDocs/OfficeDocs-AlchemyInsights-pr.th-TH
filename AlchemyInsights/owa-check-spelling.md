# <a name="use-your-browser-to-check-spelling"></a>ใช้เบราว์เซอร์ของคุณเพื่อตรวจสอบการสะกด

ไม่มีตัวเลือกการตรวจสอบการสะกดใน Outlook บนเว็บ เมื่อต้องการตรวจสอบการสะกดของคุณ คุณจำเป็นต้องใช้เว็บเบราว์เซอร์ของคุณ การตรวจการสะกดไม่พร้อมใช้งานใน Microsoft ขอบ Internet Explorer 10 และรุ่นที่ใหม่กว่า และ Firefox โครเมียม และ Safari รุ่นปัจจุบัน ตรวจสอบตัวเลือกสำหรับเว็บเบราว์เซอร์ของคุณเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีการตรวจสอบการสะกด
