# <a name="upload-files-to-onedrive-or-sharepoint"></a>อัปโหลดไฟล์ไปยัง OneDrive หรือ SharePoint

คุณสามารถอัปโหลดไฟล์ไปยังไลบรารีเอกสารใน SharePoint Online ได้โดยการลากไฟล์จากคอมพิวเตอร์ของคุณแล้วปล่อยลงในไลบรารีเอกสาร เมื่อใช้ OneDrive คุณสามารถซิงค์ไฟล์ระหว่างคอมพิวเตอร์ของคุณกับระบบคลาวด์เพื่อให้คุณสามารถเข้าถึงไฟล์ของคุณได้จากทุกที่ นอกจากนี้คุณยังสามารถทำงานกับไฟล์ที่ซิงค์ของคุณได้โดยตรงใน File Explorer และเข้าถึงไฟล์ของคุณได้แม้ในขณะที่คุณออฟไลน์อยู่ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการอัปโหลดและการซิงค์โปรดไปที่ลิงก์ด้านล่าง
