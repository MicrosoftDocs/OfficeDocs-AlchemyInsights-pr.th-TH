# <a name="recover-missing-or-deleted-site-collections"></a>กู้คืนไซต์คอลเลกชันที่สูญหายหรือถูกลบ

ดูบทความต่อไปนี้สำหรับข้อมูลเกี่ยวกับวิธีการกู้คืนไซต์คอลเลกชันที่สูญหายหรือถูกลบใน SharePoint

[การคืนค่าไซต์ที่ถูกลบ](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[การคืนค่าโดยใช้โมดู SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

ถ้าคุณไม่พบไซต์ที่คุณอาจต้องการ[ค้นหาบันทึกการตรวจสอบในศูนย์การ &amp; ปฏิบัติตามนโยบายด้านความปลอดภัยของ Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance)

**หมายเหตุ:** ไซต์ทีม "ทันสมัย" ไม่สามารถมองเห็นได้ใน UI ผู้ดูแลระบบ SharePoint แบบ**คลาสสิก** คุณสามารถเข้าถึงรายการของไซต์ "สมัยใหม่" จาก [ศูนย์การจัดการ Sharepoint ใหม่](https://docs.microsoft.com/sharepoint/get-started-new-admin-center)ได้
