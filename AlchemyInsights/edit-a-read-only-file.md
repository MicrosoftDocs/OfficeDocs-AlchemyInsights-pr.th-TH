# <a name="edit-a-read-only-file"></a>แก้ไขแฟ้มแบบอ่านอย่างเดียว

ในไลบรารีเอกสาร เมื่อคุณคลิกที่ชื่อของเอกสาร โปรแกรม Office อาจโดยไม่คาดคิดเปิดเป็นแฟ้มแบบอ่านอย่างเดียวนี้ เมื่อต้องการเปิดแฟ้มสำหรับแก้ไข คลิก**เปิด**(สามจุด), และจากนั้น คลิก**Edit.**
