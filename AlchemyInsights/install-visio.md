# <a name="install-visio"></a>ติดตั้ง Visio

Visio จะขายเป็นแอปพลิเคชันแบบสแตนด์อโลนดังนั้นหากคุณยังไม่มีมัน[ซื้อหรือลองใช้ Visio](https://products.office.com/visio) 

**หมายเหตุ**: Visio Online Plan 1 ไม่มี Visio เวอร์ชันเดสก์ท็อป ดังนั้น คุณจะไม่สามารถติดตั้งได้

ถ้าคุณมีการสมัครใช้งาน Visio Online Plan 2 คุณต้อง[กําหนดสิทธิ์การใช้งาน](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/assign-licenses-to-users?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764)ให้กับผู้ใช้ หรือจะไม่มีตัวเลือกในการติดตั้ง Visio จาก**การติดตั้งของฉัน**ใน[พอร์ทัลบัญชีของฉัน](https://portal.office.com/account#installs) 

รุ่นสิทธิ์การใช้งานแบบไม่ใช่ไดรฟ์ข้อมูลของ Visio Standard หรือ Visio Professional จะเชื่อมโยงกับบัญชี Microsoft และไม่สามารถจัดการได้ในศูนย์การจัดการ เมื่อต้องการติดตั้ง[account.microsoft.com](https://account.microsoft.com)รุ่นเหล่านี้

สําหรับวิธีใช้โดยละเอียดเพิ่มเติม โปรดดูที่[ติดตั้ง Visio](https://support.office.com/article/f98f21e3-aa02-4827-9167-ddab5b025710?wt.mc_id=OfficeAdm_ClientDIA_Alchemy1764)และทําตามขั้นตอนสําหรับ Visio รุ่นของคุณ
