# <a name="exchange-license-management"></a>การจัดการใบอนุญาตแลกเปลี่ยน

การจัดการสิทธิ์การใช้งาน Exchange สามารถเสร็จสมบูรณ์ใน**ศูนย์การจัดการ Microsoft 365**

- [เปรียบเทียบตัวเลือกการแลกเปลี่ยน](https://www.microsoft.com/microsoft-365/exchange/compare-microsoft-exchange-online-plans)เพื่อเรียนรู้เกี่ยวกับความแตกต่างระหว่าง**แผนการแลกเปลี่ยนออนไลน์ 1, 2 และพรีเมี่ยมธุรกิจ**
- หากต้องการ**ซื้อ**Exchange ให้ไปที่**บริการการเรียกเก็บเงิน>[ซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)** ใช้กล่องค้นหาเพื่อค้นหาใน "Exchange"; เลือกแผนและจํานวนผู้ใช้ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การซื้อการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/buy-another-subscription?view=o365-worldwide)
- เมื่อต้องการ**สลับแผน**ให้ดูที่[สลับ Microsoft 365 สําหรับแผนธุรกิจด้วยตนเอง](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-plans-manually?view=o365-worldwide)
- เมื่อต้องการ**มอบหมาย**สิทธิ์การใช้งาน Exchange ให้ไปที่**[สิทธิ์การใช้งาน](https://go.microsoft.com/fwlink/p/?linkid=842264)>การเรียกเก็บเงิน**ให้เลือกการสมัครใช้งาน แล้วเลือกผู้ใช้ แล้วคลิก**มอบหมายสิทธิ์การใช้งาน** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[มอบหมายสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide)
- เมื่อต้องการ**เพิ่ม/เอา**สิทธิ์การใช้งาน Exchange ออก ให้ไปที่**[บริการ&ผลิตภัณฑ์>](https://go.microsoft.com/fwlink/p/?linkid=842054)การเรียกเก็บเงิน**แล้วคลิก**เพิ่ม/เอาสิทธิ์การใช้งานออก** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[เพิ่ม/เอาสิทธิ์การใช้งานออก](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)
- เมื่อต้องการ**ยกเลิก**สิทธิ์การใช้งาน Exchange ให้ไปที่**[บริการ&>การเรียกเก็บเงินผลิตภัณฑ์](https://go.microsoft.com/fwlink/p/?linkid=842054)**&**การดําเนินการ**& แล้วคลิก**ยกเลิกการสมัครใช้งาน** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ยกเลิกการสมัครใช้งานของคุณ](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/cancel-your-subscription)

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[จัดการสิทธิ์การใช้งานการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)
