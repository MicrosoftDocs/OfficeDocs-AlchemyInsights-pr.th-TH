# <a name="set-up-teams-federation"></a>ตั้งค่าการติดต่อกับทีม

หากคุณต้องการความช่วยเหลือในการตั้งค่าการเข้าถึงภายนอก (สหพันธรัฐ) ใน Teams คุณจะต้องเปิดการเข้าถึงจากภายนอกในศูนย์การจัดการทีม

ตรวจสอบ[จัดการการเข้าถึงภายนอกใน Microsoft Teams](https://docs.microsoft.com/microsoftteams/manage-external-access)สําหรับตัวเลือกและข้อมูลการกําหนดค่าเพิ่มเติม
