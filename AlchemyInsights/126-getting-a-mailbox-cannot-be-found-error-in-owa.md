# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a>การเรียกดูกล่องจดหมายไม่พบข้อผิดพลาดใน Outlook บนเว็บหรือไม่

ถ้าคุณกำลังใช้ Outlook บนเว็บ และคุณได้รับข้อผิดพลาดที่**ไม่สามารถพบกล่องจดหมายสำหรับ**บัญชีที่คุณใช้เพื่อเชื่อมต่อกับ Outlook บนเว็บ ไม่มีสิทธิ์การใช้งานการแลกเปลี่ยนแบบออนไลน์ และดังนั้น ไม่มีกล่องจดหมายถูกเชื่อมโยงกับลูกค้าองค์กร ผู้ดูแลระบบของคุณสามารถกำหนดสิทธิ์การใช้งานบัญชีของคุณ โดยทำตามขั้นตอนเหล่านี้: 
  
1. เปิด[Office 365 admin ศูนย์](https://portal.office.com/adminportal/home#/homepage)และ**ผู้ใช้ที่ใช้งานอยู่**ใต้**แก้ไขผู้ใช้**
    
2. ใน**การแก้ไขผู้ใช้**เพจที่เปิด เลือกผู้ใช้ ในหน้าคุณสมบัติของผู้ใช้ที่เปิด คลิก**แก้ไข**สำหรับ**สิทธิ์การใช้งานผลิตภัณฑ์**
    
3. ในเพจ**สิทธิ์การใช้งานผลิตภัณฑ์**ที่เปิด เลือกค่า**ตำแหน่งที่ตั้ง**ที่เหมาะสม และกำหนดสิทธิ์การใช้งานที่ประกอบด้วยการแลกเปลี่ยนแบบออนไลน์ (ขยายสิทธิ์การใช้งานเพื่อดูรายละเอียดของ) เมื่อคุณทำเสร็จแล้ว คลิก**บันทึก**
