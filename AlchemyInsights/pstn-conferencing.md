# <a name="pstn-calling-with-skype-for-business-online"></a>การเรียก PSTN ด้วย Skype สําหรับธุรกิจออนไลน์

เมื่อต้องการทําการเรียก PSTN ใน Microsoft 365 คุณจําเป็นต้องมี[แผนการโทร Microsoft 365](https://docs.microsoft.com/microsoftteams/what-is-phone-system-in-office-365#more-about-calling-plans)หรือการเชื่อมต่อกับโครงสร้างพื้นฐานระบบโทรศัพท์ PSTN ในสถานที่ของคุณ สําหรับข้อมูลเพิ่มเติม ให้ดูทรัพยากรต่อไปนี้: 

- [ประเทศและภูมิภาคพร้อมใช้งานสําหรับการประชุมทางเสียงและแผนการโทร](https://docs.microsoft.com/microsoftteams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans) 
- [คําถามทั่วไปเกี่ยวกับการประชุมทางเสียง](https://docs.microsoft.com/microsoftteams/audio-conferencing-common-questions)
- [เพิ่มเงินทุนและจัดการเครดิตการสื่อสาร](https://docs.microsoft.com/microsoftteams/add-funds-and-manage-communications-credits)
- [ผู้โทร PSTN ไม่สามารถเข้าร่วม Skype สําหรับธุรกิจการเรียกประชุมทางโทรศัพท์](https://docs.microsoft.com/SkypeForBusiness/troubleshoot/online-conferencing/pstn-callers-cant-join-dial-in-call)
