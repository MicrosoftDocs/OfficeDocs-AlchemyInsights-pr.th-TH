# <a name="onedrive-site-provisioning"></a>การเตรียมใช้งานไซต์ OneDrive

สำหรับบัญชีที่สร้างขึ้นใหม่ผู้ใช้ต้องลงชื่อเข้าใช้ศูนย์ดูแลของ Microsoft ๓๖๕กับบัญชีของตนเองและคลิกไอคอน OneDrive เพื่อเริ่มต้นบทบัญญัติของไซต์ OneDrive ด้วยตนเอง

อย่างไรก็ตามถ้าผู้ใช้ได้คลิกที่ไอคอน OneDrive และเว็บไซต์ส่วนบุคคลของเขาไม่ได้จัดสรรภายใน24ชั่วโมงโปรดล็อกตั๋วสนับสนุนสำหรับความช่วยเหลือเพิ่มเติม
