# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive จะติดอยู่ในการประมวลผลการเปลี่ยนแปลง

ถ้า OneDrive แสดง "การประมวลผลการเปลี่ยนแปลง" เป็นเวลานาน อาจเป็นเพราะคุณมีแฟ้มออนไลน์เปิดอยู่ อ่าน[OneDrive จะติดอยู่ใน "การประมวลผลการเปลี่ยนแปลง"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd)สําหรับขั้นตอนที่แนะนําและข้อมูลเพิ่มเติม

ถ้าคุณต้องการความช่วยเหลือเพิ่มเติม คุณสามารถ[ส่งอีเมลถึงฝ่ายสนับสนุนของ OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676)
