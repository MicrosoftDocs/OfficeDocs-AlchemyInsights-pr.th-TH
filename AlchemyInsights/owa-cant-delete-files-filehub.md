# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>ไม่สามารถลบแฟ้มหรือรูปถ่ายจากแฟ้มใน Outlook บนเว็บ

**แฟ้ม**แสดงแฟ้มและรูปภาพที่แนบไปกับข้อความในกล่องจดหมายของคุณ เมื่อต้องการลบแฟ้มและรูปภาพจากแฟ้ม คุณต้องการลบข้อความอีเมลที่แนบไป หลังจากการลบข้อความ คุณต้องลบออกจากโฟลเดอร์รายการที่ถูกลบของคุณด้วย
