# <a name="get-the-microsoft-365-f3-plan"></a>รับแผน Microsoft ๓๖๕ F3

เชื่อมต่อพนักงาน Firstline ของคุณด้วยเครื่องมือที่มีอยู่แล้วภายในและประสบการณ์ที่อนุญาตให้พวกเขาทำงานที่ดีที่สุดโดยใช้ Microsoft ๓๖๕ F3 plan คุณสามารถเพิ่มการสมัครใช้งาน F3 จาก**ศูนย์การจัดการ Microsoft ๓๖๕**

1. ไปที่การ**เรียกเก็บเงิน >[บริการการซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)**

2. ใช้กล่องค้นหาเพื่อค้นหา "F3"

3. เลือกจำนวนผู้ใช้ที่จะใช้แผน "F3"

เมื่อต้องการ**เรียนรู้**เกี่ยวกับการสมัครใช้งาน F3 ให้ดูที่[Microsoft ๓๖๕ F3](https://www.microsoft.com/microsoft-365/microsoft-365-enterprise-f3?activetab=pivot%3aoverviewtab)
