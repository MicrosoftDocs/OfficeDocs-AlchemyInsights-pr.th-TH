# <a name="powerpoint-common-issues-and-resolutions"></a>ปัญหาทั่วไปและวิธีแก้ปัญหาของ PowerPoint

ต่อไปนี้คือหัวข้อที่ร้องขอบ่อยด้วย PowerPoint:

1. [PowerPoint ไม่ตอบสนองแฮงค์หรือหยุด](https://support.office.com/article/PowerPoint-isn-t-responding-hangs-or-freezes-652ede6e-e3d2-449a-a07f-8c800dfb948d)ทำงาน

2. [PowerPoint ไม่สามารถแทรกวิดีโอจากไฟล์ที่เลือก](https://support.office.com/article/PowerPoint-cannot-insert-a-video-from-the-selected-file-acd46430-9e0c-4dca-9484-19cf0afdde7c)ได้

3. **วิดีโอ**: การ[แทรกไฟล์วิดีโอจากคอมพิวเตอร์ของคุณ](https://support.office.com/article/insert-and-play-a-video-file-from-your-computer-f3fcbd3e-5f86-4320-8aea-31bff480ed02)ให้[แทรกวิดีโอจาก YouTube](https://support.office.com/article/Insert-a-video-from-YouTube-or-another-site-8340ec69-4cee-4fe1-ab96-4849154bc6db)และ[รูปแบบไฟล์เสียงและวิดีโอที่ได้รับการสนับสนุนใน PowerPoint](https://support.office.com/article/video-and-audio-file-formats-supported-in-powerpoint-d8b12450-26db-4c7b-a5c1-593d3418fb59)

4. การ**พิมพ์**: สำหรับแนวทางปฏิบัติที่ดีที่สุดและปัญหาที่เกี่ยวข้องกับการพิมพ์ให้ตรวจสอบข้อมูลใน[พิมพ์ภาพนิ่งและบันทึกย่อของ PowerPoint ของคุณ](https://support.office.com/article/Print-your-PowerPoint-slides-handouts-or-notes-194d4320-aa03-478b-9300-df25f0d15dc4) 

5. **ฟอนต์**: เมื่อต้องการแชร์ฟอนต์กับงานนำเสนอของคุณให้อ่าน[แบบอักษรฝังตัวใน Word หรือ PowerPoint](https://support.office.com/article/Embed-fonts-in-Word-or-PowerPoint-cb3982aa-ea76-4323-b008-86670f222dbc)

6. **ฟีเจอร์**: [เปรียบเทียบฟีเจอร์ PowerPoint](https://support.office.com/article/Compare-PowerPoint-features-on-different-platforms-90986850-227c-4b25-938e-1c5838166b8b#bm11) ข้ามแพลตฟอร์ม Windows, Mac, เว็บและอุปกรณ์เคลื่อนที่
