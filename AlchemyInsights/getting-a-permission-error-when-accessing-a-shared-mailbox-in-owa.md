# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>ได้รับข้อผิดพลาดสิทธิ์เมื่อเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บหรือไม่

หากคุณได้รับข้อผิดพลาด**บางสิ่งบางอย่างที่ผิด คุณไม่มีสิทธิ์ในการเปิดกล่องจดหมายนี้** เมื่อคุณพยายามเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บบัญชีของคุณอาจไม่มีสิทธิ์ในการเข้าถึงกล่องจดหมายที่ใช้ร่วมกัน ผู้ดูแลระบบองค์กรของคุณสามารถกำหนดสิทธิ์ของบัญชีของคุณให้กับกล่องจดหมายที่ใช้ร่วมกันได้โดยทำตามคำแนะนำในหัวข้อ[สร้างกล่องจดหมายที่ใช้ร่วมกัน](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox)
