# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: การแก้ไขปัญหา 'การประมวลผลการเปลี่ยนแปลง'

ถ้าคุณเห็น "ประมวลผลการเปลี่ยนแปลง" เป็นระยะเวลานานของเวลา อาจเป็น เพราะคุณมีแบบออนไลน์เปิดแฟ้ม ล็อตของแฟ้มที่ติดอยู่ในคิว หรือเป็นแฟ้มที่มีขนาดใหญ่มากทำการซิงค์ ดู[OneDrive ติดบนการประมวลผล](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd)

อ่านบทความนี้เพื่อแก้ไขปัญหา "เปลี่ยนแปลงการประมวลผล" คุณจะเห็นข้อความนี้ถ้าคุณมีแฟ้มที่เปลี่ยนแปลงตลอดเวลา (เช่นเดียวกับ Outlook แฟ้ม PST หรือฐานข้อมูล) เรียนรู้วิธีการ[ย้ายแฟ้มข้อมูล Outlook](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e)ที่คุณใช้อยู่เป็นประจำออกจาก OneDrive
