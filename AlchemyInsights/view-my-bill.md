# <a name="view-my-bill-or-invoice"></a>ดูใบเรียกเก็บเงินหรือใบแจ้งหนี้ของฉัน

1. ในศูนย์การจัดการ Microsoft 365 ไปที่หน้า**Billing**\>[ใบเรียกเก็บเงิน&การชําระเงิน](https://go.microsoft.com/fwlink/p/?linkid=848039)

2. เลือกช่วงวันที่จากรายการดรอปดาวน์ แล้วเลือกลําดับที่คุณต้องการดู

3. หากต้องการพิมพ์ใบเรียกเก็บเงิน ให้ดาวน์โหลดใบเรียกเก็บเงินเป็นไฟล์ PDF แล้วคลิก**พิมพ์**

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การดูใบเรียกเก็บเงินหรือใบแจ้งหนี้ของคุณ](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/view-your-bill-or-invoice)
