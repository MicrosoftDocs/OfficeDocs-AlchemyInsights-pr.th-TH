# <a name="too-many-licenses"></a>มีสิทธิ์การใช้งานมากเกินไปใช่หรือไม่

ถ้าคุณคิดว่าคุณกำลังถูกเรียกเก็บเงินสำหรับสิทธิ์การใช้งานจำนวนมากเกินไปซึ่งมักจะเกิดจากการซื้อสิทธิ์การใช้งานเพิ่มเติมที่คุณต้องการ
  
**ลดจำนวนสิทธิ์การใช้งานที่ซื้อ**
  
1. ในศูนย์การจัดการให้ไปที่หน้าการ**เรียกเก็บเงิน** \> **[ของผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการเอาสิทธิ์การใช้งานออกแล้วเลือก **เพิ่ม/เอาสิทธิ์**การใช้งานออก

3. ในกล่องสิทธิ์การใช้งาน**ทั้งหมด**ให้ใส่จำนวนสิทธิ์การใช้งานทั้งหมดที่คุณจำเป็นต้องใช้สำหรับการสมัครใช้งานนี้แล้วเลือก**ส่งการเปลี่ยนแปลง** ตัวอย่างเช่นถ้าคุณมีสิทธิ์การใช้งาน๑๑๐และคุณต้องการเอาออก5รายการให้ใส่๑๐๕

เมื่อต้องการเรียนรู้เพิ่มเติมให้ดูที่[เอาสิทธิ์การใช้งานออกจากการสมัครใช้งาน Microsoft ๓๖๕สำหรับธุรกิจของคุณ](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)
