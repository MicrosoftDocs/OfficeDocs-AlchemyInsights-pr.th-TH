# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive ค้างอยู่ที่การเปลี่ยนแปลงการประมวลผล

ถ้า OneDrive แสดง "กำลังประมวลผลการเปลี่ยนแปลง" เป็นเวลานานอาจเป็นเพราะว่าคุณมีไฟล์ออนไลน์เปิดอยู่ไฟล์จำนวนมากที่ค้างอยู่ในคิวหรือการซิงค์ไฟล์ที่มีขนาดใหญ่มาก อ่าน  [OneDrive ค้างอยู่ที่ "กำลังประมวลผลการเปลี่ยนแปลง"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) สำหรับขั้นตอนที่แนะนำและข้อมูลเพิ่มเติม

ถ้าคุณต้องการความช่วยเหลือเพิ่มเติมคุณสามารถ[ส่งอีเมลถึงฝ่ายสนับสนุน OneDrive](https://go.microsoft.com/fwlink/p/?LinkId=528676)
