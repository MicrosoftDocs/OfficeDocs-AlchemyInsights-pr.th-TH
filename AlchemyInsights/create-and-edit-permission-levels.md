# <a name="working-with-sharepoint-permission-levels"></a>การทำงานกับระดับสิทธิ์ของ SharePoint

SharePoint มาพร้อมกับระดับสิทธิ์เริ่มต้นที่คุณสามารถใช้ได้เช่นการมีส่วน**ร่วม**และ**มุมมองเท่านั้น** แต่ถ้าหนึ่งในนั้นไม่ตรงกับความต้องการของคุณคุณสามารถสร้างระดับสิทธิ์ใหม่ได้ โปรดทราบว่าการติดตามระดับสิทธิ์จะง่ายขึ้นหากคุณไม่ได้เปลี่ยนค่าเริ่มต้น

[วิธีการสร้างและแก้ไขระดับสิทธิ์](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
