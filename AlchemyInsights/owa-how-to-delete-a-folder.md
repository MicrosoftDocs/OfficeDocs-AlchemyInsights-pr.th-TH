# <a name="how-to-delete-a-folder-in-outlook-on-the-web"></a>วิธีการลบโฟลเดอร์ใน Outlook บนเว็บ

1. ในบานหน้าต่างโฟลเดอร์ คลิกขวาโฟลเดอร์หรือโฟลเดอร์ย่อยที่คุณต้องการลบ
2. เลือก**ลบโฟลเดอร์**
3. เลือก**ตกลง**เพื่อยืนยัน
