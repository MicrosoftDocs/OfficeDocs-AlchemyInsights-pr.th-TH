# <a name="how-to-access-outlookcom-while-traveling"></a>วิธีการเข้าถึง Outlook.com ขณะเดินทาง

เมื่อต้องการเข้าถึง Outlook.com ขณะเดินทางตรวจสอบให้แน่ใจว่าข้อมูลความปลอดภัยของบัญชี Microsoft ของคุณ-ที่อยู่อีเมลสำรองหรือหมายเลขโทรศัพท์ที่คุณจะได้รับรหัสความปลอดภัย-เป็นเวอร์ชันล่าสุด เมื่อต้องการเรียนรู้เพิ่มเติมให้ดู[ที่วิธีการเข้าถึง Outlook.com เมื่อเดินทาง](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy)
