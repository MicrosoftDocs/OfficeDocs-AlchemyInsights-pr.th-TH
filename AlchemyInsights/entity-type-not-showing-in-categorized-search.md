# <a name="entity-type-not-showing-in-search-results"></a>ชนิดของเอนทิตีที่ไม่ปรากฏอยู่ในผลลัพธ์การค้นหา

สำหรับชนิดเอนทิตีจะแสดงในการจัดประเภทค้นหาหรือความเกี่ยวข้องค้นหาในอินเทอร์เฟซสำหรับรวม Dynamics 365 นั้นต้องก่อนเพิ่มแอพลิเคชันโมดูล

หมายเหตุ: เมื่อคุณค้นหาสิ่งที่แนบมาในการค้นหาที่เกี่ยวข้อง เอนทิตีที่แนบต้องเพิ่มแอพลิเคชันโมดูลแม้ว่าระเบียนหลักคือ สิ่งที่จะแสดงในผลลัพธ์
