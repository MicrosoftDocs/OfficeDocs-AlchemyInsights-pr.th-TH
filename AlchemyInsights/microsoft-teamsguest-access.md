# <a name="microsoft-teams---guest-access"></a>ทีมงาน Microsoft - การเข้าถึงของแขก

ถ้าคุณจำเป็นต้องเปิดใช้งาน หรือการตั้งค่าคอนฟิกบทบาทการเข้าถึงของแขกในทีม Microsoft เรามีคำแนะนำคำแนะนำทีละขั้นตอนสำหรับคุณ:[รายการตรวจสอบการเข้าถึงของแขก](https://docs.microsoft.com/en-us/microsoftteams/guest-access-checklist)
  
คุณอาจต้องรอถึง 24 ชั่วโมงสำหรับทีมงาน ใช้การเปลี่ยนแปลงการตั้งค่าคอนฟิกจะใช้ได้
