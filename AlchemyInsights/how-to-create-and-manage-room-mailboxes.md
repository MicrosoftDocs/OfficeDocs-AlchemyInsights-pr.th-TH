# <a name="how-to-create-and-manage-room-mailboxes"></a>วิธีการสร้าง และจัดการกล่องจดหมายในห้อง

เมื่อต้องการสร้าง และปรับเปลี่ยนกล่องจดหมายในห้อง ดู[สร้าง และจัดการกล่องจดหมายในห้อง](https://technet.microsoft.com/library/jj215781.aspx)

เมื่อต้องการเรียนรู้เพิ่มเติมเกี่ยวกับการกล่องจดหมายของห้องและเครื่องมือ (หรือที่เรียกอีกอย่างหนึ่งว่าทรัพยากรกล่องจดหมาย), ดู[กล่องจดหมายห้องและอุปกรณ์](https://support.office.com/article/9f518a6d-1e2c-4d44-93f3-e19013a1552b)
