# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>สิทธิ์กล่องจดหมายที่ได้รับมอบสิทธิ์ในสภาพแวดล้อมไฮบริดสลี

คุณอาจต้องดําเนินการขั้นตอนเพิ่มเติมบางอย่างเพื่อตั้งค่าสิทธิ์กล่องจดหมายที่ได้รับมอบสิทธิ์ระหว่างองค์กรในสถานที่ของคุณและ Exchange Online ใน Microsoft 365 ทั้งนี้ขึ้นอยู่กับรุ่นของ Exchange ที่คุณได้ติดตั้งไว้ในองค์กรในสถานที่ของคุณ สําหรับข้อมูลเพิ่มเติมเกี่ยวกับวิธีการตั้งค่าสิทธิ์ที่มอบหมาย ให้ดูที่[การตั้งค่าคอนฟิก Exchange เพื่อสนับสนุนสิทธิ์กล่องจดหมายที่ได้รับมอบสิทธิ์ในการปรับใช้ไฮบริดสลี](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx)
