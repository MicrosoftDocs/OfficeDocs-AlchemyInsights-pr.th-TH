# <a name="learn-about-using-public-folders-in-outlook"></a>เรียนรู้เกี่ยวกับการใช้โฟลเดอร์สาธารณะใน Outlook

โฟลเดอร์สาธารณะใน Outlook มอบสถานที่ที่ดีในการเก็บรวบรวม จัดระเบียบ และแชร์ข้อมูลเกี่ยวกับหัวข้อหรือโครงการต่างๆ ภายในองค์กรของคุณ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การสร้างและการใช้โฟลเดอร์สาธารณะร่วมกันใน Outlook](https://support.office.com/article/a2835011-d524-4a5c-a207-05c159bb2a97)
