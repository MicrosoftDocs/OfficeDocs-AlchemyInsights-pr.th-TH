# <a name="onedrive-troubleshoot-processing-changes"></a>OneDrive: แก้ไขปัญหา ' การประมวลผลการเปลี่ยนแปลง '

ถ้าคุณเห็น "การประมวลผลการเปลี่ยนแปลง" สำหรับระยะเวลาที่ขยายออกไปอาจเป็นไปได้เนื่องจากการมีไฟล์แบบออนไลน์เปิดอยู่จำนวนไฟล์ที่ค้างอยู่ในคิวหรือการซิงค์ไฟล์ขนาดใหญ่มากให้ดูที่ [OneDrive: แก้ไขปัญหาการประมวลผลการเปลี่ยนแปลง '](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) เพื่อช่วยแก้ไขปัญหานี้

นอกจากนี้คุณจะเห็นข้อความนี้ถ้าคุณมีไฟล์ที่เปลี่ยนแปลงตลอดเวลา (เช่น Outlook ไฟล์ PST หรือฐานข้อมูล) เรียนรู้วิธี [ย้ายไฟล์ข้อมูล Outlook ของคุณ](https://support.office.com/article/how-to-remove-an-outlook-pst-data-file-from-onedrive-b6b9e522-59bd-40f7-949f-168d0aa9b38e) ที่คุณใช้อยู่เป็นประจำจาก OneDrive
