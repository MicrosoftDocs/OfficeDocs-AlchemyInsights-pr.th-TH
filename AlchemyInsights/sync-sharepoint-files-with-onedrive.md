# <a name="troubleshoot-open-with-explorer-issues-in-sharepoint-online"></a>การแก้ไขปัญหา "เปิดด้วย Explorer" ใน SharePoint Online

ทำตามขั้นตอนและแนวทางปฏิบัติที่ดีที่สุดในบทความต่อไปนี้:

- [วิธีใช้คำสั่ง "เปิดด้วย Explorer" เพื่อแก้ไขปัญหาใน SharePoint Online](https://docs.microsoft.com/sharepoint/troubleshoot/lists-and-libraries/troubleshoot-issues-using-open-with-explorer)

- [คัดลอกหรือย้ายไฟล์ไลบรารีโดยใช้เปิดด้วย Explorer](https://support.microsoft.com/office/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2?ui=en-us&rs=en-us&ad=us)

> [!NOTE]
- เราขอแนะนำให้ [ซิงค์ไฟล์ SharePoint กับไคลเอ็นต์การซิงค์ OneDrive ใหม่](https://support.microsoft.com/office/sync-sharepoint-and-teams-files-with-your-computer-6de9ede8-5b6e-4503-80b2-6190f3354a88?ui=en-us&rs=en-us&ad=us) ที่มี [ไฟล์ตามความต้อง](https://support.microsoft.com/office/save-disk-space-with-onedrive-files-on-demand-for-windows-10-0e6860d3-d9f3-4971-b321-7092438fb38e?ui=en-us&rs=en-us&ad=us) การเนื่องจากการซิงโครไนซ์ให้สิทธิ์การเข้าถึงไฟล์ของคุณภายในเครื่องของคุณและมีประสิทธิภาพการทำงานที่ดีที่สุด

- **เปิดด้วย Explorer** เท่านั้นที่ได้รับการสนับสนุนใน Internet Explorer 11 สำหรับข้อมูลเพิ่มเติมให้ดู [สิ้นสุดการสนับสนุนสำหรับ IE11 ด้วยแอป Microsoft ๓๖๕](https://docs.microsoft.com/lifecycle/announcements/m365-ie11-microsoft-edge-legacy)) **เปิดด้วย Explorer** ไม่ทำงานใน Windows ด้วย Microsoft Edge, Google Chrome, Mozilla Firefox หรือบนแพลตฟอร์ม Mac เนื่องจากสาเหตุนี้ตัวเลือก **มุมมอง Explorer** อาจเป็นสีเทา 

- ปุ่ม **เปิดด้วย Explorer** ไม่ปรากฏในประสบการณ์การใช้งานไลบรารีใหม่ เลือกเมนูดรอปดาวน์ **มุมมอง** ที่มุมขวาบน (ชื่อของการเปลี่ยนแปลงแบบหล่นลงทั้งนี้ขึ้นอยู่กับมุมมองปัจจุบันของคุณ) จากนั้นเลือก **ดูใน File Explorer**
