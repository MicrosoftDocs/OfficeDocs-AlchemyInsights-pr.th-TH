# <a name="upload-files-to-onedrive-or-sharepoint"></a>อัปโหลดแฟ้ม OneDrive หรือ SharePoint

คุณสามารถอัปโหลดแฟ้มไปยังไลบรารีเอกสารใน SharePoint แบบออนไลน์ โดยเพียงแค่ลากเครื่องเหล่านั้นจากคอมพิวเตอร์ของคุณ และปล่อยรายการเหล่านั้นลงในไลบรารีเอกสาร กับ OneDrive คุณสามารถซิงค์แฟ้มระหว่างคอมพิวเตอร์ของคุณและ cloud เพื่อให้คุณสามารถเข้าถึงแฟ้มของคุณจากที่ใดก็ นอกจากนี้คุณยังสามารถทำงานกับแฟ้มของคุณซิงค์โดยตรงใน Explorer แฟ้ม และเข้าถึงแฟ้มของคุณแม้ว่าคุณอยู่ในโหมดออฟไลน์ สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการอัปโหลดและการซิงค์ โปรดเยี่ยมชมลิงค์ด้านล่าง
