# <a name="outlook-error-cannot-start-microsoft-outlook"></a>ข้อผิดพลาดของ Outlook: ไม่สามารถเริ่ม Microsoft Outlook ได้

คุณมีปัญหาในการเริ่มต้น Outlook ๒๐๑๖, Outlook ๒๐๑๓หรือ Outlook ๒๐๑๐หรือได้รับ "ไม่สามารถเริ่ม Microsoft Outlook ได้ ไม่สามารถเปิดหน้าต่าง Outlook "ข้อความแสดงข้อผิดพลาดได้ใช่หรือไม่
  
ก่อนอื่นให้ใช้ขั้นตอนการแก้ไขปัญหาใน [บทความนี้](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) เพื่อช่วยแก้ไขปัญหาการเริ่มต้นเหล่านี้ 
  
นอกจากนี้คุณยังสามารถเรียกใช้ [เครื่องมือการวินิจฉัยขั้นสูง](https://aka.ms/SaRA-OutlookAdvDiagnostics) บนเครื่องจักรที่ได้รับผลกระทบ
