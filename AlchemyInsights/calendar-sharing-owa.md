# <a name="troubleshooting-issues-with-calendar-sharing"></a>การแก้ไขปัญหาเกี่ยวกับการแชร์ปฏิทิน

- เมื่อต้องการวินิจฉัยปัญหาสิทธิ์ให้ใช้ตัว [วิเคราะห์การเชื่อมต่อระยะไกลของ Exchange ว่างไม่ว่าง](https://testconnectivity.microsoft.com/Default.aspx?testId=freeBusy)ซึ่งจะให้ข้อความแสดงข้อผิดพลาดที่เฉพาะเจาะจงมาก
- นอกจากนี้คุณยังสามารถใช้ตัว [ช่วยการสนับสนุนและการกู้คืนของไมโครซอฟท์สำหรับ microsoft ๓๖๕](https://diagnostics.office.com/) เพื่อวิเคราะห์ปัญหาปฏิทินอื่นๆที่หลากหลายได้อีกด้วย 
- ถ้าผู้ใช้มีเดสก์ท็อป Outlook [เครื่องมือการตรวจสอบปฏิทินสำหรับ Outlook](https://www.microsoft.com/download/details.aspx?id=28786) สามารถแก้ไขปัญหาที่อาจเกิดขึ้นได้

แหล่งข้อมูลวิธีการเพิ่มเติม:

- [แชร์ปฏิทิน Outlook กับบุคคลอื่น](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
- [แชร์ปฏิทินของคุณใน Outlook บนเว็บสำหรับธุรกิจ](https://support.office.com/article/7ecef8ae-139c-40d9-bae2-a23977ee58d5)
