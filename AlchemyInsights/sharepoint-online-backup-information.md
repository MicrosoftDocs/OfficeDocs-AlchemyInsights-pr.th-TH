# <a name="sharepoint-online-backup-information"></a>ข้อมูลการสำรองข้อมูล SharePoint Online

เมื่อคุณลบรายการหรือไซต์ออกจาก SharePoint ไฟล์นั้นจะไม่ถูกเอาออกทันที รายการที่ถูกลบไปยังถังรีไซเคิลเป็นระยะเวลาหนึ่ง ในระหว่างช่วงเวลานั้นคุณสามารถคืนค่ารายการที่คุณลบไปยังตำแหน่งที่ตั้งเดิมได้

SharePoint Online จะเก็บข้อมูลสำรองของเนื้อหาทั้งหมดสำหรับ14วันเพิ่มเติมเกินกว่าการลบที่แท้จริง ถ้าไม่สามารถคืนค่าเนื้อหาผ่าน [ถังรีไซเคิล](https://support.microsoft.com/office/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b) หรือ [คืนค่าไฟล์](https://support.microsoft.com/office/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)ผู้ดูแลระบบสามารถติดต่อฝ่ายสนับสนุนของ Microsoft เพื่อขอคืนค่าได้ตลอดเวลาภายในหน้าต่าง14วัน บูรณะจากการสำรองข้อมูลสามารถทำให้เสร็จสมบูรณ์สำหรับไซต์คอลเลกชันหรือไซต์ย่อยเท่านั้นไม่ใช่สำหรับไฟล์รายการหรือไลบรารีที่เฉพาะเจาะจงเท่านั้น
