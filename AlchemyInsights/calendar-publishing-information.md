# <a name="calendar-publishing-information"></a>ข้อมูลการเผยแพร่ปฏิทิน

- ผู้ดูแลระบบ Microsoft ๓๖๕สามารถเปิดใช้งานหรือปิดใช้งานการเผยแพร่ปฏิทินและการแชร์ปฏิทินสำหรับองค์กรได้ สำหรับข้อมูลเพิ่มเติมโปรดดูที่[เปิดการเผยแพร่ปฏิทินเพื่อให้ผู้คนสามารถแชร์ข้อมูล](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC)และ[แชร์ปฏิทินกับผู้ใช้ภายนอก](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users)ได้

- ผู้ใช้ Outlook สามารถแชร์ปฏิทินของตนเองได้ สำหรับข้อมูลเพิ่มเติมโปรดดูที่[แชร์ปฏิทิน Outlook กับบุคคลอื่น](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88)
