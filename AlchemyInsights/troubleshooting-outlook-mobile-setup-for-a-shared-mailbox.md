# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>การแก้ไขปัญหาการตั้งค่าอุปกรณ์เคลื่อนที่ Outlook สำหรับกล่องจดหมายที่ใช้ร่วมกัน

คุณไม่สามารถเข้าถึงกล่องจดหมายที่ใช้ร่วมกันจากโปรแกรมประยุกต์ Outlook ที่ติดตั้งอยู่บนอุปกรณ์เคลื่อนที่ คุณต้องการคุณลักษณะนี้หรือไม่ ให้ทีมงานวิศวกรรมของ Outlook ทราบ [ลงคะแนนเสียงที่นี่](https://go.microsoft.com/fwlink/?linked=862116)!
  
คุณสามารถใช้กล่องจดหมายที่ใช้ร่วมกันบนอุปกรณ์เคลื่อนที่โดยเปิดเบราว์เซอร์ของคุณลงชื่อเข้าใช้ Office ๓๖๕จากนั้นไปที่ Outlook บนเว็บ สำหรับคำแนะนำโปรดดูที่[การเพิ่มกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บ](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207) จาก Outlook บนเว็บคุณจะสามารถเข้าถึงกล่องจดหมายที่ใช้ร่วมกัน
  
เรียนรู้เพิ่มเติม:[รับความช่วยเหลือในแอปสำหรับ Outlook สำหรับ iOS และ Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
