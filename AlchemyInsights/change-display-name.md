# <a name="change-your-display-name"></a>เปลี่ยนชื่อที่แสดงของคุณ
  
เมื่อต้องการเปลี่ยนชื่อที่แสดงของคุณใน Outlook ให้ใช้ขั้นตอนต่อไปนี้
  
1. ใน Outlook**File**\>ให้เลือก**การตั้งค่าบัญชี**ไฟล์**Account Settings**\>

2. เลือกบัญชีอีเมล แล้วเลือก**เปลี่ยน**

    > [!NOTE]
    > ถ้าคุณไม่เห็นตัวเลือกในการเปลี่ยนชื่อที่แสดงของคุณ ในกรณีนี้ คุณต้องติดต่อผู้ดูแลระบบของคุณเพื่อเปลี่ยนชื่อที่แสดงของคุณ ดู[เปลี่ยนชื่อผู้ใช้และที่อยู่อีเมล](https://docs.microsoft.com/office365/admin/add-users/change-a-user-name-and-email-address)
  
3. หลังจากที่คุณได้เปลี่ยนชื่อที่แสดงแล้ว ให้คลิก**ตกลง**เพื่อบันทึกการเปลี่ยนแปลงของคุณ การเปลี่ยนแปลงของคุณอาจใช้เวลาสองสามชั่วโมงในการสะท้อนทั่วทั้งองค์กรของคุณ

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การเปลี่ยนชื่อที่แสดงที่ผู้รับอีเมลเห็น](https://support.office.com/article/2b53331a-ba2a-4803-88dc-ac9fe376c8a9.aspx)
