# <a name="manage-who-can-create-microsoft-365-groups"></a>จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365

โดยค่าเริ่มต้น ทุกคนในองค์กรของคุณสามารถสร้างกลุ่ม O365 ได้
  
ถ้าคุณต้องการเปลี่ยนที่ คุณสามารถสร้างกลุ่มความปลอดภัย กับ PowerShell ที่จํากัดการสร้างกลุ่มเป็นชุดย่อยของผู้ใช้
  
ทําตามคําแนะนําทีละขั้นตอนที่นี่:[จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
