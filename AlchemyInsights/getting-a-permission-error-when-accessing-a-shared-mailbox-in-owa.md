# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>การรับข้อผิดพลาดสิทธิ์เมื่อเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บหรือไม่

ถ้าคุณได้รับข้อความแสดงข้อผิดพลาด**บางอย่างผิดพลาด คุณไม่มีสิทธิ์ในการเปิดกล่องจดหมายนี้** เมื่อคุณพยายามที่จะเปิดกล่องจดหมายที่ใช้ร่วมกันใน Outlook บนเว็บ ผู้ดูแลระบบองค์กรของคุณสามารถกําหนดสิทธิ์ในบัญชีของคุณให้กับกล่องจดหมายที่ใช้ร่วมกันได้[Create a shared mailbox](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox)
