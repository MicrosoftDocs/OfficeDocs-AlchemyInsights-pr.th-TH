# <a name="subscription-recurring-billing"></a>การเรียกเก็บเงินเป็นกิจวัตรการสมัครใช้งาน

ถ้าคุณต้องการทราบเมื่อมีการเรียกเก็บเงินการสมัครใช้งานรายเดือนหรือรายปีของคุณ (ถ้ามีการเปิดใช้งานการ **เรียกเก็บเงินเป็นกิจวัตร** ) หรือเมื่อหมดอายุ (ถ้าการ **เรียกเก็บเงินเป็นกิจวัตร** ถูกปิดใช้งาน) ให้ทำตามขั้นตอนต่อไปนี้:
  
1. ในศูนย์การจัดการให้ไปที่หน้าการ**เรียกเก็บเงิน** \> [ของผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)
2. บนแท็บ **ผลิตภัณฑ์** ให้ค้นหาการสมัครใช้งานที่คุณต้องการทราบข้อมูลเพิ่มเติม คอลัมน์ **สถานะการสมัคร** ใช้งานแสดงวันที่ต่ออายุหรือวันหมดอายุ
3. เมื่อต้องการทำการเปลี่ยนแปลงให้เลือก **การกระทำเพิ่มเติม** (สามจุด) จากนั้นเลือก **แก้ไขการเรียกเก็บเงิน**ที่เป็นกิจวัตร
4. ในบานหน้าต่าง**แก้ไขการเรียกเก็บเงินเป็นกิจวัตร**ให้เลือก**เปิด**,**เปิด, ต่ออายุ**หรือ**ปิด**
5. เลือก**บันทึก**

เมื่อต้องการเรียนรู้เพิ่มเติมให้ดูที่[เปิดหรือปิดการเรียกเก็บเงินที่เป็นกิจวัตร](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription)
