# <a name="set-up-spf"></a>ตั้งค่า SPF

อัปเดตระเบียน DNS ของคุณเพื่อให้คุณสามารถใช้กรอบนโยบายผู้ส่ง (SPF) กับโดเมนแบบกำหนดเองของคุณเพื่อตรวจสอบความถูกต้องของอีเมลขาออกที่ส่งจากโดเมนแบบกำหนดเองของคุณ การทำเช่นนี้จะช่วยป้องกันไม่ให้ผู้อื่นจาก "การหลอกลวง" จดหมายราวกับว่าถูกส่งจากโดเมนของคุณ
  
- [ตั้งค่า SPF เพื่อช่วยป้องกันการปลอมแปลง](https://docs.microsoft.com/microsoft-365/security/office-365-security/set-up-spf-in-office-365-to-help-prevent-spoofing)

- [วิธีใช้การตั้งค่าระเบียน DNS ทั้งหมด](https://docs.microsoft.com/microsoft-365/admin/get-help-with-domains/create-dns-records-at-any-dns-hosting-provider)
