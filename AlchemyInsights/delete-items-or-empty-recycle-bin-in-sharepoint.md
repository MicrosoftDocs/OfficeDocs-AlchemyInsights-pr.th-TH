# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>ลบรายการหรือล้างถังรีไซเคิลของไซต์ SharePoint 

เมื่อคุณลบรายการออกจากไลบรารีเอกสารหรือรายการ SharePoint รายการนั้นจะไม่ถูกเอาออกจาก SharePoint ทันที รายการที่ถูกลบไปลงในถังรีไซเคิลของไซต์เป็นระยะเวลาหนึ่ง คุณสามารถคืนค่ารายการที่คุณลบไปไว้ในตําแหน่งที่ตั้งเดิม หรือนํารายการเหล่านั้นออกเพื่อเพิ่มพื้นที่ว่างในการจัดเก็บ

[ลบรายการหรือล้างถังรีไซเคิลของไซต์ SharePoint](https://support.office.com/article/2e713599-d13e-40d6-96dc-66f0a366f74e)
