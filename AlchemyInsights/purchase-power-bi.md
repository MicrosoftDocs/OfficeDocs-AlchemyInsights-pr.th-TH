# <a name="purchase-power-bi"></a>ซื้อ Power BI

คุณสามารถสั่งซื้อ Power BI ได้ใน**ศูนย์การจัดการ Microsoft 365**

1. ไปที่หน้า**[บริการการซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)>การเรียกเก็บเงิน**

2. ใช้กล่องค้นหาเพื่อค้นหาใน "Power BI"

3. เลือกแผน Power BI ที่คุณต้องการซื้อ

4. เลือกจํานวนผู้ใช้ที่จะใช้ Power BI

เมื่อต้องการมอบหมายสิทธิ์การใช้งานให้กับผู้ใช้ ให้ดูที่[มอบหมายสิทธิ์การใช้งานให้กับผู้ใช้](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide)

เมื่อต้องการซื้อเพิ่มเติมหรือเอาสิทธิ์การใช้งานออก ให้ดูที่[เพิ่ม/เอาสิทธิ์การใช้งานสําหรับการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)ออก
