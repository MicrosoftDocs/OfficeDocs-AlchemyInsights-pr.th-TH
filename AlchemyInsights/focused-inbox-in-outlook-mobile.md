# <a name="focused-inbox-for-outlook-mobile"></a>กล่องจดหมายเข้าที่โฟกัสสำหรับ Outlook mobile

[กล่องจดหมายเข้าที่โฟกัสคืออะไร](https://go.microsoft.com/fwlink/p/?linkid=2001915&amp;clcid=0x409)
  
[ฉันจะแก้ไขกฎที่ย้ายอีเมลของฉันเข้าและออกจากกล่องจดหมายเข้าที่โฟกัสได้อย่างไร](https://go.microsoft.com/fwlink/p/?linkid=2002210&amp;clcid=0x409)
  
[ฉันจะปิดกล่องจดหมายเข้าที่โฟกัสหรือเปลี่ยนการซิงค์การแจ้งเตือนและการแจ้งให้ทราบของฉันในกล่องจดหมายเข้าที่โฟกัสได้อย่างไร](https://go.microsoft.com/fwlink/p/?linkid=2001916&amp;clcid=0x409)
  
[ฉันจะปิดกล่องจดหมายเข้าที่โฟกัสหรือเปลี่ยนแปลงการซิงค์การแจ้งเตือนไปยังกล่องจดหมายเข้าที่โฟกัสได้อย่างไร](https://go.microsoft.com/fwlink/p/?linkid=2001917&amp;clcid=0x409)
  
[ฉันจะสลับระหว่างโฟกัสและอื่นๆได้อย่างไร](https://go.microsoft.com/fwlink/p/?linkid=2002211&amp;clcid=0x409)
  
เรียนรู้เพิ่มเติม: [รับวิธีใช้ในแอปสำหรับ Outlook สำหรับ iOS และ Android](https://go.microsoft.com/fwlink/p/?linkid=2001918&amp;clcid=0x409)
