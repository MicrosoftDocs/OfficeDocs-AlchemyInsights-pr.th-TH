# <a name="export-chat-history-in-teams"></a>ส่งออกประวัติการแชทในทีม

- การส่งออกการสำรองการพิมพ์และการเก็บถาวรประวัติการสนทนาจะไม่พร้อมใช้งานในขณะนี้สำหรับผู้ใช้จากทีม Microsoft โดยตรง โปรดลงคะแนนเสียงในความสามารถนี้ใน[UserVoice](https://microsoftteams.uservoice.com/forums/555103-public/suggestions/16982542-backup-export-printing-archive-options?page=2&per_page=20)
- ในฐานะผู้ดูแลระบบหรือผู้จัดการ eDiscovery คุณสามารถสร้างการ [ค้นหาเนื้อหา](https://docs.microsoft.com/microsoft-365/compliance/content-search?view=o365-worldwide)  ในศูนย์การปฏิบัติตามนโยบายของ Microsoft ๓๖๕เพื่อค้นหาจากนั้นส่งออกการสนทนาข้อความโต้ตอบแบบทันทีในองค์กรของคุณได้ สำหรับข้อมูลเพิ่มเติมโปรดดูที่[ส่งออกผลลัพธ์การค้นหาเนื้อหา](https://docs.microsoft.com/microsoft-365/compliance/export-search-results?view=o365-worldwide)
