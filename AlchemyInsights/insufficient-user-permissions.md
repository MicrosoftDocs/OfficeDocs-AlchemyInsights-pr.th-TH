# <a name="insufficient-user-permissions"></a>สิทธิ์ของผู้ใช้ไม่เพียงพอ

ปัญหานี้เกิดขึ้นเมื่อไม่ตรงตาม[ข้อกำหนดเบื้องต้นที่จำเป็น](https://docs.microsoft.com/configmgr/tenant-attach/device-sync-actions#prerequisites) โปรดตรวจสอบให้แน่ใจว่าผู้ใช้ได้แจ้งสิทธิ์ของ **ทรัพยากร** ภายใต้คลาสของวัตถุ **คอลเลกชัน** ในตัวจัดการการกำหนดค่า
