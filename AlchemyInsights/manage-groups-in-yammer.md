# <a name="manage-groups-in-yammer"></a>จัดการกลุ่มใน Yammer

- **สิทธิ์การสร้างกลุ่ม** Yammer ไม่สามารถควบคุมผู้ที่สามารถสร้างกลุ่มได้ อย่างไรก็ตาม คุณสามารถจัดการการสร้างกลุ่มสําหรับกลุ่ม Microsoft 365 ที่เชื่อมต่อได้ ดู ข้อมูลเพิ่มเติมที่[จัดการผู้ที่สามารถสร้างกลุ่ม Microsoft 365](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)

- **สมาชิกกลุ่มแบบไดนามิก** โปรดดูที่[สร้างกลุ่มแบบไดนามิกใน Yammer](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group)สําหรับข้อมูลเกี่ยวกับการสร้างสมาชิกแบบไดนามิกสําหรับ Yammer ที่เชื่อมต่อกับกลุ่ม Microsoft 365 ที่เชื่อมต่อ

- **กําลังกู้คืนกลุ่มที่ถูกลบ** การคืนค่ากลุ่ม Yammer เป็นไปได้ผ่านทาง[คืนค่ากลุ่ม Microsoft 365 ที่ถูกลบ](https://docs.microsoft.com/office365/admin/create-groups/restore-deleted-group)กลุ่มไม่มีการเชื่อมต่อกลุ่มไม่สามารถคืนค่าหลังจากการลบ

สําหรับข้อมูลเพิ่มเติมเกี่ยวกับการจัดการกลุ่ม ให้ดูที่[จัดการกลุ่มใน Yammer](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2) สําหรับข้อมูลเกี่ยวกับกลุ่ม Microsoft 365 ที่เชื่อมต่อ ให้ดูที่[Yammer และกลุ่ม Microsoft 365](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups)
