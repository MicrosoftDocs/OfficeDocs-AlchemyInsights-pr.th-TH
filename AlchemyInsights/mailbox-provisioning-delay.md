# <a name="your-users-mailbox-is-being-created"></a>กําลังสร้างกล่องจดหมายของผู้ใช้ของคุณ

อาจใช้เวลาถึงหนึ่งชั่วโมงก่อนที่กล่องจดหมายใหม่จะพร้อมใช้งาน
  
ถ้ากล่องจดหมายของผู้ใช้ได้รับการ "เตรียม" มานานกว่าหนึ่งชั่วโมง คุณสามารถลองลบ แล้วสร้างผู้ใช้ใหม่ หรือใช้ตัวเลือก "ให้เราเรียกคุณ" เพื่อพูดคุยกับวิศวกรฝ่ายสนับสนุน
