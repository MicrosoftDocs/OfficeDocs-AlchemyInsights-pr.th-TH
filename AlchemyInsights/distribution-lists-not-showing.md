# <a name="distribution-group-not-showing-in-groups-list"></a>กลุ่มการแจกจ่ายไม่แสดงในรายการกลุ่ม

อาจใช้เวลาถึง๖๐นาทีก่อนที่รายการการแจกจ่ายใหม่ของคุณจะปรากฏในศูนย์กลางการดูแล Exchange
  
ถ้าคุณต้องการเพิ่มสมาชิกหรือทำการแก้ไขอื่นๆในตอนนี้โปรดใช้คุณลักษณะ Groups ใน[ศูนย์ดูแล Exchange](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)
