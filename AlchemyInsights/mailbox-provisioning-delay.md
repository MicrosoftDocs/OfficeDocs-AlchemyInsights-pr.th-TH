# <a name="your-users-mailbox-is-being-created"></a>กำลังสร้างกล่องจดหมายของผู้ใช้

ซึ่งอาจใช้เวลาถึงหนึ่งชั่วโมงก่อนที่กล่องจดหมายใหม่จะพร้อมใช้งาน
  
ถ้ากล่องจดหมายของผู้ใช้ได้รับ "เตรียม" สำหรับมากกว่าหนึ่งชั่วโมง คุณสามารถลองลบ และสร้างผู้ใช้ใหม่ หรือใช้ตัวเลือก "แจ้งให้เราเรียกคุณ" เพื่อพูดคุยกับวิศวกรฝ่ายสนับสนุน
