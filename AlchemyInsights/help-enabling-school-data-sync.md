# <a name="help-enabling-school-data-sync"></a>ช่วยเปิดใช้งานการซิงค์ข้อมูลของโรงเรียน

ต้องการความช่วยเหลือในการเปิดใช้งานการซิงค์ข้อมูลของโรงเรียนหรือไม่ คุณทราบหรือไม่ว่าคุณสามารถมีสิทธิ์ได้รับความช่วยเหลือจากโปรแกรม Microsoft 365 FastTrack

โปรดไปที่[ศูนย์ FastTrack](https://www.microsoft.com/fasttrack)เพื่อส่งคําขอขอความช่วยเหลือในการเปิดใช้การซิงค์ข้อมูลของโรงเรียนสําหรับองค์กรของคุณ
