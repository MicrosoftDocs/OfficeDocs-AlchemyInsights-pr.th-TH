# <a name="stop-messages-from-going-into-your-junk-email-folder"></a>หยุดข้อความไม่ให้เข้าสู่โฟลเดอร์อีเมลขยะ

มีบางครั้งที่คุณไม่ต้องการส่งข้อความจากบุคคลหรือโดเมนที่ระบุให้สิ้นสุดในโฟลเดอร์อีเมลขยะในOutlook.com ข้อความจากที่อยู่หรือโดเมนในรายชื่อผู้ส่งที่ปลอดภัยจะไม่ถูกย้ายไปยังโฟลเดอร์อีเมลขยะของคุณ

1. เปิด[การตั้งค่าผู้ส่งที่ปลอดภัย](https://go.microsoft.com/fwlink/?linkid=2035804)
2. ภายใต้**ผู้ส่งและโดเมนที่ปลอดภัย**ให้ป้อนที่อยู่อีเมลหรือโดเมนที่คุณต้องการเพิ่ม แล้วเลือก**เพิ่ม**
3. เมื่อต้องการเพิ่มรายชื่อผู้รับจดหมายไปยังผู้ส่งที่ปลอดภัย ของคุณ ให้ใส่รายชื่อผู้รับจดหมายภายใต้**รายชื่อผู้รับจดหมายที่ปลอดภัย**แล้วเลือก**เพิ่ม**
4. เลือก**บันทึก**

อ่านเพิ่มเติมที่[บล็อกหรือยกเลิกการบล็อกผู้ส่งใน Outlook.com](https://support.office.com/article/afba1c94-77bb-4f50-8b85-057cf52f4d5e?wt.mc_id=Office_Outlook_com_Alchemy)
