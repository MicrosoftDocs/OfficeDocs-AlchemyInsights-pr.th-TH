# <a name="versioning-in-sharepoint-and-onedrive"></a>การกําหนดรุ่นใน SharePoint และ OneDrive 


เมื่อเปิดใช้งานการกําหนดรุ่นในรายการหรือไลบรารี SharePoint คุณสามารถจัดเก็บ ติดตาม และคืนค่ารายการในรายการและแฟ้มในไลบรารีเมื่อใดก็ตามที่มีการเปลี่ยนแปลง การกําหนดเวอร์ชันรวมกับการตั้งค่าอื่น ๆ เช่นการชําระเงินจะช่วยให้คุณควบคุมเนื้อหาจํานวนมากที่มีการโพสต์บนเว็บไซต์ของคุณและสามารถให้ค่าที่แท้จริงถ้าคุณเคยมีความจําเป็นที่จะมองหรือเรียกคืนรายการหรือไฟล์รุ่นเก่า

สําหรับข้อมูลเพิ่มเติมเกี่ยวกับการกําหนดรุ่นโปรดไปที่บทความด้านล่าง

- [การกําหนดเวอร์ชันทํางานอย่างไรในรายการหรือไลบรารี SharePoint](https://support.office.com/article/how-does-versioning-work-in-a-sharepoint-list-or-library-0f6cd105-974f-44a4-aadb-43ac5bdfd247)

- [การเปิดใช้งานและกําหนดค่าการกําหนดรุ่นสําหรับรายการหรือไลบรารี](https://support.office.com/article/enable-and-configure-versioning-for-a-list-or-library-1555d642-23ee-446a-990a-bcab618c7a37?ocmsassetID=HA102772148&amp;CTT=3&amp;CorrelationId=52441bb1-a619-4375-89d5-19d28769890f)

- [วิธีดูประวัติเวอร์ชัน](https://support.office.com/article/View-the-version-history-of-an-item-or-file-in-a-list-or-library-53262060-5092-424D-A50B-C798B0EC32B1)

- [การคืนค่ารุ่นก่อนหน้าของไฟล์ใน OneDrive](https://support.office.com/article/restore-a-previous-version-of-a-file-in-onedrive-159cad6d-d76e-4981-88ef-de6e96c93893)

- [ดูไฟล์ Office เวอร์ชันก่อนหน้า](https://support.office.com/article/view-previous-versions-of-office-files-5c1e076f-a9c9-41b8-8ace-f77b9642e2c2)

- [ขีดจํากัดการกําหนดรุ่น](https://docs.microsoft.com/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits)

>[!Note] 
>ถ้าคุณเป็นลูกค้า Microsoft 365 ขณะนี้การกําหนดรุ่นจะเปิดขึ้นตามค่าเริ่มต้นเมื่อคุณสร้าง OneDrive ใหม่สําหรับไลบรารีธุรกิจ และจะเป็นการบันทึกเอกสารรุ่น 500 ครั้งล่าสุดโดยอัตโนมัติ วิธีนี้จะช่วยให้คุณป้องกันการสูญเสียเอกสารหรือข้อมูลที่สําคัญ ถ้าคุณมีไลบรารีที่มีอยู่บนไซต์ OneDrive for Business หรือบนไซต์ทีมของคุณซึ่งไม่มีการเปิดใช้งานการกําหนดเวอร์ชัน
