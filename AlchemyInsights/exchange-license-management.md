# <a name="exchange-license-management"></a>การจัดการใบอนุญาตแลกเปลี่ยน

การจัดการสิทธิ์การใช้งาน Exchange สามารถทําได้ใน**ศูนย์การจัดการ Microsoft 365**

- [เปรียบเทียบตัวเลือกการแลกเปลี่ยน](https://www.microsoft.com/microsoft-365/exchange/compare-microsoft-exchange-online-plans)เพื่อเรียนรู้เกี่ยวกับความแตกต่างระหว่างแผน**Exchange Online 1, 2 และค่าจ้างพิเศษของธุรกิจ**
- หากต้องการ**ซื้อ**Exchange ให้ไปที่**การเรียกเก็บเงิน >[บริการซื้อ](https://go.microsoft.com/fwlink/p/?linkid=868433)** ใช้กล่องค้นหาเพื่อค้นหาใน "แลกเปลี่ยน"; เลือกแผนและจํานวนผู้ใช้ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ซื้อการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/buy-another-subscription?view=o365-worldwide)
- เมื่อต้องการ**สลับแผน**ให้ดูที่[สลับแผน Microsoft 365 สําหรับธุรกิจด้วยตนเอง](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/switch-plans-manually?view=o365-worldwide)
- เมื่อต้องการ**มอบหมาย**สิทธิ์การใช้งาน Exchange ให้ไปที่**การเรียกเก็บเงิน >[สิทธิ์การใช้งาน](https://go.microsoft.com/fwlink/p/?linkid=842264)** ให้เลือกการสมัครใช้งาน**Assign licenses** สําหรับข้อมูลเพิ่มเติม โปรดดูที่[มอบหมายสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/admin/manage/assign-licenses-to-users?view=o365-worldwide)
- เมื่อต้องการ**เพิ่ม/เอา**สิทธิ์การใช้งาน Exchange ออก ให้ไปที่**การเรียกเก็บเงิน > บริการ[&](https://go.microsoft.com/fwlink/p/?linkid=842054)** และคลิก**เพิ่ม/เอาสิทธิ์การใช้งานออก** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[การเพิ่ม/เอาสิทธิ์การใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)ออก
- เมื่อต้องการ**ยกเลิก**สิทธิ์การใช้งาน Exchange ให้ไปที่**บริการ[&>](https://go.microsoft.com/fwlink/p/?linkid=842054)การเรียกเก็บเงิน**ให้เลือก**การดําเนินการ&การตั้งค่า**แล้วคลิก**ยกเลิกการสมัครใช้งาน** สําหรับข้อมูลเพิ่มเติม ให้ดูที่[ยกเลิกการสมัครใช้งานของคุณ](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/cancel-your-subscription)

สําหรับข้อมูลเพิ่มเติม ให้ดูที่[จัดการสิทธิ์การใช้งานการสมัครใช้งาน](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses?view=o365-worldwide#add-or-remove-licenses-for-your-business-subscription)
