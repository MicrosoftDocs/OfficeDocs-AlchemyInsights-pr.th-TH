# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>รายงานเกี่ยวกับการแชร์ไฟล์และโฟลเดอร์ในไซต์ SharePoint

คุณสามารถสร้างไฟล์ CSV ของไฟล์ผู้ใช้สิทธิ์และลิงก์ที่ไม่ซ้ำกันได้บนไซต์ SharePoint หรือไลบรารี OneDrive ที่กำหนด การทำเช่นนี้สามารถช่วยให้คุณเข้าใจวิธีการใช้งานการแชร์ได้และไม่ว่าจะมีไฟล์หรือโฟลเดอร์ใดบ้างที่ถูกแชร์กับผู้เยี่ยมชม สำหรับข้อมูลเพิ่มเติมโปรดดู[ที่รายงานเกี่ยวกับการแชร์ไฟล์และโฟลเดอร์ในไซต์ SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports)
