# <a name="too-many-licenses"></a>มีใบอนุญาตมากเกินไป?

หากคุณคิดว่าคุณกําลังถูกเรียกเก็บเงินสําหรับใบอนุญาตมากเกินไปนี้มักจะเกิดจากการซื้อใบอนุญาตมากขึ้นแล้วคุณต้อง
  
**ลดจํานวนสิทธิ์การใช้งานที่ซื้อ**
  
1. ในศูนย์การจัดการ ให้ไปที่หน้า**การเรียกเก็บเงิน** \> **[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการเอาสิทธิ์การใช้งานออก แล้วเลือก**เพิ่ม/เอาสิทธิ์การใช้งานออก**

3. ในกล่อง**สิทธิ์การใช้งานทั้งหมด**ให้ใส่จํานวนสิทธิ์การใช้งานทั้งหมดที่คุณต้องการสําหรับการสมัครใช้งานนี้ แล้วเลือก**ส่งการเปลี่ยนแปลง** ตัวอย่างเช่น ถ้าคุณมีสิทธิ์การใช้งาน 110 ใบ และคุณต้องการนําสิทธิ์การใช้งานออก 5 ใบ ให้ป้อน 105

เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[เอาสิทธิ์การใช้งานออกจากการสมัครใช้งาน Microsoft 365 สําหรับธุรกิจของคุณ](https://docs.microsoft.com/microsoft-365/commerce/licenses/buy-licenses)
