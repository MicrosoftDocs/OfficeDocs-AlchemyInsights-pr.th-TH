# <a name="renewing-your-subscription"></a>การต่ออายุการสมัครใช้งานของคุณ

หากคุณชําระเงินด้วยบัตรเครดิตหรือบัญชีธนาคาร การเรียกเก็บเงินตามกําหนดเวลาจะถูกเปิดใช้งานตามค่าเริ่มต้น เมื่อการเรียกเก็บเงินตามระยะเวลาการชําระเงินถูกเปิดใช้งาน คุณสามารถปิดหรือเปิดอีกครั้งในศูนย์การจัดการถ้าการสมัครใช้งานของคุณทํางานอยู่

หากคุณชําระเงินค่าสมัครด้วยหมายเลขผลิตภัณฑ์การสมัครใช้งานของคุณจะหมดอายุเมื่อสิ้นสุดระยะเวลาการสมัคร คุณสามารถเปิดการเรียกเก็บเงินตามระยะเวลาและให้ข้อมูลบัตรเครดิตหรือบัญชีธนาคารเพื่อให้การสมัครของคุณจะถูกเรียกเก็บเงินต่อไปทุกวันที่คุณสมัครใช้บริการ

วิธีเปิดการเรียกเก็บเงินตามระยะเวลา: 

1. ไปที่**การเรียกเก็บเงิน** > **[ผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**
2. เลือกการสมัครใช้งานที่คุณต้องการจัดการการเรียกเก็บเงินที่เกิดประจํา
3. ภายใต้**การเรียกเก็บเงิน**ให้เลือก**แก้ไข**แล้วทําการเปลี่ยนแปลงในหน้าต่าง**การตั้งค่าการเรียกเก็บเงินที่เกิดประจํา**
