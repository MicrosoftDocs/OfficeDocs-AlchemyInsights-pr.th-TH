# <a name="you-need-permission-to-access-this-site"></a>คุณต้องมีสิทธิ์ในการเข้าถึงไซต์นี้

มีสถานการณ์สมมติมากมายที่สามารถพร้อมท์ข้อความเหล่านี้หนึ่ง บ่อยที่สุดสิทธิ์สำหรับผู้ใช้หรือผู้ดูแลระบบถูกกำหนดค่าอย่างไม่ถูกต้องหรือไม่มีการกำหนดค่าที่ทั้งหมด 

สำหรับข้อมูลเพิ่มเติมโปรดดู[ที่ "การเข้าถึงถูกปฏิเสธ" คุณต้องมีสิทธิ์ในการเข้าถึงไซต์นี้ "หรือ" ผู้ใช้ไม่พบในไดเรกทอรี "ข้อผิดพลาดใน SharePoint แบบออนไลน์และ OneDrive สำหรับธุรกิจ](https://docs.microsoft.com/sharepoint/support/administration/access-denied-or-need-permission-error-sharepoint-online-or-onedrive-for-business)
