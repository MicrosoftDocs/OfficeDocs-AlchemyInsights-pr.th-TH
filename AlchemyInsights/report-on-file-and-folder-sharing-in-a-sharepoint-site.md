# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>รายงานเกี่ยวกับการใช้แฟ้มและโฟลเดอร์ร่วมกันในไซต์ SharePoint

คุณสามารถสร้างไฟล์ CSV ของไฟล์ ผู้ใช้ สิทธิ์ และลิงก์ที่ไม่ซ้ํากันทุกไฟล์บนไซต์ SharePoint หรือไลบรารี OneDrive ที่กําหนดได้ วิธีนี้จะช่วยให้คุณเข้าใจว่ามีการใช้การแชร์และว่าแฟ้มหรือโฟลเดอร์ใดๆ ถูกแชร์กับแขกหรือไม่ สําหรับข้อมูลเพิ่มเติม โปรดดู[รายงานเกี่ยวกับการใช้แฟ้มและโฟลเดอร์ร่วมกันในไซต์ SharePoint](https://docs.microsoft.com/sharepoint/sharing-reports)
