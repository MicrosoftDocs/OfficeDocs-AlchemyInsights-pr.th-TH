# <a name="merge-microsoft-accounts"></a>ผสานบัญชีไมโครซอฟท์

- คุณไม่สามารถผสานหรือเชื่อมโยงบัญชี Microsoft และบัญชี Office 365 ทํางานหรือโรงเรียนได้เนื่องจากทั้งสองบัญชีอยู่ในบริการต่างๆ แต่สามารถใช้บัญชีแบบเคียงข้างกันได้
- คุณไม่สามารถผสานหรือเชื่อมโยงบัญชี Microsoft (แม้กับที่อยู่อีเมลที่แยกต่างหาก) หรือบัญชีที่ไม่ใช่ของ Microsoft (เช่น บัญชี Gmail) กับบัญชี Microsoft ที่มีอยู่
