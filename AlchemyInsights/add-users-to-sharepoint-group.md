# <a name="add-users-to-a-sharepoint-group"></a>การเพิ่มผู้ใช้ลงในกลุ่ม SharePoint

กลุ่ม SharePoint คือคอลเลกชันของผู้ใช้ที่ทุกคนมีชุดของสิทธิ์เดียวกันกับไซต์และเนื้อหา แทนที่จะกำหนดสิทธิ์หนึ่งคนในแต่ละครั้งคุณสามารถใช้กลุ่มเพื่อมอบหมายระดับสิทธิการได้รับอนุญาตเดียวกันให้กับคนจำนวนมากพร้อมกันได้

- [เพิ่มผู้ใช้ลงในกลุ่ม](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [เพิ่มหรือเอาสมาชิกออกจากกลุ่ม Office ๓๖๕โดยใช้ศูนย์ดูแล](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
