# <a name="use-your-browser-to-check-spelling"></a>ใช้เบราว์เซอร์ของคุณเพื่อตรวจสอบการสะกด

แม้ว่า Outlook.com จะไม่มีฟังก์ชันการตรวจสอบการสะกดเว็บเบราว์เซอร์จำนวนมากจะตรวจสอบการสะกดของคุณโดยอัตโนมัติรวมถึง Microsoft Edge, Internet Explorer 10 และเวอร์ชันที่ใหม่กว่าและเวอร์ชันปัจจุบันของ Firefox, Chrome และ Safari ตรวจสอบตัวเลือกสำหรับเว็บเบราว์เซอร์ของคุณเพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีการตรวจสอบการสะกด
