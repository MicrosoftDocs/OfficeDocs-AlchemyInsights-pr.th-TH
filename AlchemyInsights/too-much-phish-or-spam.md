# <a name="are-you-receiving-too-much-phish-or-spam-in-your-mailbox"></a>คุณได้รับ phish มากเกินไป หรือสแปมในกล่องจดหมายของคุณหรือไม่

ฐานะเป็นผู้ดูแล Microsoft 365 วิธีดีที่สุดเพื่อกำหนดว่าเหตุใดข้อความฟิชชิ่งหรือสแปมถูกส่งเป็นการ ส่งตัวอย่างไปยัง Microsoft ใน[Explorer ส่ง](https://protection.office.com/reportsubmission) คุณจะได้รับผลป้อนกลับในทันทีที่เกี่ยวข้องกับการกรองข้อความและ เหตุผลที่ทำให้มีปัญหาต่าง ๆ

เมื่อต้องการให้ easer คุณสามารถอนุญาตให้ผู้ใช้ของคุณเพื่อรายงานโดยตรงไปยัง Microsoft โดยที่[ข้อความของรายงานใน](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview)
