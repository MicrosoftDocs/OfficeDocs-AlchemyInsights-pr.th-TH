# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>เปิดและใช้กล่องจดหมายที่แชร์ใน Outlook บนเว็บ

กล่องจดหมายที่ใช้ร่วมกันที่คุณมีสิทธิ์เข้าถึงจะไม่แสดงใน Outlook บนเว็บโดยอัตโนมัติ (เดิมเรียกว่า Outlook Web App)

คุณสามารถแสดงกล่องจดหมายที่แชร์และกล่องจดหมายหลักของคุณร่วมกัน หรือเปิดกล่องจดหมายที่ใช้ร่วมกันในหน้าต่างเบราว์เซอร์ที่แยกต่างหาก เมื่อต้องการรับการแจ้งเตือนข้อความใหม่สําหรับกล่องจดหมายที่แชร์ สําหรับข้อมูลเพิ่มเติม ให้ดูที่[เปิดและใช้กล่องจดหมายที่ใช้ร่วมกันใน Outlook Web App](https://support.office.com/en-us/article/Add-a-shared-mailbox-to-Outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207)
