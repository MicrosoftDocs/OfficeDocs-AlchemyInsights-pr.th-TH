# <a name="create-a-folder-in-outlookcom"></a>สร้างโฟลเดอร์ในOutlook.com

1. ที่ด้านล่างของบานหน้าต่างโฟลเดอร์ ให้เลือก**โฟลเดอร์ใหม่** คุณอาจต้องเลื่อนลงในบานหน้าต่างโฟลเดอร์เพื่อดูปุ่ม**โฟลเดอร์ใหม่**
2. ในกล่องข้อความ ให้ใส่ชื่อสําหรับโฟลเดอร์แล้วกด Enter

เมื่อต้องการเรียนรู้เพิ่มเติม ให้ดูที่[การสร้างโฟลเดอร์ใน Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)
