# <a name="deploy-microsoft-365-apps"></a>ปรับใช้แอป Microsoft 365

สําหรับการปรับใช้โปรแกรมประยุกต์ Microsoft 365 สําหรับผู้ใช้ปลายทางหลายผ่านเครื่องมือการปรับใช้ Office, Microsoft ปลายทางจัดการการตั้งค่าคอนฟิก เป็นส่วนหนึ่งของรูปดิสก์ระบบปฏิบัติการ หรือในสถานการณ์คอมพิวเตอร์ที่ใช้ร่วมกันโดยใช้ Windows Server กับบริการเดสก์ท็อประยะไกล (RDS) หรือบริการเทอร์มินัล พิจารณาใช้ขั้นตอนตามคําแนะนําขั้นตอนใน**Microsoft 365 Apps สําหรับองค์กรแนะนําการปรับใช้**
