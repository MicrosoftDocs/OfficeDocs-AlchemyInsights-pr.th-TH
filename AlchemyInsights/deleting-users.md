# <a name="deleting-users"></a>การลบผู้ใช้

ถ้าคุณต้องการบล็อกการเข้าถึงของผู้ใช้ในทันทีและจำเป็นต้องเก็บข้อมูลของพวกเขาโปรดดู[ที่บล็อกการเข้าถึงข้อมูลของพนักงานเก่าไปยัง Microsoft ๓๖๕](https://docs.microsoft.com/microsoft-365/admin/add-users/remove-former-employee#block-a-former-employees-access-to-microsoft-365-data)
  
ถ้าคุณเพียงแค่ต้องการลบผู้ใช้และไม่เกี่ยวข้องกับข้อมูลใดๆของพวกเขาโปรดดู[ลบผู้ใช้ออกจากองค์กรของคุณ](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user) การทำเช่นนี้จะเพิ่มสิทธิ์การใช้งานของพวกเขาทันที
