# <a name="workflow-troubleshooting"></a>การแก้ไขปัญหาลำดับงาน

สำหรับข้อมูลเกี่ยวกับการดีบักเวิร์กโฟลว์ SharePoint ตรวจทานต่อไปนี้:
- การดีบักเวิร์กโฟลว์คล้ายกับรุ่นก่อนหน้านี้  สำหรับข้อมูลเพิ่มเติมให้ดูการ[ดีบักเวิร์กโฟลว์ SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows)
- สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการแก้ไขปัญหาเบื้องต้นให้ดูที่การ[แก้ไขปัญหาข้อผิดพลาดการตรวจสอบลำดับงานของ SharePoint Server](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio)
