<span data-ttu-id="282cd-103">มีตัวเลือก 4 สำหรับผู้ใช้ภายนอกเข้าถึงเครือข่ายของคุณ</span><span class="sxs-lookup"><span data-stu-id="282cd-103">There are 4 options for external users to access your network.</span></span>
  
1. <span data-ttu-id="282cd-104">**ผู้ใช้ที่เป็นแขก**- Yammer Admins สามารถเชิญผู้ติดต่อจากโดเมนภายนอกเป็นผู้ใช้บนเครือข่ายภายในบ้านได้</span><span class="sxs-lookup"><span data-stu-id="282cd-104">**Guest users** - Yammer Admins can invite contacts from external domains as users on the home network.</span></span> <span data-ttu-id="282cd-105">ผู้ใช้เหล่านี้จะเห็น บริษัททั้งหมดตัวดึงข้อมูล และสามารถเข้าร่วมกลุ่มอื่น ๆ ในเครือข่ายเช่นกัน</span><span class="sxs-lookup"><span data-stu-id="282cd-105">These users will see the All Company feed and can join other groups in the networks as well.</span></span>

2. <span data-ttu-id="282cd-106">**ผู้ใช้ภายนอกในกลุ่มแบบภายนอก**- กลุ่มภายนอกที่อนุญาตให้ผู้ใช้ภายนอก</span><span class="sxs-lookup"><span data-stu-id="282cd-106">**External users in an external group** - External groups allow external users.</span></span> <span data-ttu-id="282cd-107">ขอบเขตของการเข้าถึงของผู้ใช้ภายนอกไม่ถูกจำกัดให้กับกลุ่มภายนอก</span><span class="sxs-lookup"><span data-stu-id="282cd-107">The scope of an external user's access is limited to the external group.</span></span> <span data-ttu-id="282cd-108">มีกลุ่มภายนอก คุณสามารถเชิญผู้ติดต่อจากเครือข่าย Yammer ที่อนุญาตให้ผู้ใช้มีส่วนร่วมในการสนทนาภายนอกเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="282cd-108">With external groups, you can only invite contacts from Yammer networks that allow users to participate in external conversations.</span></span>

3. <span data-ttu-id="282cd-109">**ผู้ใช้ภายนอกในการสนทนา Yammer ในกลุ่มแบบภายใน**- ตัวเลือกนี้อนุญาตให้ผู้เข้าร่วมภายนอกที่จะเพิ่มการสนทนาที่เฉพาะเจาะจง</span><span class="sxs-lookup"><span data-stu-id="282cd-109">**External users in a Yammer conversation in an internal group** - This option allows external participants to be added to a particular conversation.</span></span> <span data-ttu-id="282cd-110">ผู้เข้าร่วมภายนอกสามารถเข้าร่วมในการสนทนาที่พวกเขาได้รับอย่างชัดเจนเชิญเพื่อ และสิ่งที่แนบใด ๆ ในการสนทนานั้นเท่านั้น</span><span class="sxs-lookup"><span data-stu-id="282cd-110">External participants can only participate in conversations they have been explicitly invited to and to any attachments in that conversation.</span></span> <span data-ttu-id="282cd-111">จะเข้าถึงการสนทนาเหล่านี้ได้ผ่านทางกล่องขาเข้าของ Yammer (บนเครือข่ายของตนเอง)</span><span class="sxs-lookup"><span data-stu-id="282cd-111">They access these conversations via their Yammer inbox (on their network).</span></span> <span data-ttu-id="282cd-112">ไม่สามารถเข้าถึงส่วนเหลือของเครือข่ายของคุณมี</span><span class="sxs-lookup"><span data-stu-id="282cd-112">They have no access to the rest of your network.</span></span>

4. <span data-ttu-id="282cd-113">**เครือข่ายภายนอก**- ตัวเลือกนี้ช่วยให้คุณสามารถเชิญผู้ใช้ภายนอกไปยังเครือข่ายภายนอกที่ตรวจสอบ Admins สามารถสร้างในเครือข่ายของคุณ</span><span class="sxs-lookup"><span data-stu-id="282cd-113">**External Networks** - This option allows you to invite external users to an external network that verified Admins can create in your network.</span></span> <span data-ttu-id="282cd-114">ตัวเลือกนี้จะเหมาะสมเหมาะสมที่สุดสำหรับการจัดการโฮสต์สำหรับชุมชนภายนอก</span><span class="sxs-lookup"><span data-stu-id="282cd-114">This option is best suited for hosting external communities.</span></span>
