# <a name="prepaid-microsoft-365-renewal"></a>การต่ออายุของ Microsoft ๓๖๕แบบชำระเงินล่วงหน้า

1. ในศูนย์การจัดการให้ไปที่หน้าการ**เรียกเก็บเงิน** \> **[ของผลิตภัณฑ์ของคุณ](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. ค้นหาการสมัครใช้งานที่คุณต้องการขยาย ในส่วนการ**เรียกเก็บเงิน**ให้เลือก**ขยายวันที่สิ้นสุด**

3. ในบานหน้าต่าง**ขยายการสมัคร**ใช้งานให้เลือก**ใช้คีย์ผลิตภัณฑ์ใหม่และยัง**ไม่ได้ใช้งานจากนั้นเลือก**ถัดไป**

4. ใส่คีย์ผลิตภัณฑ์จากนั้นเลือก**ถัดไป**

> [!NOTE]
> ถ้าคุณมีคีย์ผลิตภัณฑ์มากกว่าหนึ่งรายการคุณสามารถเลือก **เพิ่มคีย์ผลิตภัณฑ์อื่น** เพื่อใส่ลงไปได้

5. ตรวจทานรายละเอียดการสั่งซื้อของคุณจากนั้นเลือก **แลก**ใช้

เมื่อต้องการเรียนรู้เพิ่มเติมให้ดู[เพิ่มสิทธิ์การใช้งานหรือขยายการสมัครใช้งานที่ชำระเงินสำหรับการใช้คีย์ผลิตภัณฑ์](https://docs.microsoft.com/microsoft-365/commerce/licenses/add-licenses-using-product-key)
